# 洛阳市人才补贴政策

> 最后核实日期：2026-07-01
>
> 政策依据：洛人社〔2025〕9号、洛建〔2025〕6号，由洛阳市人社局、洛阳市住建局发布并执行。

---

## 一、洛阳市青年人才生活补贴

| 项目 | 内容 |
|------|------|
| 政策名称 | 洛阳市青年人才生活补贴 |
| 政策文号 | 洛人社〔2025〕9号 |
| 适用对象 | 全日制本科、硕士、博士毕业生 |
| 补贴金额/标准 | 博士2000元/月、硕士1500元/月、本科1000元/月，补贴2年（民营企业、公立医院引进博士3年）。2017年起国家统招非全日制研究生可参照全日制享受 |
| 学历要求 | 全日制本科、硕士、博士（2017年起国家统招非全日制研究生可参照全日制享受） |
| 年龄限制 | 博士≤45周岁，硕士、本科≤40周岁 |
| 社保要求 | 个人参保缴费证明+个人所得税纳税记录（含工资薪金） |
| 申请条件 | 1. 2024年6月1日后在洛阳市首次就业/创业；2. 博士≤45周岁，硕士、本科≤40周岁；3. 就业单位不含供电、烟草、通信、金融类主体；4. 暂未就业者补贴6个月；5. 有效期至2027年5月31日 |
| 申请流程 | 1. 用人单位登录"洛阳市高层次人才服务系统"注册，组织申报人注册并提交材料、初审；2. 区人社局会同相关部门审核，每季度结束次月一周内报市人社局；3. 市人社局官网公示5个自然日；4. 公示无异议后编制资金分配计划，按季度核发 |
| 申请平台 | "洛阳社保卡"微信公众号 / "洛阳人才"微信公众号 / "洛快办"APP → 洛阳市高层次人才服务系统 |
| 申请材料 | 1. 统一社会信用代码证书或营业执照+近6个月完税证明（用人单位注册）；2. 授权委托书、法人及经办人身份证（用人单位注册）；3. 《洛阳市青年人才生活补贴申请表》+身份证；4. 毕业证、学位证（国内加《教育部学历证书电子注册备案表》，留学回国加《国(境)外学历学位认证书》）；5. 劳动合同；6. 个人参保缴费证明+个人所得税纳税记录（含工资薪金）；7. 创业人员加营业执照+近6个月纳税证明 |
| 审核周期 | 常态化受理（未申报者需于2025年12月31日前申报），按季度核发 |
| 发放方式 | 按季度核发 |
| 生效时间 | 2024-06-01 |
| 失效时间 | 2027-05-31 |
| 政策状态 | 有效 |
| 官方来源 | 洛阳市人民政府官网（https://www.ly.gov.cn/2025/05-09/566081.html） |
| 最后核实日期 | 2026-07-01 |

## 二、洛阳市人才租房补贴

| 项目 | 内容 |
|------|------|
| 政策名称 | 洛阳市人才租房补贴 |
| 政策文号 | 洛建〔2025〕6号 |
| 适用对象 | 全日制本科、硕士、博士毕业生 |
| 补贴金额/标准 | 博士1200元/月、硕士800元/月、本科500元/月，补贴3年（36个月） |
| 学历要求 | 全日制本科、硕士、博士 |
| 年龄限制 | ≤40周岁（博士≤45周岁） |
| 社保要求 | 每年在洛阳工作≥6个月且缴纳社保≥6个月 |
| 申请条件 | 1. 2024年6月1日后全职引进；2. ≤40周岁（博士≤45周岁）；3. 每年在洛阳工作≥6个月且缴纳社保≥6个月；4. 本人及家庭成员在洛阳城市区无自有住房，5年内无住房登记和交易记录；5. 与购房补贴可租转购（需扣除已发租房补贴） |
| 申请流程 | 1. 个人线上申请；2. 单位初审；3. 市/区住建局（住房保障中心）每季度第二个月底前复审；4. 市住建局每季度最后一个月1-20日组织人社、社保、医保等部门联审；5. 审核结果在市住房保障中心网站公示5个自然日；6. 发放 |
| 申请平台 | 洛阳人才住房补贴系统 |
| 申请材料 | 1. 人才住房补贴申请表；2. 身份证、户口本、婚姻证明；3. 学历学位证书及学信网备案表/国(境)外学历认证书；4. 劳动合同、社保缴费证明；5. 租房合同；6. 单位统一社会信用代码证书、营业执照、纳税证明 |
| 审核周期 | 常态化受理（每季度集中审核） |
| 发放方式 | 按季度核发 |
| 生效时间 | 2024-06-01 |
| 失效时间 | null（持续执行） |
| 政策状态 | 有效 |
| 官方来源 | 洛阳本地宝（https://m.ly.bendibao.com/live/33228.shtm） |
| 最后核实日期 | 2026-07-01 |
