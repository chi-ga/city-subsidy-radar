#### 1. 西安硕博人才奖

| 字段 | 内容 |
|------|------|
| **政策名称** | 西安硕博人才奖 |
| **政策文号** | 市人社函〔2020〕615号（《"西安硕博人才奖""硕博人才求职创业生活补贴"和"企业引进硕博人才奖励"实施细则》） |
| **适用对象** | 在"西安市高层次研究生就业储备池"登记的应届硕博研究生，毕业起1年内在西安市实现就业或创业 |
| **补贴金额/标准** | 博士每人1万元、硕士每人5000元，一次性发放 |
| **学历要求** | 应届硕士、博士研究生 |
| **年龄限制** | null |
| **社保要求** | 依法缴纳社会保险6个月以上（含6个月） |
| **申请条件** | 1. 在"西安市高层次研究生就业储备池"登记的应届硕博研究生（入池范围：西安地区已毕业未就业的应届博士研究生、符合西安市"6+5+6+1"现代产业体系急需紧缺专业的应届硕士研究生）；2. 毕业起1年内（以毕业证日期为准）在西安市实现就业（在西安依法登记注册的各类机关及企事业单位，含中央驻陕单位和省属单位）并签订1年以上劳动（聘用）合同，或在西安市行政区域内自主创业取得营业执照及相关经营手续；3. 依法缴纳社会保险6个月以上（含6个月）；4. 自实现就业或创业之日起1年内申请 |
| **申请流程** | 1. 完成"西安市高层次研究生就业储备池"登记（入池）→2. 登录西安市人力资源和社会保障局官网（http://xahrss.xa.gov.cn/）→3. 选择"网上服务"→"进入大厅"→"登录"→4. 选择"个人登录"或"政务网登录"→5. 登录完成后点击"首页"→6. 选择"硕博人才引进"→7. 选择"硕博人才奖申报"→8. 填写《硕博人才奖申报表》，按系统提示申报并上传相关资料 |
| **申请平台** | 西安市人力资源和社会保障局官网（http://xahrss.xa.gov.cn/），网上服务大厅→硕博人才引进→硕博人才奖申报 |
| **申请材料** | 实现就业（签订劳动合同）人员：1. 劳动（聘用）合同首尾页电子扫描件；2. 党政机关或参照公务员法管理单位人员，提交公务员管理部门出具的公务员（参公）登记表复印件或相关证明；3. 社保证明：如系统未查找出个人参保信息，则需补充上传由社保经办机构开具盖章的当前参保证明（6个月以上，含6个月）。自主创业（取得营业执照及相关经营手续）人员：1. 企业营业执照（副本）及创业实体纳税证明材料电子扫描件；2. 社保证明：如系统未查找出个人参保信息，则需补充上传由社保经办机构开具盖章的当前参保证明（6个月以上，含6个月） |
| **审核周期** | null |
| **发放方式** | 一次性发放 |
| **生效时间** | 2020年 |
| **失效时间** | 待核实 |
| **政策状态** | 有效。2024年度博士研究生安家补助费发放工作通知（市人社函〔2024〕412号）中仍涉及硕博人才相关补贴，硕博人才奖申报通道持续开放。入池范围为应届博士研究生及符合"6+5+6+1"现代产业体系急需紧缺专业的应届硕士研究生。 |
| **官方来源** | http://xahrss.xa.gov.cn/ |
| **最后核实日期** | 2026-07-01 |

#### 2. 硕博人才求职创业生活补贴

| 字段 | 内容 |
|------|------|
| **政策名称** | 硕博人才求职创业生活补贴 |
| **政策文号** | 市人社函〔2020〕615号（《"西安硕博人才奖""硕博人才求职创业生活补贴"和"企业引进硕博人才奖励"实施细则》） |
| **适用对象** | 领取"西安硕博人才奖"的博士、硕士（即在"就业储备池"登记的应届硕博研究生） |
| **补贴金额/标准** | 博士每人每月4000元、硕士每人每月2000元，最长补贴期限为1年。补贴时段为自"就业储备池"登记之日至实现就业（创业）之前 |
| **学历要求** | 应届硕士、博士研究生 |
| **年龄限制** | null |
| **社保要求** | null（补贴期间为就业前阶段，无需社保；实现就业后缴纳社保6个月以上可申领硕博人才奖） |
| **申请条件** | 1. 已在"西安市高层次研究生就业储备池"登记的应届硕博研究生；2. 补贴时段为自"就业储备池"登记之日至实现就业（创业）之前；3. 最长补贴期限为1年；4. 领取"西安硕博人才奖"的博士、硕士可同时享受本补贴 |
| **申请流程** | 1. 完成"西安市高层次研究生就业储备池"登记（入池）→2. 享受自登记之日至实现就业前的生活补贴→3. 实现就业后登录西安市人社局官网申请硕博人才奖及生活补贴→4. 审核通过后发放 |
| **申请平台** | 西安市人力资源和社会保障局官网（http://xahrss.xa.gov.cn/），网上服务大厅→硕博人才引进 |
| **申请材料** | 1. 劳动（聘用）合同首尾页电子扫描件（实现就业人员）；2. 党政机关或参公单位人员提交公务员（参公）登记表复印件或相关证明；3. 社保证明（6个月以上，如系统未查找出个人参保信息需补充上传）；4. 企业营业执照（副本）及创业实体纳税证明材料电子扫描件（自主创业人员） |
| **审核周期** | null |
| **发放方式** | 按月计发 |
| **生效时间** | 2020年 |
| **失效时间** | 待核实 |
| **政策状态** | 有效。与西安硕博人才奖配套实施，入池后至就业前期间按月发放生活补贴。实现就业后缴纳社保满6个月可申领硕博人才奖。 |
| **官方来源** | http://xahrss.xa.gov.cn/ |
| **最后核实日期** | 2026-07-01 |

#### 3. 博士研究生安家补助费

| 字段 | 内容 |
|------|------|
| **政策名称** | 西安市博士研究生安家补助费 |
| **政策文号** | 市人社函〔2024〕412号（《西安市人力资源和社会保障局关于做好2024年度博士研究生安家补助费发放工作的通知》）；政策依据：《西安市进一步加快人才汇聚若干措施》 |
| **适用对象** | 西安市属单位公开招聘的博士研究生 |
| **补贴金额/标准** | 每人15万元，分5年发放（每年3万元）。此前标准为5年10万元，已提高至5年15万元 |
| **学历要求** | 博士研究生 |
| **年龄限制** | null |
| **社保要求** | 签订聘用合同 |
| **申请条件** | 1. 西安市属单位公开招聘的博士研究生；2. 签订聘用合同后享受转正定级工资待遇；3. 引进博士不实行试用期；4. 每人每年增发1个月工资并补助安家费3万元，5年共补助15万元 |
| **申请流程** | 1. 个人申请→2. 用人单位审核申报→3. 主管部门复审→4. 公示→5. 发放 |
| **申请平台** | 西安市人力资源和社会保障局官网（http://xahrss.xa.gov.cn/） |
| **申请材料** | null |
| **审核周期** | null |
| **发放方式** | 分5年发放（每年3万元） |
| **生效时间** | 2024年（2024年度发放工作通知文号市人社函〔2024〕412号） |
| **失效时间** | 待核实 |
| **政策状态** | 有效。2024年度博士研究生安家补助费发放工作已开展，西安市教育局发布公示名单。安家补助费标准已由5年10万提高为5年15万。适用于市属单位公开招聘的博士研究生。 |
| **官方来源** | http://xahrss.xa.gov.cn/ |
| **最后核实日期** | 2026-07-01 |
