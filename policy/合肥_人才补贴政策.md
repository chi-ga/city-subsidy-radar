#### 1. 合肥市高校毕业生住房租赁补贴

| 字段 | 内容 |
|------|------|
| **政策名称** | 合肥市高校毕业生住房租赁补贴 |
| **政策文号** | 合人才〔2022〕1号（试行）、合人才〔2024〕1号（修订版） |
| **适用对象** | 在我市产业及相关领域重点单位就业的35周岁以下博士、硕士、普通高校本科毕业生 |
| **补贴金额/标准** | 博士每年3.6万元、硕士每年2万元、本科每年1.5万元，累计发放不超过36个月 |
| **学历要求** | 博士研究生、硕士研究生、普通高校本科毕业生 |
| **年龄限制** | 35周岁（含）以下 |
| **社保要求** | 通过工作单位在肥缴纳企业职工养老保险累计3个月以上（不含补缴） |
| **申请条件** | ①申请人及其配偶、未成年子女在合肥市行政区域（含四县一市）无住房不动产登记信息；②自行租赁住房的，须通过合肥市住房租赁交易服务平台办理住房租赁合同备案；③所在企业属于政策规定的产业及相关领域重点单位；④2022年6月16日（含）后新来我市就业，且近三年在肥无参保记录 |
| **申请流程** | ①注册/登录安徽政务服务网或皖事通APP；②进入合肥市人才综合服务平台完善个人信息；③发起"住房租赁补贴（2022）"申请；④大数据自动核验学历、住房、社保、企业纳税地等信息；⑤所在企业审核；⑥县区住房保障部门公示、兑现 |
| **申请平台** | 合肥市人才综合服务平台：https://rcaj.hfrsggff.com:8088/talent/；安徽政务服务网合肥分厅/皖事通APP"人才安居"通道 |
| **申请材料** | 1.身份证；2.社会保障卡；3.毕业证/学位证；4.国（境）外学历学位认证（留学人员）；5.婚姻材料（已婚）；6.银行卡号；7.住房租赁合同备案信息 |
| **审核周期** | 约15-30个工作日 |
| **发放方式** | 自公示通过次月起按月发放至申请人银行账户；初次申请时已连续缴纳3个月社保的，可补发2个月 |
| **生效时间** | 2022-06-16 |
| **失效时间** | 待核实（2022版试行至2025-12-31；2024修订版继续有效） |
| **政策状态** | 有效 |
| **官方来源** | https://www.hfxf.gov.cn/rsks/ysfwsc/18854737.html |
| **最后核实日期** | 2026-06-30 |

#### 2. 合肥市博士购房补贴

| 字段 | 内容 |
|------|------|
| **政策名称** | 合肥市博士购房补贴 |
| **政策文号** | 合人才〔2022〕1号（试行）、合人才〔2024〕1号（修订版） |
| **适用对象** | 在我市产业及相关领域重点单位就业的35周岁以下博士研究生 |
| **补贴金额/标准** | 最高12万元（购房补贴金额不超过实际购房金额） |
| **学历要求** | 博士研究生 |
| **年龄限制** | 35周岁（含）以下 |
| **社保要求** | 通过工作单位在肥连续缴纳社会保险满1年 |
| **申请条件** | ①在肥购买首套自住住房；②购房时申请人及配偶、未成年子女在合肥市行政区域无住房且无住房转让记录；③商品住房网签时间在2022年6月16日（含）后；④购房补贴金额不超过实际购房金额；⑤所在企业属于政策规定的产业及相关领域重点单位 |
| **申请流程** | ①注册/登录安徽政务服务网或皖事通APP；②进入合肥市人才综合服务平台完善个人信息；③进入购房补贴模块发起申请；④所在企业审核；⑤政府审核、公示；⑥按年度拨付 |
| **申请平台** | 合肥市人才综合服务平台：https://rcaj.hfrsggff.com:8088/talent/；安徽政务服务网合肥分厅/皖事通APP |
| **申请材料** | 1.身份证；2.婚姻材料；3.博士毕业证/学位证；4.银行卡号；5.购房网签合同；6.社保缴纳证明；7.住房情况查询授权材料 |
| **审核周期** | 约15-30个工作日 |
| **发放方式** | 分5年拨付；购房后每年可申请续发，房产仍须在申请人名下 |
| **生效时间** | 2022-06-16 |
| **失效时间** | 待核实（2022版试行至2025-12-31；2024修订版继续有效） |
| **政策状态** | 有效 |
| **官方来源** | https://www.hfxf.gov.cn/rsks/ysfwsc/18854737.html |
| **最后核实日期** | 2026-06-30 |

### 区级政策

> 最后核实日期：2026-06-30
> 数据来源：合肥各区人民政府官网、合肥本地宝等官方渠道

---

经公开渠道检索，合肥市各区（高新区、经开区、新站高新区、蜀山区、包河区、庐阳区、瑶海区、肥东县、肥西县、长丰县、庐江县、巢湖市等）目前主要落实市级人才政策，未发现面向普通高校毕业生（本科/硕士/博士）的独立学历分档人才生活补贴或租房补贴政策。各区购房补贴多为普惠性购房补贴政策（不按学历分档），或有县级政策已到期停止执行，故暂不纳入本系统。

> **说明**：合肥市高校毕业生住房租赁补贴和博士购房补贴虽由市级统一制定，但执行中由申请人所在企业注册地所在县（市）区负责审核兑现。同一申请人只能在一个县（市）区申请，但该限制属于市级政策的正常执行分工，不属于区级政策互斥。
