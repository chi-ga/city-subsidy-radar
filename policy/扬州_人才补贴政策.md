## 一、扬州市企业引进人才租房补贴

| 项目 | 内容 |
|------|------|
| 政策名称 | 扬州市企业引进人才租房补贴 |
| 政策文号 | 《关于深入实施"绿扬金凤计划"+强化新质生产力人才支撑的若干政策》（扬发〔2025〕15号）；原人才新政15条租购房补贴实施细则 |
| 适用对象 | 35周岁以下、来扬州市相关企业工作的全职博士、硕士、本科、大专（高级工）人才 |
| 补贴金额/标准 | 博士3500元/月；硕士1500元/月；本科1000元/月；大专（高级工）500元/月；发放3年 |
| 学历要求 | 大专（高级工）及以上 |
| 年龄限制 | 35周岁以下 |
| 社保要求 | 在扬企业就业并缴纳社会保险 |
| 申请条件 | 1.35周岁以下；2.全职来扬州市相关企业工作（银行、保险、证券、电讯、供电等企业除外）；3.大专（高级工）及以上学历；4.在扬缴纳社会保险 |
| 申请流程 | 1.登录扬州人才智慧云平台或"我的扬州"APP，进入人才码首页；2.选择租房补贴申请；3.条件自检，阅读承诺书、授权书并勾选；4.填写表单（系统自动获取学历、参保、社保缴费、社保卡等信息）；5.上传材料；6.查看办事进度 |
| 申请平台 | 扬州人才智慧云平台（https://hrss.yangzhou.gov.cn/rcm）；"我的扬州"APP；扬州人社APP |
| 申请材料 | 1.本人身份证；2.毕业证书（海外留学人员需提供海外学历毕业证书）；3.劳动合同（人力资源机构或劳务派遣人员须上传）；4.社保缴费证明（系统自动获取）。具体材料清单以申报系统提示为准 |
| 审核周期 | 由各县（市、区）人社部门受理审核，公示后发放 |
| 发放方式 | 按月发放，发放3年 |
| 生效时间 | 2022-01-01（人才政策3.0起执行，2025年升级为4.0延续） |
| 失效时间 | null |
| 政策状态 | 有效。人才政策4.0（扬发〔2025〕15号）持续执行中 |
| 官方来源 | 扬州本地宝（https://m.yz.bendibao.com/live/21690.shtm）；扬州人才智慧云平台（https://hrss.yangzhou.gov.cn/rcm） |
| 最后核实日期 | 2026-07-01 |

## 二、扬州市企业引进人才购房补贴

| 项目 | 内容 |
|------|------|
| 政策名称 | 扬州市企业引进人才购房补贴 |
| 政策文号 | 《关于深入实施"绿扬金凤计划"+强化新质生产力人才支撑的若干政策》（扬发〔2025〕15号）；《扬州市市区"人才安家券"补贴实施细则》（人才新政4.0） |
| 适用对象 | 市区企业新引进35周岁以下全职博士（正高职称）、硕士（副高职称、高级技师）、双一流高校本科、普通本科（技师）人才 |
| 补贴金额/标准 | 博士（正高职称）30万元；硕士（副高职称、高级技师）15万元；双一流高校本科7万元；普通本科（技师）5万元；一次性发放（补贴资金由市财政和企业所在地区级财政各承担50%） |
| 学历要求 | 全日制本科及以上学历或技师及以上职业资格 |
| 年龄限制 | 35周岁以下 |
| 社保要求 | 在扬企业就业并缴纳社会保险 |
| 申请条件 | 1.35周岁以下；2.全职来扬州市区企业工作；3.全日制本科及以上学历或技师及以上职业资格；4.5年内在扬首次购买住房 |
| 申请流程 | 1.每年集中组织申报上年度企业引进人才购房补贴；2.由各县（市、区）人社部门、功能区组织人事部门组织符合条件的人才填写申报材料；3.申报工作逐步过渡为网上申报，通过扬州人才智慧云平台人才码申请 |
| 申请平台 | 扬州人才智慧云平台（https://hrss.yangzhou.gov.cn/rcm）；"我的扬州"APP；扬州人社APP |
| 申请材料 | 1.本人身份证；2.毕业证书（学位证书）；3.劳动合同；4.社保缴费证明；5.商品房买卖合同；6.不动产权证书；7.契税完税证明。具体材料清单以申报系统提示为准 |
| 审核周期 | 每年集中组织申报，审核公示后发放 |
| 发放方式 | 一次性 |
| 生效时间 | 2022-01-01（人才政策3.0起执行，2025年升级为4.0延续） |
| 失效时间 | null |
| 政策状态 | 有效。人才政策4.0（扬发〔2025〕15号）持续执行中 |
| 官方来源 | 扬州本地宝（https://m.yz.bendibao.com/job/23521.shtm）；扬州市人民政府（http://www.yangzhou.gov.cn/xwzx/zwyw/art/2025/art_f86bf07b5dcb46d5a1108c9f6d72c117.html） |
| 最后核实日期 | 2026-07-01 |

| 失效时间 | null |
| 政策状态 | 有效。人才政策4.0持续执行中 |

### 区级政策互斥关系说明

扬州市区（含广陵区、邗江区、江都区、经济技术开发区、生态科技新城、蜀冈-瘦西湖风景名胜区）执行统一的市级人才补贴标准。各县（市、区）、功能区结合地区实际制定了个性化补贴政策（如生态科技新城对租房补贴标准有所提高），符合条件的市级补贴与区级个性化补贴按各区规定执行。市级租房补贴、购房补贴为不同类型可分别申请；来扬留扬回扬专项补贴与探亲交通补贴可叠加享受。本文件收录的均为市级统一标准政策，区级个性化政策请咨询各区人社部门。
