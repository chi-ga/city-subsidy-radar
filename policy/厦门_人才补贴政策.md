#### 1. 厦门市新引进人才生活补贴

| 字段 | 内容 |
|------|------|
| **政策名称** | 厦门市新引进人才生活补贴 |
| **政策文号** | 厦人社〔2024〕239号、厦人社〔2025〕227号、厦府规〔2025〕14号 |
| **适用对象** | 2025年11月1日起新引进到厦门市辖区范围内企事业单位（不含参公事业单位）和社会组织的人才；台港澳籍和外籍人才可参照申请 |
| **补贴金额/标准** | 博士每人8万元、硕士每人5万元、“双一流”建设高校（学科）及世界最新排名前200名大学本科每人3万元、其他高校应届本科毕业生每人1万元；技工院校全日制预备技师班应届毕业生每人1万元、高级工班应届毕业生每人0.6万元。均一次性发放。重点产业方向（半导体和集成电路、生物医药、新能源新材料、人工智能、未来网络、深海空天开发）补贴标准可上浮30% |
| **学历要求** | 全日制普通高等教育本科及以上学历，或经教育部认证在国（境）外取得的本科学士及以上学位（含统招非全日制研究生学历）；“双一流”或世界前200名大学以毕业当年名单为准；2024年1月1日后毕业的技工院校全日制预备技师班、2025年1月1日后毕业的技工院校全日制高级工班应届毕业生 |
| **年龄限制** | 博士不超过40周岁，硕士不超过35周岁，本科、预备技师和高级工不超过30周岁；不含周岁生日当天 |
| **社保要求** | 申请时自引进之日起在厦缴纳社会保险费满3个月及以上，且社会保险关系仍在申报单位 |
| **申请条件** | 1. 具有厦门户口；2. 引进时已具有符合要求的学历学位或技工院校培养层级；3. 经市组织、人社、教育部门核准并已办理入厦人事关系；4. 已在厦就业且社保在缴；5. 未重复申领同类型毕业生生活补贴；6. 自引进之日起1年内由用人单位提出申请 |
| **申请流程** | 1. 申请人向用人单位提出申请；2. 用人单位登录厦门市人力资源和社会保障局网上服务大厅提交申请；3. 各受理部门（市人社局、市教育局、市科技局、市工信局或各区人社局/园区管委会）受理审核；4. 公示5个工作日；5. 一次性拨付至人才个人账户 |
| **申请平台** | 厦门市人力资源和社会保障局网上服务大厅（https://hrss.xm.gov.cn/） |
| **申请材料** | 1. 申请人有效身份证件；2. 毕业证、学位证（留学回国人员提供教育部学历学位认证书，技工院校毕业生提供毕业证书）；3. 与厦门用人单位签订的劳动合同或聘用合同；4. 在厦社会保险缴纳证明；5. 个人承诺书、授权书；6. 用人单位需配合上传单位相关印证材料 |
| **审核周期** | 公示5个工作日，具体审核周期以受理部门为准 |
| **发放方式** | 一次性发放至人才个人账户 |
| **生效时间** | 2025年11月1日（本指南施行时间，此前引进人员按原政策执行） |
| **失效时间** | 待核实 |
| **政策状态** | 有效 |
| **官方来源** | https://hrss.xm.gov.cn/xxgk/zfxxgkzl/zfxxgkml/qtxx/rcfw/202511/t20251111_2966394.htm |
| **最后核实日期** | 2026-07-01 |

#### 2. 厦门市新就业大学生等青年群体租赁住房保障（5年5折租房）

| 字段 | 内容 |
|------|------|
| **政策名称** | 厦门市新就业大学生等青年群体租赁住房保障（5年5折租房） |
| **政策文号** | 厦府办规〔2021〕7号 |
| **适用对象** | 2021年1月1日后首次来厦就业、创业且30周岁（含）以下的全日制本科（含）以上毕业生；经各区（管委会）认定的、符合产业发展需要的30周岁（含）以下非全日制本科以上学历技术骨干人才可参照执行 |
| **补贴金额/标准** | 采取实物配租与租金补贴相结合。租金补贴标准：思明区、湖里区每人每年8000元，集美区、海沧区每人每年6000元，同安区、翔安区每人每年5000元；保障期限最长不超过5年。实物配租原则上按不超过40平方米/人以市场评估租金五折配租 |
| **学历要求** | 全日制本科（含）以上；技术骨干人才可放宽至非全日制本科以上 |
| **年龄限制** | 30周岁（含）以下 |
| **社保要求** | 在本市缴纳职工养老保险或医疗保险满3个月 |
| **申请条件** | 1. 2021年1月1日后首次来厦就业、创业；2. 来厦就业的须提供2021年1月1日后与本市用工单位签订1年（含）以上劳动合同；自主创业的须提供2021年1月1日后在本市办理的营业执照、办公场所租赁合同等材料；3. 在本市缴纳职工养老或医疗保险满3个月；4. 申请人及其配偶在本市无自有住房（或与父母共有家庭唯一一套住房）；5. 未享受直管公房、拆迁安置房等住房优惠政策 |
| **申请流程** | 1. 申请人通过统一信息服务平台提交申请（全程网办）；2. 各区（管委会）审核，住房、公安、财政、人社、资源规划、医保、税务等部门数据支持；3. 审核通过后在各区官网公示5个工作日；4. 选择实物配租的由各区组织配租；选择租金补贴的按周期发放 |
| **申请平台** | 厦门市住房保障和房屋管理局统一信息服务平台（具体以“厦门住房”微信公众号/官网入口为准） |
| **申请材料** | 1. 申请人有效身份证件；2. 2021年1月1日后签订的1年（含）以上劳动合同，或2021年1月1日后办理的营业执照及办公场所租赁合同；3. 在本市缴纳职工养老或医疗保险满3个月的证明；4. 申请人及配偶在厦无自有住房证明（或与父母共有家庭唯一住房证明）；5. 租赁备案或租房合同（申请租金补贴时提供）；6. 其他平台要求补充的材料 |
| **审核周期** | 公示5个工作日 |
| **发放方式** | 户籍已迁入本市的按年度一次性全额发放；户籍未迁入本市的每半年发放一次。实物配租按五折租金直接减免 |
| **生效时间** | 2021年7月28日 |
| **失效时间** | 2026年7月27日（有效期5年） |
| **政策状态** | 有效（有效期至2026-07-27） |
| **官方来源** | https://www.xm.gov.cn/zwgk/flfg/sfbwj/202107/t20210729_2570715.htm |
| **最后核实日期** | 2026-07-01 |

### 区级政策说明

> 最后核实日期：2026-07-01
> 数据来源：厦门市各区人民政府官网、区人社局公示信息、厦门本地宝

经公开渠道检索，厦门市各区（思明区、湖里区、集美区、海沧区、同安区、翔安区及火炬管委会、自贸区管委会）目前主要执行上述市级人才补贴政策：新引进人才生活补贴由市、区（园区）分级受理，5年5折租房由各区（管委会）按区域标准审核发放。未检索到各区独立发布、与市级政策互斥的普惠性区级人才生活/租房/购房补贴。各区现有人才住房保障政策（如湖里区社工人才住房补贴、海沧区人才住房补贴、集美区重点产业企业人才租房补贴等）多需特定人才认定或面向特定职业/产业，不满足本系统“普通毕业生凭学历+年龄+社保即可判断”的普惠性标准，暂不纳入。
