#### 1. 贵阳贵安人才租房补贴（毕业三年内高校毕业生）

| 字段 | 内容 |
|------|------|
| **政策名称** | 贵阳贵安人才租房补贴 |
| **政策文号** | 筑人才发〔2024〕6号（《贵阳贵安实施人才兴市战略若干政策措施》） |
| **适用对象** | 毕业三年内在贵阳贵安就业创业的高校毕业生，租住贵阳市人才保障性租赁住房或经认定市场化房源 |
| **补贴金额/标准** | 按月发放：博士800元/月；硕士400元/月；本科及大专200元/月。租住人才保障性租赁住房的累计发放不超过18个月；租住经认定市场化房源的累计发放不超过12个月 |
| **学历要求** | 大专、本科、硕士、博士（高校毕业生，不要求全日制） |
| **年龄限制** | null（以毕业三年内为准，不设年龄上限） |
| **社保要求** | 须在贵阳贵安就业或自主创业（依法参加社会保险或办理营业执照） |
| **申请条件** | 1.毕业三年内的高校毕业生；2.在贵阳贵安就业创业；3.租住人才保障性租赁住房或经认定市场化房源，且租赁合同已网签备案 |
| **申请流程** | 人才登录贵阳贵安一站式人才网上办事平台申报→提交身份、学历、社保、租赁合同网签备案等材料→审核→公示→按年/按月发放租金补贴 |
| **申请平台** | 贵阳贵安一站式人才网上办事平台，https://www.gyrc.cn/yzs |
| **申请材料** | 1.本人身份证；2.学历（或职称、人才卡）证明材料；3.社会保险参保证明（自主创业提供营业执照）；4.人才住房租赁合同及网签备案证明；5.个人承诺书；6.其他需要补充说明情况的材料（具体材料以平台提示为准） |
| **审核周期** | 待核实 |
| **发放方式** | 按月发放，租住保障性租赁住房累计不超过18个月、市场化房源累计不超过12个月 |
| **生效时间** | 2024年（筑人才发〔2024〕6号） |
| **失效时间** | 待核实（现行有效） |
| **政策状态** | 有效。毕业三年内博士800元/月、硕士400元/月、本科及大专200元/月；保租房累计≤18个月、市场化房源累计≤12个月 |
| **官方来源** | https://sp.gznc.edu.cn/info/1259/1481.htm |
| **最后核实日期** | 2026-07-01 |

#### 2. 贵阳贵安人才购房补贴（全日制高校毕业生）

| 字段 | 内容 |
|------|------|
| **政策名称** | 贵阳贵安人才购房补贴 |
| **政策文号** | 筑人社发〔2025〕2号（《贵阳贵安人才购房补贴实施细则（试行）》） |
| **适用对象** | 重点产业企业全职新引进3个月以上、依法参加社会保险、购房时年龄不超过35周岁的全日制高校毕业生 |
| **补贴金额/标准** | 一次性发放：博士研究生5万元；硕士研究生3万元；本（专）科毕业生1万元 |
| **学历要求** | 全日制本（专）科、硕士、博士 |
| **年龄限制** | 购房时不超过35周岁（含） |
| **社保要求** | 在重点产业企业全职新引进3个月以上，依法参加社会保险 |
| **申请条件** | 1.所在单位为主营业务属于《贵阳贵安重点发展产业目录》的重点产业企业；2.全职新引进3个月以上、依法参保；3.购房时年龄不超过35周岁；4.所购为本人及配偶在贵阳贵安新购买的首套新建商品住房或二手住房，并完成网签备案或不动产权转移登记；5.购房时间为新引进到贵阳贵安之后3年内；6.应在购房完成网签备案或取得不动产权证书12个月内申报 |
| **申请流程** | 人才通过贵阳贵安一站式人才网上办事平台申报→主管部门或各区（市、县、开发区）人才工作部门初审→市人才服务中心会同相关部门审核→公示5个工作日→资金拨付至个人银行账户 |
| **申请平台** | 贵阳贵安一站式人才网上办事平台，https://www.gyrc.cn/yzs |
| **申请材料** | 1.《贵阳贵安人才购房补贴申请表》；2.与用人单位签订的劳动（聘用）合同及相应社会保险证明；3.婚姻证明材料（已婚提供结婚证明，未婚提供未婚承诺书，有离异经历的补充提供离婚证和离婚协议书或判决书等）；4.购买新建商品住房的提交商品房购买合同网签备案证明或预告登记不动产登记证明，购买二手房的提交二手房不动产权证书；5.住房为申请人与亲属共有的应提供亲属关系相关证明材料 |
| **审核周期** | 公示期5个工作日（自初审至拨付全流程以实际为准） |
| **发放方式** | 一次性拨付至个人银行账户 |
| **生效时间** | 2025-03-05（新购买住房以2024-07-26（含）后购买为准） |
| **失效时间** | 待核实（试行） |
| **政策状态** | 有效。重点产业企业全日制高校毕业生购房时≤35周岁，博士5万、硕士3万、本专1万一次性。每人只享一次，已享受其他财政购房补贴的不再享受 |
| **官方来源** | http://www.shegongw.com/sgchuji/jiuye/guizhou/202503/61362.htm |
| **最后核实日期** | 2026-07-01 |

#### 3. 贵阳贵安新引进人才安家费

| 字段 | 内容 |
|------|------|
| **政策名称** | 贵阳贵安新引进人才安家费 |
| **政策文号** | 筑人才发〔2024〕6号（《贵阳贵安实施人才兴市战略若干政策措施》，具体发放细则待核实） |
| **适用对象** | 重点产业企业新引进符合《贵阳贵安重点产业人才需求目录》的应往届高校毕业生和各类人才，且落户贵阳贵安 |
| **补贴金额/标准** | 按层次发放：正高职称（首席技师、特级技师）6万元；博士（高级技师）4.8万元；硕士（副高职称人才、技师）2.4万元；本科（高级工）1.2万元。引进人才工作满6个月后兑现50%，在工作单位持续工作满一年后兑现50% |
| **学历要求** | 本科、硕士、博士（正高职称/技能人才同等享受） |
| **年龄限制** | null |
| **社保要求** | 在重点产业企业新引进、依法参加社会保险并持续工作 |
| **申请条件** | 1.所在单位为重点产业企业（主营业务属《贵阳贵安重点发展产业目录》）；2.新引进符合《贵阳贵安重点产业人才需求目录》；3.落户贵阳贵安；4.工作满6个月后兑现50%，持续工作满一年后兑现50% |
| **申请流程** | 人才通过贵阳贵安一站式人才网上办事平台申报→主管部门或各区人才工作部门初审→市人才服务中心审核→公示→工作满6个月兑现50%、满一年兑现50% |
| **申请平台** | 贵阳贵安一站式人才网上办事平台，https://www.gyrc.cn/yzs |
| **申请材料** | 1.《贵阳贵安新引进人才安家费申请表》；2.本人身份证；3.学历学位证书（或职称/技能等级证书）；4.与重点产业企业签订的劳动（聘用）合同；5.社会保险参保证明；6.落户贵阳贵安的户籍证明；7.用人单位营业执照及重点产业企业资质证明；8.个人承诺书；9.其他需要补充说明情况的材料（具体材料以平台提示为准） |
| **审核周期** | 待核实（工作满6个月、满一年分两次兑现） |
| **发放方式** | 分期，工作满6个月兑现50%、持续工作满一年兑现50% |
| **生效时间** | 2024年（筑人才发〔2024〕6号） |
| **失效时间** | 待核实（现行有效） |
| **政策状态** | 有效。重点产业企业新引进人才且落户贵阳贵安，正高6万、博士4.8万、硕士2.4万、本科1.2万，分两次兑现 |
| **官方来源** | https://sp.gznc.edu.cn/info/1259/1481.htm |
| **最后核实日期** | 2026-07-01 |
