#### 1. 福州市高校毕业生就业创业一次性生活补贴

| 字段 | 内容 |
|------|------|
| **政策名称** | 福州市高校毕业生就业创业一次性生活补贴 |

| **适用对象** | 全日制本科及以上高校毕业生、在我市首次参保的非全日制研究生、技工院校预备技师班及以上毕业生（2024年及以后毕业） |
| **补贴金额/标准** | 每人1万元，一次性发放 |
| **学历要求** | 全日制本科及以上学历；或在我市首次参保的非全日制研究生；或技工院校预备技师班及以上 |
| **年龄限制** | null |
| **社保要求** | 在福州市范围内各类企业稳定就业或自主创业，按规定缴纳城镇职工养老保险三个月以上（同一单位连续缴费三个月及以上，不含补缴）；申请时仍在符合条件的单位缴纳养老保险 |
| **申请条件** | 1. 已具有福州市户口（含六区六市县户口及集体户）；2. 在福州市范围内各类企业稳定就业或自主创业；3. 按规定缴纳城镇职工养老保险三个月以上；4. 毕业一年内（以毕业证书签发时间起算）线上申报；5. 政策有效期三年，预计至2027年12月到期 |
| **申请流程** | 1. 下载并注册“e福州”APP或“闽政通”APP；2. 实名注册账号；3. 在APP首页或专题服务中找到“一次性生活补助”或相关模块；4. 阅读申报提醒并填写个人基本信息（毕业学校、毕业证书编号、社保卡金融账户号码、毕业时间、学历类型、毕业生来源、落户信息、就业去向、参保时间、就业单位所在地等）；5. 提交申请；6. 市人社部门通过学信网、社保大数据审核，落户信息由市公安部门核查；7. 审核通过后5个工作日内发放至社会保障卡银行账户 |
| **申请平台** | “e福州”APP、“闽政通”APP |
| **申请材料** | 1. 个人基本信息；2. 毕业证书编号（国外学历填写国外学历学位认证书编号）；3. 有效的社保卡金融账户号码；4. 学信网、社保大数据自动核验学历和社保，落户信息由公安部门核查 |
| **审核周期** | 审核通过后5个工作日内发放 |
| **发放方式** | 一次性发放至申请人社会保障卡银行账户 |
| **生效时间** | 2024年12月18日（《福州市鼓励高校毕业生来榕留榕就业创业八条措施的通知》发布） |
| **失效时间** | 2027年12月（政策有效期三年） |
| **政策状态** | 有效。面向全日制本科及以上毕业生、首次在榕参保的非全日制研究生、技工院校预备技师班及以上毕业生，属于普惠性生活补贴 |
| **官方来源** | https://m.fz.bendibao.com/live/58420.shtm |

### 区级政策说明

> 最后核实日期：2026-07-01
> 数据来源：福州市鼓楼区、台江区、仓山区、晋安区人民政府官网，长乐区人民政府官网，马尾区人民政府官网

经公开渠道检索，福州市鼓楼区、台江区、仓山区、晋安区（四城区）目前主要执行市级“好年华 聚福州”人才住房保障政策，未检索到四城区独立发布的普惠性区级人才生活/租房/购房补贴。长乐区独立实施人才住房租赁补贴和人才购房补贴，面向在长乐区工作并经认定的人才或全日制大专及以上毕业生，与市级四城区及市属单位人才住房保障政策互斥，同一人不得重复享受。马尾区虽出台《福州市马尾区人才住房保障实施办法》（榕马委人才〔2024〕1号），但公开渠道未检索到完整的补贴标准、学历条件及申请细则，暂不纳入本系统政策库。福清市、闽侯县、连江县、罗源县、闽清县、永泰县暂未纳入本次公开检索范围。
