## 昆明

### 市级政策框架

> 最后核实日期：2026-07-01
> 数据来源：昆明市人力资源和社会保障局官网

昆明市高校毕业生“来昆留昆”就业创业补贴政策主要依据《昆明市人民政府关于进一步做好当前和今后一个时期就业创业工作的实施意见》（昆人社通〔2022〕6号）及各区贯彻落实的申报办法执行。市级文件明确落户补贴、租房补贴、购房补贴、来昆留昆就业补贴等四类支持，具体受理、审核、资金拨付由毕业生就业/创业所在县（市、区）人社部门负责。因此，以下按区录入实际可申请的区级普惠性补贴。同一毕业生只能在一个区落户、就业，故各区同类补贴之间为互斥关系。

#### 1. 西山区高校毕业生来昆留昆租房补贴

| 字段 | 内容 |
|------|------|
| **政策名称** | 西山区高校毕业生来昆留昆租房补贴 |
| **政策文号** | 昆人社通〔2022〕6号（区级贯彻落实） |
| **适用对象** | 毕业年度（毕业当年1月至12月）全日制高校毕业生 |
| **补贴金额/标准** | 博士研究生8000元、硕士研究生5000元、双一流高校本科生3000元、普通高校本科生2000元，一次性发放 |
| **学历要求** | 全日制博士、硕士、本科 |
| **年龄限制** | null |
| **社保要求** | 与用人单位（不含党政机关、事业单位和国有企业）签订2年以上劳动合同且依法缴纳社会保险费（含城镇职工基本养老、医疗、失业保险） |
| **申请条件** | 1. 毕业年度内全日制高校毕业生；2. 来西山区（含度假区）就业创业；3. 与符合要求的用人单位签订2年以上劳动合同；4. 依法缴纳社保；5. 未入住廉租房、公租房、人才公寓；6. 每人只能享受一次 |
| **申请流程** | 1. 准备材料；2. 向西山区公共就业和人才服务中心提交申请；3. 材料审定；4. 官网公示不少于5个工作日；5. 向市人才市场管理办公室报备；6. 发放至本人社保卡/银行卡 |
| **申请平台** | 线下提交至西山区公共就业和人才服务中心 |
| **申请材料** | 1. 《高校毕业生来昆留昆租房补贴申请表》原件2份；2. 劳动合同复印件1份（期限不低于2年）；3. 最近1个月社会保险参保缴费证明原件1份；4. 用人单位营业执照复印件1份；5. 租房协议或租房合同复印件1份（原件查验），承租房业主证明（房产证/不动产权证/购房合同）及业主身份证复印件各1份；6. 本人身份证、毕业证复印件各1份（原件查验，本科需注明是否双一流）；7. 本人具有二代金融功能社保卡复印件1份（需开通激活并写明开户银行） |
| **审核周期** | 公示不少于5个工作日 |
| **发放方式** | 一次性发放至个人社保卡 |
| **生效时间** | 2022年2月7日 |
| **失效时间** | 待核实 |
| **政策状态** | 有效 |
| **官方来源** | https://km.bendibao.com/live/202616/101277.shtm |
| **最后核实日期** | 2026-07-01 |

### 区级政策互斥关系说明

昆明市各区（西山区、呈贡区、盘龙区、官渡区、晋宁区、嵩明县等）的来昆留昆租房补贴、落户补贴等政策，均以毕业生在对应区就业/创业/落户为前提。由于一名毕业生在同一时间只能在一个县（市、区）就业参保并落户，因此各区同类补贴之间为**互斥关系**，系统统一归入 `kunming-district` 互斥组。同一区内的多项补贴（如租房补贴与落户补贴）可同时申请，但不同区的同类补贴不可同时享受。
