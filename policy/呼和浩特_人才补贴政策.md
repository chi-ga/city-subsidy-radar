# 呼和浩特市人才补贴政策

> 最后核实日期：2026-07-01
>
> 政策依据：《关于高质量推进新时代首府人才工作的意见》，由呼和浩特市住建局、人社局发布并执行。

---

## 一、呼和浩特市高校毕业生租房补贴

| 项目 | 内容 |
|------|------|
| 政策名称 | 呼和浩特市高校毕业生租房补贴 |
| 政策文号 | 《关于高质量推进新时代首府人才工作的意见》 |
| 适用对象 | 毕业2年内全日制普通高校毕业生 |
| 补贴金额/标准 | 博士1000元/月、硕士800元/月、本科500元/月、专科300元/月，累计不超过24个月 |
| 学历要求 | 全日制专科、本科、硕士、博士 |
| 年龄限制 | null |
| 社保要求 | 缴纳企业职工养老保险累计3个月及以上 |
| 申请条件 | 1. 毕业2年内全日制普通高校毕业生；2. 在市域内企业就业（签1年以上劳动合同）或自主创业（注册满1年）；3. 缴纳企业职工养老保险累计3个月及以上；4. 在市域内无购房记录且自行租房（须经住建部门备案）；5. 与购房补贴可先后衔接（先租后购），但不可同时享受；6. 专科租房时间须在2023年8月24日以后；7. 国有企业就业毕业生租房时间须在2024年12月26日以后 |
| 申请流程 | 1. 通过"呼和浩特市人才就业服务平台"或"青城人才服务"微信小程序线上申请；2. 次月住建部门和自然资源部门批量核查房屋租赁备案/不动产登记信息；3. 旗县区人社部门审核人才身份（学历、毕业时间、参保信息）；4. 市住建部门公示（5个工作日）；5. 公示无异议后向财政申请拨付资金；6. 市住建部门指定银行发放补贴 |
| 申请平台 | "呼和浩特市人才就业服务平台"网页端或"青城人才服务"微信小程序 |
| 申请材料 | 1. 经房屋所在地旗县区住建部门备案的《房屋租赁登记备案证明》（承租人与申请人一致）；2. 申请人身份证；3. 最高学历毕业证书及《教育部学历证书电子注册备案表》（有效期3个月内可查）或教育部留学服务中心《国外学历学位认证书》；4. 所在企业证照信息（2024年12月26日前申请的须提供企业性质证明）；5. 《呼和浩特市高校毕业生租购房补贴申请表》（线上生成，手写签名+单位负责人签名+加盖单位公章后上传）；6. 社保缴纳单位为人力资源公司的，须提交劳动合同和工资银行流水 |
| 审核周期 | 每月5日至月底受理，每半年集中发放一次 |
| 发放方式 | 按月发放，累计不超过24个月 |
| 生效时间 | 2025-02-12 |
| 失效时间 | null（年度公告，每年更新） |
| 政策状态 | 有效（年度公告，每年更新） |
| 官方来源 | 呼和浩特本地宝（https://m.hu.bendibao.com/news/63581.shtm） |
| 最后核实日期 | 2026-07-01 |

## 二、呼和浩特市高校毕业生购房补贴

| 项目 | 内容 |
|------|------|
| 政策名称 | 呼和浩特市高校毕业生购房补贴 |
| 政策文号 | 《关于高质量推进新时代首府人才工作的意见》 |
| 适用对象 | 毕业5年内全日制普通高校毕业生 |
| 补贴金额/标准 | 博士10万元、硕士5万元、本科3万元、专科2万元，一次性发放 |
| 学历要求 | 全日制专科、本科、硕士、博士 |
| 年龄限制 | null |
| 社保要求 | 缴纳企业职工养老保险累计12个月及以上 |
| 申请条件 | 1. 毕业5年内全日制普通高校毕业生；2. 在市域内企业就业（签1年以上劳动合同）或自主创业（注册满1年）；3. 缴纳企业职工养老保险累计12个月及以上；4. 首次在市域内购房，购房时间在2021年6月22日以后；5. 2人以上购买同一套不动产仅1人可享受（夫妻双方同时符合条件的除外）；6. 与租房补贴可先后衔接，但不可同时享受；7. 专科购房时间须在2023年8月24日以后；8. 国有企业就业毕业生购房时间须在2024年12月26日以后 |
| 申请流程 | 1. 通过"呼和浩特市人才就业服务平台"或"青城人才服务"微信小程序线上申请；2. 次月住建/自然资源部门核查网签备案及不动产登记；3. 旗县区人社部门审核人才身份；4. 市住建部门公示（5个工作日）；5. 财政拨付；6. 指定银行发放（接到通知3个月内领取，逾期视为放弃） |
| 申请平台 | "呼和浩特市人才就业服务平台"网页端或"青城人才服务"微信小程序 |
| 申请材料 | 1. 申请人身份证；2. 最高学历毕业证书及《教育部学历证书电子注册备案表》（有效期3个月内可查）或《国外学历学位认证书》；3. 所在企业证照信息（2024年12月26日前购房的须提供企业性质证明）；4. 《呼和浩特市高校毕业生租购房补贴申请表》（线上生成，手写签名+单位公章）；5. 夫妻双方同时申请的须提供结婚证；6. 社保缴纳单位为人力资源公司的须提交劳动合同和工资银行流水 |
| 审核周期 | 每月5日至月底受理，每半年集中发放一次 |
| 发放方式 | 一次性发放 |
| 生效时间 | 2025-02-12 |
| 失效时间 | null（年度公告，每年更新） |
| 政策状态 | 有效（年度公告，每年更新） |
| 官方来源 | 呼和浩特本地宝（https://m.hu.bendibao.com/news/63599.shtm） |
| 最后核实日期 | 2026-07-01 |
