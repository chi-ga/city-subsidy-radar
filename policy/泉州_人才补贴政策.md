#### 1. 泉州市青年人才"新八条"升级性生活补贴

| 字段 | 内容 |
|------|------|
| **政策名称** | 泉州市青年人才"新八条"升级性生活补贴 |
| **政策文号** | 泉委办发〔2024〕1号 |
| **适用对象** | 2024年2月6日以后首次新引进到泉州市民营企业和民办非企业单位工作或自主创业的博士研究生、硕士研究生、全日制"双一流"高校本科生（全球前200名高校〈QS〉学士参照执行） |
| **补贴金额/标准** | 博士研究生10万元、硕士研究生5万元、全日制"双一流"高校本科生2万元 |
| **学历要求** | 博士、硕士、全日制"双一流"高校本科 |
| **年龄限制** | 申请时年龄不超过40周岁 |
| **社保要求** | 在泉州市同一单位参保（企业职工基本养老保险）连续缴费至少3个月，且申请时仍在岗 |
| **申请条件** | 1. 首次在泉州市民营企业、民办非企业单位就业或自主创业；2. 按规定在同一单位缴纳城镇职工养老保险3个月以上；3. 申请时仍在岗；4. 符合条件的申请人应于首次引进到泉州市一年内（以参保缴费记录为准）线上申报，逾期视为放弃申请 |
| **申请流程** | 1. 登录"海丝泉州·人才港湾"网站或微信小程序进入"青年人才新八条"模块注册申报；2. 用人单位系统初审；3. 各县（市、区）公共就业服务机构会同社保经办机构数据审核；4. 公示5个工作日；5. 补贴发放至申请人社会保障卡银行账户 |
| **申请平台** | 海丝泉州·人才港湾网站（https://www.qzrcgw.gov.cn）或微信小程序 |
| **申请材料** | 1. 身份证明（身份证或公安部门出具的户籍证明）；2. 学历证明（毕业证书、带有二维码的《教育部学历证书电子注册备案表》，国境外学历需经教育部留学服务中心认证）；3. 工作证明（劳动合同或劳动协议等就业证明材料及缴交社保证明；自主创业的提供工商营业执照或公司章程等创业证明材料及缴交社保证明） |
| **审核周期** | 公示5个工作日 |
| **发放方式** | 分两次发放：申请审核通过后的第一年发放50%（发放时需在泉参保），第三年根据其就业创业情况发放另外50%（发放时需三年内均在泉连续参保且仍在泉州工作或自主创业） |
| **生效时间** | 2024-02-06 |
| **失效时间** | 2026-02-05（文件有效期2年） |
| **政策状态** | 有效。按"从新从优不重复"原则，不重复享受泉委办发〔2022〕5号文规定的每人1万元标准的一次性生活补贴；若同时认定为泉州市引进类高层次人才，按"就高从优不重复"原则选择申领 |
| **官方来源** | https://www.quanzhou.gov.cn/xsdmyjjfwzq/zqztfwrk/jjjy/zcdlb/zc/ycyc/202402/t20240216_3005068.htm |
| **最后核实日期** | 2026-06-30 |

#### 2. 泉州市青年人才"新八条"购房补助

| 字段 | 内容 |
|------|------|
| **政策名称** | 泉州市青年人才"新八条"购房补助 |
| **政策文号** | 泉委办发〔2024〕1号 |
| **适用对象** | 2024年2月6日以后首次新引进到泉州市民营企业和民办非企业单位工作或自主创业的博士研究生、硕士研究生、全日制"双一流"高校本科生、普通高校本科生、大专生，技工学院全日制预备技师（技师）班、高级工班毕业生 |
| **补贴金额/标准** | 博士研究生8万元、硕士研究生6万元、全日制"双一流"高校本科生4万元、普通高校本科生2万元、大专生1万元；技工学院全日制预备技师（技师）班、高级工班毕业生分别按相当于本科学历、大专学历落实 |

| **年龄限制** | 申请时年龄不超过40周岁 |
| **社保要求** | 在泉州市同一单位参保（企业职工基本养老保险）连续缴费至少3个月，且申请时仍在岗 |
| **申请条件** | 1. 首次在泉州市民营企业、民办非企业单位就业或自主创业；2. 在同一单位缴纳社保3个月以上且申请时仍在岗；3. 本人、配偶及未成年子女在泉州无住房，未享受其他住房优惠政策；4. 符合条件的申请人应于首次购买商品住房后一年内线上申报，逾期视为放弃申请 |
| **申请流程** | 1. 登录"海丝泉州·人才港湾"网站或微信小程序进入"青年人才新八条"模块注册申报；2. 用人单位系统初审；3. 购房税费受益县（市、区）人社部门资格审核，住建、资源规划部门核查家庭住房情况；4. 公示5个工作日；5. 补贴发放至申请人社会保障卡银行账户 |
| **申请平台** | 海丝泉州·人才港湾网站（https://www.qzrcgw.gov.cn）或微信小程序 |

| **发放方式** | 分两次发放：申请审核通过后的第一年发放50%（发放时需在泉参保），第三年根据其就业创业情况发放另外50%（发放时需三年内均在泉连续参保且仍在泉州工作或自主创业） |
| **生效时间** | 2024-02-06 |
| **失效时间** | 2026-02-05（文件有效期2年） |
| **政策状态** | 有效。购房补助由市财政和购房税费受益县（市、区）按3:7比例承担 |
| **官方来源** | https://www.quanzhou.gov.cn/xsdmyjjfwzq/zqztfwrk/jjjy/zcdlb/zc/ycyc/202402/t20240216_3005068.htm |
| **最后核实日期** | 2026-06-30 |

### 各区通用"涌泉行动"政策说明

> 最后核实日期：2026-06-30
> 数据来源：泉州市人社局（泉人社文〔2022〕164号）、泉州市人民政府政务指南

**互斥性说明：** 以下"涌泉行动"一次性生活补贴和社保补助由市级政策统一规定、各区分别执行。申请人须在泉州某一县（市、区）的民营企业、民办非企业单位就业或自主创业，并落户该县（市、区）后向所在区申请。**同一申请人只能在一个区申领，各区之间为互斥关系。** 鲤城区、丰泽区、洛江区、泉港区、石狮市、晋江市、南安市、惠安县、安溪县、永春县、德化县、泉州开发区、泉州台商投资区等均按此标准执行（个别区有加码政策，见下文）。

> 数据来源：惠安县人民政府、泉州本地宝

---

#### 1. 惠安县高校毕业生一次性生活补贴

| 字段 | 内容 |
|------|------|
| **政策名称** | 惠安县高校毕业生一次性生活补贴 |
| **政策文号** | 泉人社文〔2022〕164号 |
| **适用对象** | 在惠安县民营企业、民办非企业单位稳定就业或自主创业的全日制本科及以上应届高校毕业生和技工院校预备技师（技师）班及以上毕业生（2022年及以后毕业） |
| **补贴金额/标准** | 每人1万元，一次性发放；属困难家庭高校毕业生的，按每人1.2万元标准发放 |
| **学历要求** | 全日制本科及以上 |
| **年龄限制** | null |
| **社保要求** | 在惠安县以灵活就业人员身份参加城镇职工养老保险连续缴费3个月及以上（就业单位参保要求应在同一单位参保连续缴费3个月及以上） |
| **申请条件** | 1. 已具有惠安县户口；2. 毕业后在惠安县民营企业、民办非企业单位就业或个人创业，并参加城镇职工养老保险连续缴费3个月及以上；3. 毕业一年内（以毕业证书签发时间起算）申报 |
| **申请流程** | 1. 下载"泉房通"APP；2. 在线填写信息并上传材料；3. 数据审核；4. 公示；5. 一次性发放 |
| **申请平台** | 泉州市城建集团"泉房通"APP |
| **申请材料** | 1. 身份证明；2. 学历证明（毕业证书、教育部学历证书电子注册备案表）；3. 工作证明（劳动合同或营业执照）及缴交社保证明 |
| **审核周期** | 公示5个工作日 |
| **发放方式** | 一次性发放 |
| **生效时间** | 2022-06-30 |
| **失效时间** | 2025-06-30（泉人社文〔2022〕164号有效期三年） |
| **政策状态** | 有效。困难家庭高校毕业生按1.2万元标准发放 |
| **官方来源** | http://m.qz.bendibao.com/mip/21082.shtm |
| **最后核实日期** | 2026-06-30 |

#### 2. 惠安县高校毕业生社保补助

| 字段 | 内容 |
|------|------|
| **政策名称** | 惠安县高校毕业生社保补助 |
| **政策文号** | 泉人社文〔2022〕164号 |
| **适用对象** | 首次在惠安县民营企业、民办非企业单位稳定就业或自主创业的全日制普通高等学校本科及以上应届毕业生和技工院校全日制预备技师（技师）班应届毕业生（2022年及之后毕业） |
| **补贴金额/标准** | 个人缴纳城镇职工养老保险部分予以补助，每人每月最高不超过600元，最多发放24个月 |
| **学历要求** | 全日制本科及以上 |
| **年龄限制** | null |
| **社保要求** | 在惠安县同一单位参保连续缴费（企业职工基本养老保险）至少3个月，且申请时仍在岗 |
| **申请条件** | 1. 已具有惠安县户口；2. 毕业后首次在惠安县民营企业、民办非企业单位就业或自主创业；3. 同一单位社保连续缴费至少3个月且申请时仍在岗；4. 毕业一年内（以毕业证书签发时间起算）申报 |
| **申请流程** | 1. 下载"泉房通"APP；2. 在线填写信息并上传材料；3. 数据审核；4. 公示；5. 按月发放 |
| **申请平台** | 泉州市城建集团"泉房通"APP |
| **申请材料** | 1. 身份证明；2. 学历证明（毕业证书、教育部学历证书电子注册备案表）；3. 工作证明（劳动合同或营业执照）及缴交社保证明 |
| **审核周期** | 公示5个工作日 |
| **发放方式** | 按月发放，最多24个月 |
| **生效时间** | 2022-06-30 |
| **失效时间** | 2025-06-30（泉人社文〔2022〕164号有效期三年） |
| **政策状态** | 有效。不与泉政办〔2021〕1号等规定重复享受 |
| **官方来源** | http://m.qz.bendibao.com/mip/21082.shtm |
| **最后核实日期** | 2026-06-30 |

#### 1. 泉港区"涌泉行动·人才进港"生活补贴

| 字段 | 内容 |
|------|------|
| **政策名称** | 泉港区"涌泉行动·人才进港"生活补贴 |
| **政策文号** | 待核实（泉港区"涌泉行动·人才进港"系列引才政策） |
| **适用对象** | 首次新引进到泉港区企业就业或自主创业的大专及以上学历毕业生 |
| **补贴金额/标准** | 大专毕业生每人5000元，本科及以上学历毕业生每人1万元，一次性发放 |
| **学历要求** | 大专及以上学历 |
| **年龄限制** | null |
| **社保要求** | 在泉港区企业就业或自主创业并缴纳社保 |
| **申请条件** | 1. 首次新引进到泉港区企业就业或自主创业；2. 按规定缴纳社保；3. 具体申报时间以泉港区通知为准 |
| **申请流程** | 1. 登录"泉州人才港湾网"或相关申报平台；2. 在线填写信息并上传材料；3. 审核；4. 公示；5. 一次性发放 |
| **申请平台** | 泉州人才港湾网站/微信小程序 |
| **申请材料** | 1. 身份证明；2. 学历证明；3. 工作证明及社保证明（具体以平台要求为准） |
| **审核周期** | null |
| **发放方式** | 一次性发放 |
| **生效时间** | 2023年（泉港区"涌泉行动·人才进港"系列政策出台时间） |
| **失效时间** | 待核实 |
| **政策状态** | 有效。泉港区在市级政策基础上对大专毕业生额外给予5000元补贴 |
| **官方来源** | http://m.toutiao.com/group/7140096411999224331/ |
| **最后核实日期** | 2026-06-30 |
