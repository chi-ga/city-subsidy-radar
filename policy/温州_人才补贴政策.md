#### 1. 温州市新全职来温博士生活补贴

| 字段 | 内容 |
|------|------|
| **政策名称** | 温州市新全职来温博士生活补贴 |
| **政策文号** | 温人社发〔2025〕50号 |
| **适用对象** | 新全职来温工作的40周岁以下博士研究生 |
| **补贴金额/标准** | 8万元，分4次发放，依法缴纳社保月数累计达12、24、36、48个月后分别拨付总额的25% |
| **学历要求** | 博士 |
| **年龄限制** | 40周岁以下 |
| **社保要求** | 依法连续缴纳社会保险费 |
| **申请条件** | 1. 新全职来温工作；2. 年龄在40周岁以下；3. 在温依法缴纳社保；4. 国内应为全日制高校毕业生，国（境）外高校毕业生学历需经教育部留学服务中心认证 |
| **申请流程** | 登录温州市产业政策奖励兑现系统或“温州人才云服务平台”→搜索“就业补贴”→按提示填报并上传材料→单位审核→人社部门审核拨付 |
| **申请平台** | 温州市产业政策奖励兑现系统（reward.wenzhou.gov.cn）、“温州人才”微信公众号-“人才云” |
| **申请材料** | 1. 高校毕业生就业补贴申请表；2. 单位年报等证明企业划型的材料；3. 高等学校毕业证书；4. 由人力资源服务机构派遣就业的，需提供派遣协议和劳动合同；5. 学历与系统自动读取不一致的，需上传毕业证书和学信网学历证明或教育部留学服务中心国（境）外学历学位认证；6. 其他证明材料 |
| **审核周期** | 30个工作日 |
| **发放方式** | 分4次发放，社保累计达12、24、36、48个月后各拨付25% |
| **生效时间** | 2025-07-01 |
| **失效时间** | 2027-12-31（510+行动方案执行期） |
| **政策状态** | 有效。属“510+行动计划”（2025-2027年）生活和就业补贴条款 |
| **官方来源** | https://www.wenzhou.gov.cn/art/2025/6/18/art_1229118497_2039500.html |
| **最后核实日期** | 2026-06-30 |

#### 2. 温州市高校毕业生就业/生活补贴

| 字段 | 内容 |
|------|------|
| **政策名称** | 温州市高校毕业生就业/生活补贴 |
| **政策文号** | 温人社发〔2025〕50号 |
| **适用对象** | 毕业2年内新来温中小微企业就业的全日制硕士、本科、专科毕业生 |
| **补贴金额/标准** | 硕士4万元、本科1万元、专科0.5万元；硕士分4次发放，本科/专科分2期发放 |
| **学历要求** | 全日制硕士、本科、专科 |
| **年龄限制** | null |
| **社保要求** | 与单位签订1年及以上劳动合同，依法连续缴纳社会保险费满12个月 |
| **申请条件** | 1. 毕业2年内；2. 新来温中小微企业就业；3. 签订1年及以上劳动合同；4. 依法连续缴纳社保满12个月；5. 工资收入低于全省上年度非私营和私营单位就业人员加权平均工资；6. 国内全日制高校毕业生，国（境）外高校毕业生学历需经教育部留学服务中心认证 |
| **申请流程** | 登录浙江政务服务网或浙里办APP→搜索“高校毕业生就业补贴申领”→根据社保参保地选择辖区→按提示填报并上传材料→经办机构审核→拨付 |
| **申请平台** | 浙江政务服务网（http://www.zjzwfw.gov.cn/）或浙里办APP |
| **申请材料** | 1. 高校毕业生就业补贴申请表；2. 单位年报等证明企业划型的材料；3. 高等学校毕业证书；4. 由人力资源服务机构派遣就业的，需提供派遣协议和劳动合同；5. 学历与系统自动读取不一致的，需上传毕业证书和学信网学历证明或教育部留学服务中心国（境）外学历学位认证；6. 其他证明材料 |
| **审核周期** | 30个工作日 |
| **发放方式** | 硕士分4次发放（社保累计达12、24、36、48个月各拨付25%）；本科/专科分2期发放（每满12个月拨付50%） |
| **生效时间** | 2025-07-01 |
| **失效时间** | 2027-12-31 |
| **政策状态** | 有效。属“510+行动计划”（2025-2027年）生活和就业补贴条款 |
| **官方来源** | https://www.wenzhou.gov.cn/col/col1229892247/ |
| **最后核实日期** | 2026-06-30 |

#### 3. 温州市人才租房补贴

| 字段 | 内容 |
|------|------|
| **政策名称** | 温州市人才租房补贴 |
| **政策文号** | 温住建发〔2025〕20号 |
| **适用对象** | 新全职来温企业或高校、科研院所工作的ABCDEF类人才 |
| **补贴金额/标准** | F类最高1万元/年，DE类最高2万元/年，BC类最高4万元/年，A类一事一议；累计不超过5年 |
| **学历要求** | 需先认定为温州市ABCDEF类人才（F3类对应全日制本科、F2类对应全日制硕士、E类对应博士等） |
| **年龄限制** | null |
| **社保要求** | 已与用人单位签订3年（含）以上全职聘用合同，到申请之日已以申请单位为主体连续缴纳社保12个月以上 |
| **申请条件** | 1. 新全职来温工作（2025年7月1日以后，以在温初次社保缴纳时间为准）；2. 属于温州市ABCDEF类人才；3. 签订3年及以上全职聘用合同；4. 连续缴纳社保12个月以上；5. 本人及配偶在温州市区/隶属地无住房；6. 截至申请之日已在市区/隶属地连续租房满12个月以上；7. 未享受过温州市政府住房优惠政策 |
| **申请流程** | 登录“温州人才之家-租房补贴”或浙里办APP-“浙里人才之家-租房补贴”→填报信息并上传附件→用人单位审核公示→职能部门资格审核→准入登记→按发票金额据实发放 |
| **申请平台** | 温州人才之家（https://reward.wenzhou.gov.cn/wzrc/）、浙里办APP-浙里人才之家 |
| **申请材料** | 1. 申请人及配偶身份证（护照）；2. 3年以上全职聘用合同和相应证明材料；3. 连续缴纳社保12个月以上证明；4. 个人同意核查书；5. 婚姻状况材料；6. 连续满12个月的房屋租赁合同；7. 房屋出租数电普票或增值税普通发票；8. F类人才需上传毕业证书、积分材料、年薪金、职称等相应材料 |
| **审核周期** | 职能部门每月25日前完成前一个月申请件的资格审核 |
| **发放方式** | 每连续满12个月申领一次，累计不超过5次，按住房租赁税务发票载明金额据实核发 |
| **生效时间** | 2025-07-01 |
| **失效时间** | 待核实 |
| **政策状态** | 有效。2025年7月1日起施行，原温住建发〔2022〕95号同步修订 |
| **官方来源** | https://zjj.wenzhou.gov.cn/art/2025/6/30/art_1229129350_2039652.html |
| **最后核实日期** | 2026-06-30 |

**区级政策互斥说明：** 龙湾区就业补贴与鹿城区租房补贴属于不同行政区的引才政策，申请人通常只能在一个区就业/落户并享受该区对应补贴，因此同一城市的各区级补贴构成互斥关系，系统按“温州区级补贴（只能在一个区享受）”互斥组处理，金额取最高值。
