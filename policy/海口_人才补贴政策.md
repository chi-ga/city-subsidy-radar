#### 1. 海口市引进人才住房租赁补贴（租房补贴）

| 字段 | 内容 |
|------|------|
| **政策名称** | 海口市引进人才住房租赁补贴 |
| **政策文号** | 《海口市引进人才住房保障实施细则》（依据《中共海南省委办公厅 海南省人民政府办公厅关于引进人才住房保障的指导意见》琼办发〔2018〕40号） |
| **适用对象** | 2018年5月13日后第一次在海口就业或自主创业的引进人才，含40岁以下全日制硕士、35岁以下全日制本科毕业生，以及50岁以下高层次D/E类人才、35岁以下专技人才等 |
| **补贴金额/标准** | 租房补贴按月发放：博士3000元/月；硕士2000元/月；本科1500元/月。累计发放不超过36个月 |
| **学历要求** | 全日制本科、硕士、博士（含国（境）外高校毕业生）；另含具有中级职称/技师/执业医师等专技人才 |
| **年龄限制** | 硕士40周岁以下（含）、本科35周岁以下（含）；高层次D/E类人才50周岁以下（急需紧缺可放宽至55周岁） |
| **社保要求** | 人才引进时间以2018年5月13日后第一次在海南省缴纳个人所得税或社会保险时间为准；须在海口就业或自主创业 |
| **申请条件** | 1.2018年5月13日后第一次在海口就业或自主创业；2.符合五类人才之一（高层次D/E类、全日制硕士≤40、全日制本科≤35、35岁以下专技人才、总部企业高管等）；3.不含公务员及参公管理人员；4.累计发放不超过36个月，须在引进之日起5年内申领完毕 |
| **申请流程** | 个人完善信息→用人单位关联员工并公示→单位通过海南政务服务网提交申报→各区行政审批服务局受理审核→公示→按月发放 |
| **申请平台** | 海南政务服务网/海口市政务服务网（https://wssp.haikou.gov.cn/） |
| **申请材料** | 初次申领个人材料：1.人才类型证明材料（学历学位证书、职称证书、职业资格证书等；国（境）外高校毕业生需教育部留学生服务中心学历认证书）；2.住房证明材料（申请租房补贴提供《房屋租赁证》）；3.劳动关系材料（劳动/聘用合同，自主创业提供营业执照）；4.引进人才住房补贴个人承诺书；5.其他需要补充说明情况的材料。初次申领单位材料：1.引进人才住房补贴单位承诺书；2.事业单位法人证书或编制方案（公益一类、二类事业单位提供）；3.单位公示情况有关材料；4.其他需要补充说明情况的材料 |
| **审核周期** | 待核实（各区行政审批服务局受理） |
| **发放方式** | 按月发放，累计不超过36个月 |
| **生效时间** | 2018-05-13 |
| **失效时间** | 待核实（现行有效） |
| **政策状态** | 有效。博士3000元/月、硕士2000元/月、本科1500元/月，累计不超过36个月，须在引进之日起5年内申领完毕。已领取购房补贴的不再发放租房补贴 |
| **官方来源** | http://haikou.bendibao.com/live/2021324/49150.shtm |
| **最后核实日期** | 2026-07-01 |

#### 2. 海口市引进人才购房补贴

| 字段 | 内容 |
|------|------|
| **政策名称** | 海口市引进人才购房补贴 |
| **政策文号** | 《海口市引进人才住房保障实施细则》（依据《中共海南省委办公厅 海南省人民政府办公厅关于引进人才住房保障的指导意见》琼办发〔2018〕40号） |
| **适用对象** | 2018年5月13日后第一次在海口就业或自主创业的引进人才，含40岁以下全日制硕士、35岁以下全日制本科毕业生，以及50岁以下高层次D/E类人才、35岁以下专技人才等 |
| **补贴金额/标准** | 购房补贴按年集中发放：博士3.6万元/年；硕士2.4万元/年；本科1.8万元/年。累计发放不超过3年（合计博士10.8万元、硕士7.2万元、本科5.4万元） |
| **学历要求** | 全日制本科、硕士、博士（含国（境）外高校毕业生）；另含具有中级职称/技师/执业医师等专技人才 |
| **年龄限制** | 硕士40周岁以下（含）、本科35周岁以下（含）；高层次D/E类人才50周岁以下（急需紧缺可放宽至55周岁） |
| **社保要求** | 人才引进时间以2018年5月13日后第一次在海南省缴纳个人所得税或社会保险时间为准；须在海口就业或自主创业 |
| **申请条件** | 1.2018年5月13日后第一次在海口就业或自主创业；2.符合五类人才之一；3.不含公务员及参公管理人员；4.在海口购买商品住房；5.累计发放不超过3年，须在引进之日起5年内申领完毕；6.已领取购房补贴的不再发放住房租赁补贴，已领租房补贴的应扣除已领金额 |
| **申请流程** | 个人完善信息→用人单位关联员工并公示→单位通过海南政务服务网提交申报→各区行政审批服务局受理审核→公示→按年集中受理和发放 |
| **申请平台** | 海南政务服务网/海口市政务服务网（https://wssp.haikou.gov.cn/） |
| **申请材料** | 初次申领个人材料：1.人才类型证明材料（学历学位证书、职称证书、职业资格证书等；国（境）外高校毕业生需教育部留学生服务中心学历认证书）；2.住房证明材料（申请购房补贴提供《海口市商品房买卖合同备案表》或《海口市房屋交易和产权状况确认书（转让）》）；3.劳动关系材料（劳动/聘用合同，自主创业提供营业执照）；4.引进人才住房补贴个人承诺书；5.其他需要补充说明情况的材料。初次申领单位材料：1.引进人才住房补贴单位承诺书；2.事业单位法人证书或编制方案（公益一类、二类事业单位提供）；3.单位公示情况有关材料；4.其他需要补充说明情况的材料 |
| **审核周期** | 待核实（每年集中受理和发放一次） |
| **发放方式** | 按年集中发放，累计不超过3年 |
| **生效时间** | 2018-05-13 |
| **失效时间** | 待核实（现行有效） |
| **政策状态** | 有效。博士3.6万/年、硕士2.4万/年、本科1.8万/年，累计不超过3年。与租房补贴二选一，已领租房补贴的购房补贴应扣除已领金额 |
| **官方来源** | http://haikou.bendibao.com/live/2021324/49150.shtm |
| **最后核实日期** | 2026-07-01 |
