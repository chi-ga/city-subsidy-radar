# 镇江人才补贴政策

> 最后核实日期：2026-07-01
>
> 政策依据：《镇江市区高校毕业生租房券实施细则》（镇江市住建局等五部门，2025年9月29日）；《镇江市市区人才购房补贴实施细则》（镇江市住建局）。

---

## 一、镇江市区高校毕业生租房券

| 项目 | 内容 |
|------|------|
| 政策名称 | 镇江市区高校毕业生租房券 |
| 政策文号 | 《镇江市区高校毕业生租房券实施细则》（镇江市住建局、市委人才办、市人社局、市财政局、市自然资源和规划局，2025年9月29日发布） |
| 适用对象 | 普通（含在国外、境外取得国家教育部门认可学历）的高校毕业生及职业院校（含技工院校）毕业生 |
| 补贴金额/标准 | 全日制大专、职业院校（含技工院校）高级工毕业生500元/月；全日制本科、职业院校（含技工院校）技师毕业生900元/月；全日制硕士研究生1200元/月；全日制博士研究生1500元/月 |
| 学历要求 | 全日制大专及以上学历（含技工院校高级工/技师） |
| 年龄限制 | null |
| 社保要求 | 在市区按时足额缴纳城镇企业职工基本养老保险，且申报时在该企业处于缴纳状态 |
| 申请条件 | 1.具有普通高校全日制大专及以上学历，或在职业院校（含技工院校）毕业获得技师职业资格（技能等级二级以上）、高级工职业资格（技能等级三级）；2.毕业2年内（以毕业证书登记时间为准）来市区企业就业；3.签订劳动合同，并在市区按时足额缴纳城镇企业职工基本养老保险，且申报时在该企业处于缴纳状态；4.市区个人名下无住房 |
| 申请流程 | 1.首次申报由毕业生所在企业集中申报，企业于每季度首月第二周工作日到市住建局行政服务中心窗口申报上季度新增申请；2.市住建局行政服务中心窗口核查材料齐备后正式受理，转人社部门和不动产登记部门审核；3.首次申报审核通过后在市住建局网站公示5个工作日，公示无异议后确定最终审核结果并公告；4.毕业生在人才公寓目录库中选择人才公寓租赁，缴纳费用时抵用房租（月度抵用金额不得高于月实际房租，抵用时间在发放月份后半年内）；5.次年3月市住建局会同相关部门联合会审，确定兑付金额；次年4月向市级财政申请资金结算后拨付运营单位 |
| 申请平台 | 镇江市住建局行政服务中心窗口（电话：85589792） |
| 申请材料 | 1.企业营业执照复印件；2.申报人身份证复印件；3.劳动合同复印件；4.毕业证书（技职院校毕业生需同时提供职业资格证书）原件与复印件；5.企业加盖公章的租房券申报情况一览表原件及电子版；6.申请毕业生签名的告知书等材料原件 |
| 审核周期 | 每季度集中审核，首次申报公示5个工作日 |
| 发放方式 | 租房券抵用房租，在人才公寓目录库中选择人才公寓租赁时抵用相应房租，发放期限不超过3年 |
| 生效时间 | 2025-09-29 |
| 失效时间 | null |
| 政策状态 | 有效（2025年9月29日起执行）。租房券与人才"镇兴"行动等人才政策中的租房补贴不重复享受。适用范围：丹徒区、京口区、润州区、镇江经开区和镇江高新区 |
| 官方来源 | 镇江市住房和城乡建设局（https://jsj.zhenjiang.gov.cn/jsj/bszn/202509/a9b01df7e9644afcb936acd59ebffb0e.shtml） |
| 最后核实日期 | 2026-07-01 |

## 二、镇江市区人才购房补贴

| 项目 | 内容 |
|------|------|
| 政策名称 | 镇江市区人才购房补贴 |
| 政策文号 | 《镇江市市区人才购房补贴实施细则》（镇江市住建局） |
| 适用对象 | 初次来镇就业创业人才，在市区购买首套自住商品住房。E类人才中的"双一流"高校、在镇高校全日制博士；F类人才中的"双一流"高校、在镇高校全日制硕士；出国（境）留学取得国家教育部门认可的全日制博士、硕士学历且在校学习期间所在院校为世界综合大学排名前500名高校的毕业生。A-D类高层次人才购房补贴15万-100万 |
| 补贴金额/标准 | E类（双一流/世界500强/在镇高校全日制博士）按所购房屋价格份额的30%给予补贴，最高20万元；F类（双一流/世界500强/在镇高校全日制硕士）最高15万元。分3年发放（第一年40%、第二年30%、第三年30%） |
| 学历要求 | 全日制硕士研究生、博士研究生（限"双一流"高校、在镇高校或世界综合大学排名前500名海外高校） |
| 年龄限制 | null |
| 社保要求 | 与企业签订一年以上劳动合同并缴纳城镇企业职工基本养老保险；与事业单位签订聘用合同并缴纳机关事业单位养老保险 |
| 申请条件 | 1.需通过E类或F类人才认定；2.E/F类人才须为"双一流"高校、在镇高校全日制博士/硕士，或世界500强海外高校全日制博士/硕士；3.初次来镇就业创业；4.在市区企业就业（签订一年以上劳动合同并缴纳城镇企业职工基本养老保险）；5.在市区购买首套自住商品住房；6.A-D类人才须2024年1月1日后初次来镇就业创业 |
| 申请流程 | 1.人才所在单位于每年8月至9月登录"镇合意"网上办事服务大厅线上填写基本信息；2.上传企业营业执照/事业单位法人登记证、人才身份证、房屋买卖合同、契税完税凭证、不动产权证书、劳动/聘用合同等材料；3.相关部门审核人才资格、社保缴纳情况及房产情况；4.审核通过后公示；5.按第一年40%、第二年30%、第三年30%的比例分3年发放购房补贴 |
| 申请平台 | "镇合意"网上办事服务大厅 |
| 申请材料 | 1.企业营业执照或事业单位法人登记证；2.人才本人身份证；3.房屋买卖合同；4.契税完税凭证；5.不动产权证书；6.劳动或聘用合同 |
| 审核周期 | 每年8-9月集中申报，联审后公示发放 |
| 发放方式 | 分3年发放（第一年40%、第二年30%、第三年30%） |
| 生效时间 | 待核实 |
| 失效时间 | null（2026年新政预计三季度发布，过渡期政策状态待官方明确） |
| 政策状态 | 待确认。该政策需E/F类人才认定（双一流/世界500强/在镇高校全日制博士/硕士），非纯普惠政策。原政策框架下购房补贴预登记通道曾开放，2026年新人才补贴政策预计三季度发布，请关注官方通知 |
| 官方来源 | 镇江本地宝（http://m.zj.bendibao.com/live/14308.shtm） |
| 最后核实日期 | 2026-07-01 |
