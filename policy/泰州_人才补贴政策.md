# 泰州人才补贴政策

> 最后核实日期：2026-07-01
>
> 政策依据：《泰州市区人才购房券发放及兑付实施细则》（泰人才办〔2023〕11号），由泰州市人才工作领导小组办公室、泰州市人力资源和社会保障局发布执行；租房补贴依据泰州市人才公共服务平台现行政策。区级（县级市）政策依据靖江市《关于加快集聚"马洲英才"支撑高质量发展的十条激励措施》。

## 一、泰州市区高校毕业生租房补贴

| 项目 | 内容 |
|------|------|
| 政策名称 | 泰州市区高校毕业生租房补贴 |
| 政策文号 | 泰州市人才公共服务平台现行政策（凤城英才计划配套） |
| 适用对象 | 来泰州市实体经济企业就业或创业的应届高校毕业生，在工作所在地无房并租房居住 |
| 补贴金额/标准 | 大专毕业生500元/月；本科毕业生（或取得高级工以上职业资格证书的大专毕业生）800元/月；硕士研究生1500元/月；博士研究生2000元/月；享受期最长36个月 |
| 学历要求 | 大专及以上 |
| 年龄限制 | null（应届高校毕业生） |
| 社保要求 | 在泰实体经济企业就业或创业，依法缴纳社会保险 |
| 申请条件 | 1.应届高校毕业生；2.在泰州市实体经济企业就业或创业；3.在工作所在地无房并租房居住；4.依法缴纳社会保险 |
| 申请流程 | 1.登录泰州人才公共服务平台（https://tac.rsj.taizhou.gov.cn）；2.进入租房补贴申请；3.提交个人信息及相关材料；4.人社部门审核；5.审核通过后按月发放 |
| 申请平台 | 泰州人才公共服务平台（https://tac.rsj.taizhou.gov.cn） |
| 申请材料 | 1.本人身份证；2.毕业证书；3.劳动合同；4.社保缴费证明；5.房屋租赁合同。具体材料以申报系统提示为准 |
| 审核周期 | 人社部门审核后发放 |
| 发放方式 | 按月发放，最长36个月 |
| 生效时间 | null（现行政策持续执行） |
| 失效时间 | null |
| 政策状态 | 有效。凤城英才计划配套租房补贴持续执行中 |
| 官方来源 | 泰州人才公共服务平台（https://tac.rsj.taizhou.gov.cn/guide/413739e1005a42cfbeef7eaa065d2c38）；泰州本地宝（https://m.taizhou.bendibao.com/live/18305.shtm） |
| 最后核实日期 | 2026-07-01 |

## 二、泰州市区人才购房券

| 项目 | 内容 |
|------|------|
| 政策名称 | 泰州市区人才购房券（青年人才） |
| 政策文号 | 《泰州市区人才购房券发放及兑付实施细则》（泰人才办〔2023〕11号） |
| 适用对象 | 首次来泰并在市区范围内各类企业、民办非企业单位、社会团体和基金会就业或创业的青年人才 |
| 补贴金额/标准 | 各类企业就业：博士研究生（正高职称、特级技师）20万元；硕士研究生（副高职称、高级技师）10万元；本科生（中级职称、技师）6万元；大专生（高级工）3万元。实体企业（金融、通讯、烟草、供电、盐业等行业企业除外）就业提高至：博士30万元、硕士15万元、本科8万元；一次性购房券 |
| 学历要求 | 全日制普通高校大专及以上 |
| 年龄限制 | 兑付购房券补贴时，青年人才年龄不超过40周岁 |
| 社保要求 | 就业人才须与企业签订1年以上劳动合同，并在泰连续缴纳社会保险满3个月；创业人才所创办企业须在泰依法缴纳社会保险满3个月 |
| 申请条件 | 1.首次来泰并在市区范围内各类企业、民办非企业单位、社会团体和基金会就业或创业；2.全日制普通高校大专及以上学历；3.签订1年以上劳动合同并在泰连续缴纳社保满3个月（创业人才企业缴纳社保满3个月）；4.兑付时年龄不超过40周岁；5.在泰州市区购买商品住宅房并取得不动产权证，购房时间不得早于在泰首次缴纳社保起始时间 |
| 申请流程 | 1.登录泰州人才公共服务平台进行人才类别认证；2.通过认证后申请购房券；3.人社部门审核发放购房券；4.购房时持购房券至住建部门办理购房合同备案冻结；5.取得不动产权证后向人社部门申请兑付 |
| 申请平台 | 泰州人才公共服务平台（https://tac.rsj.taizhou.gov.cn）；泰州市人才服务中心（泰州市洪泽湖路66号，咨询电话86883116） |
| 申请材料 | 1.本人身份证；2.毕业证书（学位证书）；3.劳动合同；4.社保缴费证明；5.商品房买卖合同；6.不动产权证书；7.契税完税证明。具体材料以申报系统提示为准 |
| 审核周期 | 人才类别认证及购房券审核后发放，兑付审核后拨付 |
| 发放方式 | 一次性购房券（兑付时抵扣购房款） |
| 生效时间 | 2023-01-01（泰人才办〔2023〕11号） |
| 失效时间 | null |
| 政策状态 | 有效。泰州市区人才购房券政策持续执行中 |
| 官方来源 | 泰州本地宝（https://m.taizhou.bendibao.com/live/15953.shtm）；泰州人才公共服务平台（https://tac.rsj.taizhou.gov.cn/guide/1edb1faea14541fbad5d69ce84863c68） |
| 最后核实日期 | 2026-07-01 |

## 靖江市

> 靖江市为泰州市代管县级市，执行独立的"马洲英才"人才购房补贴政策，与泰州市区购房券互斥（人才只能在工作所在地享受一项购房补贴）。

### 区级政策互斥关系说明

泰州市区（海陵区、高港区、姜堰区、医药高新区）执行统一的市级人才购房券和租房补贴标准。靖江市作为代管县级市执行独立的"马洲英才"购房补贴政策，与泰州市区购房券属于互斥关系——人才只能在工作所在地（市区或靖江市）享受一项购房补贴。姜堰区另有本科生生活补贴等区级个性化政策（须在姜堰工业企业就业），具体标准请咨询姜堰区人社局。市级租房补贴与购房券为不同类型可分别申请。
