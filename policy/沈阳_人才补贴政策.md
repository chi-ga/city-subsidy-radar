#### 1. 沈阳市吸引人才就业创业租房补贴

| 字段 | 内容 |
|------|------|
| **政策名称** | 沈阳市吸引人才就业创业租房补贴 |
| **政策文号** | 沈人社发〔2018〕78号《沈阳市吸引人才就业创业租房补贴实施细则》 |
| **适用对象** | 原籍非沈阳市、首次来沈就业创业的高校毕业生及技能人才。分为两类：（1）2018年1月1日以后落户沈阳、首次来沈就业创业的全日制博士、硕士（＜35周岁）、本科（＜35周岁）及技师以上（含技师）技能人才（＜35周岁）；（2）2015年1月1日后首次在沈就业或自主创业的博士（＜35周岁）、硕士（＜30周岁），2015年1月1日后首次在沈自主创业的学士（＜25周岁），以及2017年1月1日后首次在沈就业的学士（＜25周岁）。两类不可交叉、重复享受 |
| **补贴金额/标准** | 第1类（2018年后落户）：博士1250元/月、硕士850元/月、学士及技师500元/月，最长不超过2年；第2类（2015/2017年后就业）：博士800元/月、硕士400元/月、学士200元/月，最长不超过3年。按月发放 |
| **学历要求** | 全日制博士研究生、硕士研究生、本科毕业生；技师以上（含技师）技能人才 |
| **年龄限制** | 第1类：博士/硕士/本科/技师均＜35周岁；第2类：博士＜35周岁、硕士＜30周岁、学士＜25周岁。享受补贴期间均应符合年龄要求 |
| **社保要求** | 就业人员需缴纳社会保险并提供社保缴费凭证；创业人员提供创业实体纳税/免税证明 |
| **申请条件** | 1. 原籍非沈阳市；2. 按落户/就业时间对应类别满足条件；3. 在沈阳市无任何形式自有住房；4. 未享受过沈阳市公租房政策；5. 在沈实现就业（签订劳动合同、缴纳社保）或创业（持有工商营业执照）。1年内未领取补贴视为自动放弃，未按时申报不予补发 |
| **申请流程** | 1. 申请人准备要件→2. 到用人单位所在区（就业）或创业项目所在区（创业）的人社局申报→3. 区县初审→4. 市级复核→5. 拨付补贴资金 |
| **申请平台** | 沈阳市人力资源和社会保障局（http://rsj.shenyang.gov.cn/），各区县人社局线下受理；表格下载网址同上 |
| **申请材料** | 1.《在沈就业创业人员租房补贴申请表》；2. 身份证、毕业证、学位证原件及复印件（留学回国提供教育部《国（境）外学历学位认证书》），技师提供相应资格证书；3. 就业人员：劳动合同、社保缴费凭证（加盖单位公章）、用人单位营业执照复印件（加盖公章）；4. 创业人员：创业实体工商营业执照原件及复印件（加盖公章）、纳税/免税证明；5. 新落户人员提供户口簿原件及复印件 |
| **审核周期** | 常年受理，每次申报上半个年度的租房补贴；区县初审+市级复核 |
| **发放方式** | 按月发放至申请人银行账户 |
| **生效时间** | 2018年1月1日 |
| **失效时间** | 待核实 |
| **政策状态** | 有效。各区县人社局持续受理公示，2024—2025年和平区、大东区、浑南区等均有公示批次 |
| **官方来源** | http://rsj.shenyang.gov.cn/ |
| **最后核实日期** | 2026-07-01 |

#### 2. 沈阳市高校毕业生就业创业生活补贴

| 字段 | 内容 |
|------|------|
| **政策名称** | 沈阳市高校毕业生就业创业生活补贴 |
| **政策文号** | 沈人社发〔2023〕4号《沈阳市人力资源和社会保障局等七部门关于印发〈沈阳市高校毕业生首次购房补贴实施细则〉等6项人才政策实施细则的通知》 |
| **适用对象** | 沈阳市户籍普通高校毕业生（含国、境外高校毕业生）。本科或硕士学位毕业生：毕业两年内（其中毕业超一年的须在毕业后新落户沈阳市）；全日制博士毕业生或出站博士后：年龄不超过45周岁，博士毕业（出站）后首次在沈就业或创业未超过一年，非公务员和参公编制人员，2021年8月1日前未在沈实现就业或创业 |
| **补贴金额/标准** | 本科600元/月、硕士1200元/月、博士2500元/月；自首次申请之日起36个月为政策享受期 |
| **学历要求** | 全日制博士研究生、硕士研究生、本科毕业生（普通高校，含国境外高校） |
| **年龄限制** | 博士≤45周岁；本科/硕士无单独年龄限制（以毕业年限控制） |
| **社保要求** | 毕业后首次在沈实现就业或创业，完成就业登记，并按规定缴纳城镇企业职工基本养老保险 |
| **申请条件** | 1. 沈阳市户籍；2. 本科/硕士：毕业两年内（毕业超一年的须毕业后新落户）；3. 博士：≤45周岁，首次在沈就业创业未超1年，非公务员参公，2021年8月1日前未在沈就业创业；4. 按规定缴纳养老保险。本补贴与县域地区企业就业专项生活补贴不可兼得 |
| **申请流程** | 1. 申请人通过'盛事通'APP提交申请→2. 各区县人社部门即时受理并审核→3. 每月比对发放条件，符合人员在区县政府网站公示7天→4. 公示无异议后报市人社部门→5. 拨付至申请人银行账户 |
| **申请平台** | '盛事通'APP；沈阳市人力资源和社会保障局 http://rsj.shenyang.gov.cn/ |
| **申请材料** | 通过'盛事通'线上提交（身份证、户口簿、毕业证/学位证、劳动合同、社保缴纳证明等在线填报核验） |
| **审核周期** | 即时受理，每月比对公示，公示期7天 |
| **发放方式** | 按月发放至申请人银行账户 |
| **生效时间** | 2023年（沈人社发〔2023〕4号） |
| **失效时间** | 待核实 |
| **政策状态** | 有效。与沈阳市高校毕业生到县域地区企业就业专项生活补贴不可兼得 |
| **官方来源** | https://rsj.shenyang.gov.cn/ |
| **最后核实日期** | 2026-07-01 |

#### 3. 沈阳市高校毕业生首次购房补贴

| 字段 | 内容 |
|------|------|
| **政策名称** | 沈阳市高校毕业生首次购房补贴 |
| **政策文号** | 沈人社发〔2023〕4号《沈阳市高校毕业生首次购房补贴实施细则》 |
| **适用对象** | 在沈企业和事业单位（不含参公事业单位）就业创业的全日制普通高校本科、研究生（硕士、博士）毕业生。国内知名高校定向选调生可享受相应层次补贴。不含机关事业单位全额拨款在编人员 |
| **补贴金额/标准** | 博士毕业生7万元、硕士毕业生4万元、本科毕业生2万元（技师参照本科标准2万元）；一次性发放。本补贴与沈阳市高级技师首次购房补贴不兼得 |
| **学历要求** | 全日制普通高校本科、硕士、博士毕业生 |
| **年龄限制** | null（以毕业年限和购房时间控制，均不超过五年） |
| **社保要求** | 近期在沈企业或事业单位连续缴纳六个月以上社会养老保险，并提供缴费证明 |
| **申请条件** | 1. 在沈企业或事业单位（不含参公事业单位）就业创业；2. 全日制本科/硕士/博士毕业生；3. 在沈首次购房（2017年1月1日后购房）；4. 申请时毕业年限、首次购房时间均不超过五年；5. 连续缴纳6个月以上社保。每人只能享受一次，已领取的不按新标准补差额 |
| **申请流程** | 1. 申请人持要件到缴纳养老保险所在区县人社部门申报（企业养老保险按参保分局划分，其他按单位注册地）→2. 区县人社部门审核→3. 区县网站公示7天→4. 填报汇总表上报市人社部门→5. 拨付补贴资金 |
| **申请平台** | 各区、县（市）人力资源社会保障部门线下申报；沈阳市人力资源和社会保障局 http://rsj.shenyang.gov.cn/ |
| **申请材料** | 1.《沈阳市高校毕业生首次购房补贴申请表》；2. 身份证（港澳台居民通行证、外籍留学生护照）原件及复印件；3. 毕业证（教育部留学服务中心出具的学历学位认证书）原件及复印件；4. 近期在沈企业或事业单位连续缴纳六个月以上社会养老保险缴费证明原件及缴费单位营业执照复印件（机关事业单位社保需提供事业单位法人证书复印件）；5. 不动产权证书原件及复印件（以网签合同签订时间为购房时间的还需提供网签合同复印件） |
| **审核周期** | 区县审核+公示7天+市级汇总 |
| **发放方式** | 一次性发放 |
| **生效时间** | 2023年（沈人社发〔2023〕4号）；购房时间适用范围为2017年1月1日后 |
| **失效时间** | 待核实。过渡期安排：2026年5月31日前符合条件因系统调整或个人原因超期的，可于2026年7月31日前办理 |
| **政策状态** | 有效。每人只能享受一次，与高级技师首次购房补贴不兼得 |
| **官方来源** | https://www.shenyang.gov.cn/zwgk/fdzdgknr/zdlyxxgk/jycyfw/rcbtfw/gxdxs/202312/t20231208_4569991.html |
| **最后核实日期** | 2026-07-01 |

#### 4. 沈阳市人才公寓补贴（基础人才通道）

| 字段 | 内容 |
|------|------|
| **政策名称** | 沈阳市人才公寓补贴 |
| **政策文号** | 待核实（沈阳市人力资源和社会保障局发布，依据沈阳市人才公寓相关政策） |
| **适用对象** | 基础人才：具有国家承认的大专及以上学历，非沈阳市户口或落户沈阳市未满2年。博士生≤45周岁、硕士生≤40周岁、本科生≤35周岁、大专生≤30周岁。经认定的A、B、C、D类人才另行标准（需人才认定，不纳入本通道） |
| **补贴金额/标准** | D类人才及博士毕业生1200元/人/月、硕士毕业生900元/人/月、本科毕业生500元/人/月、大专毕业生300元/人/月；自首次享受补贴月份起36个月为政策享受期。当住房租赁合同租金低于补贴标准的，按实际租金执行 |
| **学历要求** | 国家承认的大专及以上学历（博士、硕士、本科、大专） |
| **年龄限制** | 博士≤45周岁、硕士≤40周岁、本科≤35周岁、大专≤30周岁 |
| **社保要求** | A、B、C类以外人才所在工作单位（创业实体）应为申请人缴纳沈阳市城镇企业职工基本养老保险；省企业、机关事业单位等养老保险不予补贴 |
| **申请条件** | 1. 大专及以上学历，非沈阳户口或落户未满2年；2. 在我市（不含辽中区、新民市、法库县、康平县）无自有住房（以现实备案和不动产权登记为准）；3. 未享受我市公租房及租赁补贴政策；4. 工作单位缴纳沈阳市城镇企业职工基本养老保险；5. 租赁指定人才公寓项目并签订《住房租赁合同》。在沈阳域外就业创业的不在补贴范围 |
| **申请流程** | 1. 申请人与有意向的人才公寓项目门店签订《住房租赁合同》→2. 通过项目门店提交《沈阳市人才公寓租赁申请条件认定表》及相关审批要件→3. 审核通过后按足月计算发放补贴。待人才公寓线上平台开通后全部实行线上申请 |
| **申请平台** | 人才公寓项目门店线下受理（待线上平台开通后转为线上）；沈阳市人力资源和社会保障局 http://rsj.shenyang.gov.cn/ |
| **申请材料** | 1.《住房租赁合同》；2.《沈阳市人才公寓租赁申请条件认定表》；3. 居民身份证（正、反面）；4. 基础人才提供户口簿（首页、本人页）、毕业证书（海外高校毕业生提供教育部留学服务中心国外学历学位认证书）；D类及以上人才提供工作单位（创业实体）营业执照副本 |
| **审核周期** | 项目门店受理后审核，按足月计算发放 |
| **发放方式** | 按月发放（足月计算） |
| **生效时间** | 待核实 |
| **失效时间** | 待核实 |
| **政策状态** | 有效。A、B、C、D类人才通道需人才认定，不纳入系统自动匹配；基础人才通道（大专至博士）按学历+年龄+户籍+社保条件直接匹配。浑南区等区均有人才公寓补贴发放公示 |
| **官方来源** | https://rsj.shenyang.gov.cn/ |
| **最后核实日期** | 2026-07-01 |

#### 5. 沈阳市高校毕业生到县域地区企业就业专项生活补贴

| 字段 | 内容 |
|------|------|
| **政策名称** | 沈阳市高校毕业生到县域地区企业就业专项生活补贴 |
| **政策文号** | 沈人社发〔2023〕4号《沈阳市人力资源和社会保障局等七部门关于印发〈沈阳市高校毕业生首次购房补贴实施细则〉等6项人才政策实施细则的通知》 |
| **适用对象** | 毕业5年内、在县域地区企业缴纳城镇企业职工基本养老保险的沈阳户籍普通高校（含教育部认可的境外高校）全日制大专及以上学历毕业生。县域地区企业指在辽中区、新民市、法库县或康平县工商注册的企业 |
| **补贴金额/标准** | 大专生500元/月、本科生800元/月、硕士研究生1200元/月、博士研究生2500元/月；自首次申请之日起36个月为政策享受期。本补贴与沈阳市高校毕业生就业创业生活补贴不可兼得 |
| **学历要求** | 全日制大专、本科、硕士、博士（普通高校含境外高校） |
| **年龄限制** | null（以毕业年限控制，毕业5年内） |
| **社保要求** | 在县域地区企业（辽中区、新民市、法库县、康平县注册）缴纳城镇企业职工基本养老保险；按当月实际缴纳养老保险情况据实发放，未缴纳不予补发 |
| **申请条件** | 1. 沈阳户籍；2. 毕业5年内；3. 全日制大专及以上学历；4. 在县域地区企业缴纳城镇企业职工基本养老保险。补贴标准以首次申请时学历认定，享受期间学历变更不随之变更。只能享受一次，与高校毕业生就业创业生活补贴不可兼得 |
| **申请流程** | 1. 申请人到养老保险参保单位注册所在区县人社部门申请→2. 区县人社部门审核材料→3. 符合人员在区县政府网站公示7天→4. 公示无异议后向区县财政局申请全额资金并拨付至申请人银行卡（每季度拨付一次）→5. 区县汇总报市人社部门 |
| **申请平台** | 各区县（辽中区、新民市、法库县、康平县）人力资源社会保障部门线下申报；沈阳市人力资源和社会保障局 http://rsj.shenyang.gov.cn/ |
| **申请材料** | 1.《沈阳市高校毕业生到县域地区企业就业专项生活补贴申请表》；2. 居民身份证（原件及复印件）；3. 户口簿（原件及首页、本人页复印件）；4. 毕业证书（原件及复印件，境外高校提供教育部留学服务中心国外学历学位认证书）；5. 缴纳辽宁省城镇企业职工基本养老保险的，提供养老保险缴费证明及参保单位营业执照复印件 |

### 区级政策互斥关系说明

> 最后核实日期：2026-07-01
> 数据来源：沈阳市各区人民政府官网、沈阳市人力资源和社会保障局

1. **区级人才补贴互斥**：沈阳市各区级人才补贴（如沈北新区'沈北汇·英才计划'租房补贴）属于互斥关系，同一申请人不可同时享受两个及以上区级的同类人才租房/生活补贴。若申请人在不同区间变更工作单位，应按当前参保单位所在区申领对应区级补贴。

2. **市级与区级关系**：沈北新区租房补贴可与沈阳市级生活补贴同时享受，但不可与市级吸引人才就业创业租房补贴重复享受同类住房补贴。其他区（和平区、沈河区、皇姑区、大东区、铁西区、浑南区、于洪区、苏家屯区等）目前主要执行市级租房补贴、生活补贴、购房补贴及人才公寓补贴政策，由各区人社部门按属地原则受理审核，同一申请人不能重复享受市级同类补贴。

3. **市级政策内部互斥**：
   - 沈阳市高校毕业生就业创业生活补贴 与 沈阳市高校毕业生到县域地区企业就业专项生活补贴 **不可兼得**（二选一）；
   - 沈阳市高校毕业生首次购房补贴 与 沈阳市高级技师首次购房补贴 **不兼得**，且每人只能享受一次；
   - 沈阳市吸引人才就业创业租房补贴内部第1类（2018年后落户）与第2类（2015/2017年后就业）**不可交叉、重复享受**。
