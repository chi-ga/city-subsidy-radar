### 一、赣州市高校毕业生生活补贴

| 项目 | 内容 |
|------|------|
| 政策名称 | 赣州市高校毕业生生活补贴 |
| 政策文号 | 赣市府办字〔2022〕56号 |
| 适用对象 | 全日制专科、本科、硕士、博士毕业生（含技工院校预备技师班、高级工班毕业生） |
| 补贴金额/标准 | 博士5万元、硕士3万元、本科2万元、大专1万元。分3年按4:3:3比例发放，社保每缴满1年发放一次 |
| 学历要求 | 全日制专科、本科、硕士、博士（含技工院校预备技师班、高级工班毕业生） |
| 年龄限制 | null |
| 社保要求 | 首次在市域内缴纳社保 |
| 申请条件 | 1. 毕业2年内落户赣州（含本地户籍）；2. 2022年5月17日后来赣州市企业就业；3. 与企业签订1年以上劳动合同；4. 首次在市域内缴纳社保 |
| 申请流程 | 1. 支付宝进入"赣服通"赣州分厅；2. "亲清赣商"惠企政策兑现平台提出申请；3. 企业注册地人社部门审核（5个工作日）；4. 注册地行政审批局发放至申请人社保卡 |
| 申请平台 | 支付宝"赣服通"赣州分厅→"亲清赣商"惠企政策兑现平台 |
| 申请材料 | 1. 高校毕业生生活补贴申请表；2. 毕业证书、学信网学籍在线验证报告（技工院校毕业生只需提供毕业证书）；3. 与企业签订的劳动合同 |
| 审核周期 | 毕业2年内申请，人社部门5个工作日审核 |
| 发放方式 | 分3年按4:3:3比例发放至社保卡 |
| 生效时间 | 2022-05-17 |
| 失效时间 | 原试行期至2025年5月16日，2025修订版待核实 |
| 政策状态 | 待核实（原试行期已到期，修订版延续情况待确认） |
| 官方来源 | 赣州市人民政府官网（https://www.ganzhou.gov.cn/gzszf/c102790vpcnhgkvgt/202210/b86081e8da1341d5a7fe3c825b64872c.shtml） |
| 最后核实日期 | 2026-07-01 |

### 二、章贡区高校毕业生生活补贴

| 项目 | 内容 |
|------|------|
| 政策名称 | 章贡区高校毕业生生活补贴 |
| 政策文号 | 区府办字〔2022〕53号 |
| 适用对象 | 全日制专科、本科、硕士、博士毕业生（含技工院校预备技师班、高级工班毕业生） |
| 补贴金额/标准 | 博士5万元、硕士3万元、本科2万元、大专1万元。分3年按4:3:3发放 |
| 学历要求 | 全日制专科、本科、硕士、博士（含技工院校预备技师班、高级工班毕业生） |
| 年龄限制 | null |
| 社保要求 | 首次在章贡区域内缴纳社保 |
| 申请条件 | 1. 毕业2内落户赣州；2. 与章贡区纳税企业签订劳动合同；3. 首次缴纳社保且在章贡区域内 |
| 申请流程 | 1. 每年3月、10月将材料交至章贡高新区管委会；2. 审核（5个工作日内办结）；3. 由企业于次年9月底前统一申报；4. 发放至申请人社保卡 |
| 申请平台 | 线下：章贡高新区管委会；线上（拟）："亲清赣商"惠企政策兑现平台 |
| 申请材料 | 1. 《章贡区企业录用高校毕业生就业人员明细表》；2. 《高校毕业生生活补贴资金申请表》；3. 毕业证书复印件、学信网学籍在线验证报告（技工院校毕业生只需提供毕业证书） |
| 审核周期 | 每年3月、10月集中受理，5个工作日内办结 |
| 发放方式 | 分3年按4:3:3发放至社保卡 |
| 生效时间 | 2022-05-17 |
| 失效时间 | 待核实（与市级政策同步到期可能性大） |
| 政策状态 | 待核实 |
| 官方来源 | 章贡区人民政府官网（https://www.zgq.gov.cn/zgqxxgk/c108293/202211/4623aba2fd2b4457a162bb79719abead.shtml） |
| 最后核实日期 | 2026-07-01 |

> **互斥说明：** 章贡区、蓉江新区、经开区的生活补贴属于互斥关系，只能在一个区享受。
