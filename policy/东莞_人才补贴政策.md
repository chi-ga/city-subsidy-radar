#### 1. 东莞市本科生引进综合补贴（创梦东莞计划）

| 字段 | 内容 |
|------|------|
| **政策名称** | 东莞市本科生引进综合补贴（创梦东莞计划） |
| **政策文号** | 东人社发〔2022〕17号《东莞市本科生引进培养暂行办法》 |
| **适用对象** | 新引进的具有本科学历的人才，或已具备初级职称（或专业技术人员职业资格）的人才，或已取得国家职业资格证书二级/技师（或职业技能等级证书二级/技师）的人才 |
| **补贴金额/标准** | 最高补贴额度1万元。具有东莞市户籍的人才按最高补贴额度的100%补贴；不具有东莞市户籍的人才按最高补贴额度的60%补贴。补贴资金一次性发放，由市财政和用人单位所在镇（街道、园区）财政按7:3比例分担 |
| **学历要求** | 本科及以上学历（或初级职称、技师职业资格） |
| **年龄限制** | null |
| **社保要求** | 需与东莞市用人单位（党政机关、财政全额供养事业单位除外）签订3年或以上劳动合同，在该单位工作并依法缴纳社会保险满1年 |
| **申请条件** | 1.人事档案关系在东莞市；2.引进东莞市（引进时间以首个东莞市用人单位为其缴纳社会保险的起始时间为准）；3.与东莞市用人单位签订3年或以上劳动合同并依法缴纳社会保险满1年；4.符合学历/职称/技能条件之一 |
| **申请流程** | 准备材料→填写申请表并签字盖章→登录"东莞市就莞用信息平台"门户网站提交申请→镇级审核→市级审核和核实→公示名单及征求各部门意见→资金下达拨付（按偶然所得20%税率代扣代缴个人所得税后拨付） |
| **申请平台** | 东莞市就莞用信息平台（https://dghrss.dg.gov.cn/jgyportal/portal/rsrc_Xyjrcbt）；也可通过微信小程序"就莞用"查询审核进度 |
| **申请材料** | 1.《东莞市创新人才综合补贴申请表》（本人签名并加盖指模，单位负责人确认签字并加盖单位公章）；2.本人有效身份证件；3.学历学位证书（国境外学历需提供教育部留学服务中心认证书）；4.劳动合同或聘用合同；5.社会保险缴纳证明；6.户籍证明材料（户口簿）；7.职称证书或职业资格证书（按相应条件类别提供）；8.人事档案管理单位信息 |
| **审核周期** | null（经镇级审核、市级审核、公示等环节，具体天数未公布） |
| **发放方式** | 一次性发放（按偶然所得20%税率代扣代缴个人所得税） |
| **生效时间** | 2022年（东人社发〔2022〕17号印发） |
| **失效时间** | 2025年12月31日（政策有效期至2024年12月31日止，2025年12月为实施有效期内符合政策条件的最后一次申请） |
| **政策状态** | 收尾期。政策有效期已于2024年12月31日截止，2025年12月1日至31日开放最后一次申报，后续不再受理新申请。已申领我市相同类型补贴的，按"就高从优不重复"原则执行 |
| **官方来源** | https://dghrss.dg.gov.cn/gkmlpt/content/4/4444/post_4444348.html |
| **最后核实日期** | 2026-07-01 |

#### 2. 东莞市新时代新引进创新人才综合补贴

| 字段 | 内容 |
|------|------|
| **政策名称** | 东莞市新时代新引进创新人才综合补贴 |
| **政策文号** | 东府〔2018〕106号《东莞市新时代创新人才引进培养实施方案》 |
| **适用对象** | 新引进的具有硕士及以上学位，或具有中级以上职称的人才。其中以中级职称申请的人才，须在《东莞市新时代引进创新人才综合补贴限定用人单位范围名单》中的企业工作 |
| **补贴金额/标准** | 按条件类别分档：正高级职称30万元/人、副高级职称20万元/人、中级职称6万元/人。具有东莞市户籍的新引进人才按最高补贴额度的100%发放，非东莞市户籍按40%发放。综合补贴分3年或5年等额发放，每年需申请一次。博士、硕士按学位类别的具体补贴金额以官方公布标准表为准 |
| **学历要求** | 硕士及以上学位，或中级以上职称 |
| **年龄限制** | null |
| **社保要求** | 与东莞市用人单位（党政机关、财政全额供养事业单位除外）签订3年或以上劳动合同，在该单位工作并依法缴纳社会保险满1年 |
| **申请条件** | 1.人事档案关系在东莞市；2.2017年1月1日至2021年12月31日引进东莞市（引进时间以首个东莞市用人单位为其缴纳社会保险的起始时间为准）；3.与东莞市用人单位签订3年或以上劳动合同并依法缴纳社会保险满1年；4.具有硕士及以上学位或中级以上职称；5.中级职称人才须在限定用人单位范围名单内的企业工作 |
| **申请流程** | 准备材料→填写申请表并签字盖章→登录"东莞市就莞用信息平台"门户网站提交申请→镇级审核→市级审核和核实→公示名单及征求各部门意见→资金下达拨付（按偶然所得20%税率代扣代缴个人所得税后拨付） |
| **申请平台** | 东莞市就莞用信息平台（https://dghrss.dg.gov.cn/jgyportal/portal/rsrc_Xyjrcbt）；也可通过微信小程序"就莞用"查询审核进度 |
| **申请材料** | 1.《东莞市新引进创新人才综合补贴申请表》（本人签名并加盖指模，单位负责人确认签字并加盖单位公章）；2.本人有效身份证件；3.学历学位证书（国境外学历需提供认证书）；4.劳动合同或聘用合同；5.社会保险缴纳证明；6.户籍证明材料（户口簿）；7.职称证书或专业技术人员职业资格证书；8.人事档案管理单位信息；9.非全日制学历需提供学历学位验证报告 |
| **审核周期** | null（经镇级审核、市级审核、公示等环节，具体天数未公布） |
| **发放方式** | 分3年或5年等额发放，每年需申请一次（按偶然所得20%税率代扣代缴个人所得税） |
| **生效时间** | 2018年（东府〔2018〕106号印发） |
| **失效时间** | 待核实（2025年仅受理往期未完成期数的续期申请，不再受理首期申请） |
| **政策状态** | 收尾期。2025年仅受理《东莞市新时代创新人才引进培养实施方案》实施有效期内符合政策条件的往期未完成期数的申请，不再受理首期申请。申请人限以一种条件类别申领，每人限申领一次，按"就高从优不重复"原则执行 |
| **官方来源** | https://dghrss.dg.gov.cn/gkmlpt/content/4/4444/mpost_4444364.html |
| **最后核实日期** | 2026-07-01 |
