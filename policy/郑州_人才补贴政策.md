#### 1. 郑州市青年人才生活补贴

| 字段 | 内容 |
|------|------|
| **政策名称** | 郑州市青年人才生活补贴 |
| **政策文号** | 待核实（"智汇郑州"人才工程"1+N"政策体系，配套《郑州市青年人才储备计划实施细则（暂行）》等文件） |
| **适用对象** | 2017年1月1日以后毕业的全日制博士研究生、硕士研究生、本科毕业生（含海外留学归国人员）和技工院校全日制预备技师（技师） |
| **补贴金额/标准** | 博士1500元/月、硕士1000元/月、本科500元/月；技工院校全日制预备技师（技师）按本科毕业生标准申领。最长发放36个月，5年内根据实际正常缴纳养老保险月份发放 |
| **学历要求** | 全日制博士、硕士、本科；技工院校全日制预备技师（技师） |
| **年龄限制** | 硕士研究生、本科毕业生年龄应在35周岁以下（含35周岁），博士研究生不设年龄限制 |
| **社保要求** | 在郑州市域内用人单位签订劳动合同（聘用合同）或在郑州自主创业，按规定缴纳企业职工养老或城镇居民养老保险，且在郑州连续缴纳社会保险（不含补缴）满3个月 |
| **申请条件** | 1. 2017年1月1日以后毕业的全日制博士、硕士、本科毕业生（含海外留学归国人员）和技工院校全日制预备技师（技师）；2. 硕士研究生、本科毕业生年龄35周岁（含）以下，博士不设年龄限制；3. 2017年1月1日后落户郑州（本市户籍未迁出的毕业生不需重复落户）；4. 在郑州市域内就业创业并按规定缴纳社保满3个月；5. 已办理并激活社会保障卡金融账户 |
| **申请流程** | 1. 申请人通过微信城市服务"智汇郑州"人才工程、支付宝城市服务"智汇郑州"人才工程、郑好办APP"青年人才生活补贴"一件事专区，或前往郑州市各人才服务窗口申请；2. 在线填写个人信息并上传相关材料；3. 系统信息比对及人工核查；4. 审核通过后在郑州市人社局官网公示7个工作日；5. 公示无异议后按月拨付至个人社保卡金融账户 |
| **申请平台** | 微信城市服务—"智汇郑州"人才工程、支付宝城市服务—"智汇郑州"人才工程、郑好办APP—"一件事专区"—"青年人才生活补贴"，或郑州市各人才服务窗口 |
| **申请材料** | 1. 郑州市青年人才生活补贴申请表；2. 毕业证、身份证和户口簿原件及复印件；3. 《教育部学历证书电子注册备案表》或教育部留学服务中心出具的国（境）外学历学位认证书；4. 技工院校全日制预备技师（技师）提供全国技工院校电子注册和统计信息管理系统认定的证明，无法查证的由毕业院校出具相关证明；5. 社会保险缴纳证明材料；6. 自主创业的需提供工商营业执照原件及复印件和纳税证明材料；7. 社会保障卡 |
| **审核周期** | 按月审核，常态化申报，审核通过后按月发放 |
| **发放方式** | 按月发放至申请人社会保障卡金融账户 |
| **生效时间** | 2017年1月1日 |
| **失效时间** | 待核实 |
| **政策状态** | 有效。郑州市常态化受理，按月公示发放 |
| **官方来源** | https://public.xinzheng.gov.cn/D3602Y/2080025.jhtml、https://m.zz.bendibao.com/live/68162.shtm |
| **最后核实日期** | 2026-06-30 |

#### 2. 郑州市青年人才首次购房补贴

| 字段 | 内容 |
|------|------|
| **政策名称** | 郑州市青年人才首次购房补贴 |
| **政策文号** | 郑房〔2022〕60号（《郑州市青年人才首次购房补贴发放实施细则》） |
| **适用对象** | 符合条件的全日制博士研究生、硕士研究生、本科毕业生（技工院校预备技师、技师）、专科（不含中专）毕业生，在我市首次购买住房 |
| **补贴金额/标准** | 博士每人10万元、硕士每人5万元、本科毕业生（技工院校预备技师、技师）每人2万元、专科毕业生每人1万元，一次性发放 |
| **学历要求** | 全日制博士、硕士、本科、专科（不含中专）；技工院校预备技师、技师 |
| **年龄限制** | 硕士研究生、本科毕业生（技工院校预备技师、技师）、专科毕业生年龄须在35周岁（含）以下，博士研究生不设年龄限制 |
| **社保要求** | 郑州户籍，在郑州市域范围内就业创业，并正常缴纳社会保险满一个月 |
| **申请条件** | 1. 硕士研究生、本科毕业生（技工院校预备技师、技师）、专科毕业生年龄须在35周岁（含）以下，博士研究生不设年龄限制；2. 郑州户籍，在郑州市域范围内就业创业，并正常缴纳社会保险满一个月；3. 取得毕业证书时间须在2022年4月14日（含）之后，并在学信网、教育部留学服务中心网上服务大厅或全国技工院校电子注册和统计信息管理系统之一的认证平台完成认证；4. 申请人必须是购房人本人或其配偶；5. 所购房屋用途须是住宅，获取途径须是市场购买；6. 所购房屋是申请人或其配偶在年满18周岁后首次在郑州市通过市场化途径购买的住房，且购买时间在2017年1月1日（含）之后 |
| **申请流程** | 1. 申请人将申报材料提交到住房保障部门人才窗口，或通过郑州市政务服务网、郑好办APP申请；2. 市内五区购房的由郑州市房地产市场和产权交易管理中心受理，其他区域由房屋所在地住房保障部门受理；3. 各受理单位核验后出具《郑州市青年人才首次购房补贴核准确认单》；4. 补贴资金全额拨付至申请人个人账户；5. 年终统一清算 |
| **申请平台** | 郑州市政务服务网、郑好办APP，或市内五区及各区县（市）住房保障部门人才窗口 |
| **申请材料** | 1. 青年人才首次购房补贴申请表；2. 申请人有效身份证件；3. 中国高等教育学历认证报告、国（境）外学历学位认证书或全国技工院校电子注册和统计信息管理系统认定的证明；4. 户口簿；5. 婚姻状况证明或单身承诺书；6. 契税发票或不动产权证书；7. 社保卡 |
| **审核周期** | 常年受理，具体以各区住房保障部门审核为准 |
| **发放方式** | 一次性拨付至申请人个人账户 |
| **生效时间** | 2022年4月14日（本细则适用对象毕业时间） |
| **失效时间** | 待核实 |
| **政策状态** | 有效。2022年4月14日之前毕业的青年人才仍按郑政办〔2020〕24号执行。所购房屋自取得不动产权证书之日起不满3年不得上市转让；夫妻双方均符合条件的可分别申领 |
| **官方来源** | https://public.zhengzhou.gov.cn/D0105Y/6531835.jhtml |
| **最后核实日期** | 2026-06-30 |

#### 3. 郑州市人才公寓租金优惠

| 字段 | 内容 |
|------|------|
| **政策名称** | 郑州市人才公寓租金优惠 |
| **政策文号** | 郑政办〔2023〕11号（《郑州市人才公寓管理暂行办法》） |
| **适用对象** | 在郑就业创业的大学专科（技工院校全日制高级工）（含）以上学历人才 |
| **补贴金额/标准** | 政府筹建的人才公寓租金原则上按照不高于同区域同类型住房市场租金70%的标准确定，具体以各项目公布租金为准 |
| **学历要求** | 大学专科（技工院校全日制高级工）（含）以上学历 |
| **年龄限制** | 45周岁（含）以下；高层次人才、博士和经市人社局认定的市重点产业急需紧缺人才不受学历和年龄限制 |
| **社保要求** | 在郑就业创业（未明确社保缴纳时长，以实际配租项目要求为准） |
| **申请条件** | 1. 申请人、配偶、未成年子女在郑州市内五区和郑东新区、郑州高新区、郑州经开区无自有住房，未承租保障性住房且未享受相关保障性住房货币补贴；2. 年龄在45周岁（含）以下（高层次人才、博士和市重点产业急需紧缺人才不受限制）；3. 申请人及共同申请人均未被依法依规列为失信联合惩戒对象 |
| **申请流程** | 1. 登录"郑州市保障性租赁住房（人才公寓）系统"查看房源；2. 在线提交申请并上传材料；3. 系统实时审核；4. 审核通过后在线选定房源并即时签约；5. 按合同约定缴纳租金入住 |
| **申请平台** | 郑州市保障性租赁住房（人才公寓）系统（郑好办APP等入口） |
| **申请材料** | 1. 申请人及共同申请人有效身份证件；2. 学历学位证书或技工院校相关证明；3. 婚姻及家庭成员关系证明；4. 住房情况相关材料；5. 就业创业证明（以系统实际要求为准） |
| **审核周期** | 申请即审核，随选随签 |
| **发放方式** | 租金减免（按月缴纳，按不高于同区域同类型市场租金70%确定） |
| **生效时间** | 2023年4月13日 |
| **失效时间** | 待核实 |
| **政策状态** | 有效。政府筹建人才公寓常态化配租；承租家庭购买住房的，实行过渡期保障，租金上浮30%。各区县（市）、郑州航空港区、上街区可参照执行 |
| **官方来源** | https://public.zhengzhou.gov.cn/D0105Y/7063906.jhtml |
| **最后核实日期** | 2026-06-30 |
