# 金华市人才补贴政策

> 最后核实日期：2026-07-01
>
> 政策依据：《"智选金华"大学生人才集聚工程若干意见的实施细则》，由金华市人力资源和社会保障局发布并执行；就业岗位补贴依据浙江省高校毕业生就业政策。

---

## 一、金华市区引进人才生活补助

| 项目 | 内容 |
|------|------|
| 政策名称 | 金华市区引进人才生活补助 |
| 政策文号 | 《"智选金华"大学生人才集聚工程若干意见的实施细则》 |
| 适用对象 | 博士、博士后、教授、正高级工程师等青年高层次人才；硕士；双一流本科；普通本科；专科毕业生 |
| 补贴金额/标准 | 博士25万元（第一年5万、第二年8万、第三年12万）；硕士10万元（第一年2万、第二年3万、第三年5万）；双一流本科6万元（每年2万）；普通本科4万元（第一年1万、第二年1万、第三年2万）；专科1.5万元（每年0.5万）。生活补助分三年支付。金华籍应届本科毕业生、在金高校本科及专科应届毕业生可提一档享受 |
| 学历要求 | 全日制普通高校博士、硕士、双一流本科、普通本科、专科（本科、专科须入职企业才能申领） |
| 年龄限制 | null（博士申报当年1月1日不超过45周岁；其他学历不限） |
| 社保要求 | 按规定缴纳社会保险，按实际缴纳社会保险时间享受生活补助，补助期不超过36个月 |
| 申请条件 | 1. 2020年5月4日（含）起首次从市外引进，在市（区）属企事业单位（含民办非企业单位，不含参公事业）就业；2. 按规定缴纳社会保险；3. 本科、专科毕业生须入职企业才能申领；4. 博士申报当年1月1日不超过45周岁 |
| 申请流程 | 1. 符合条件人才登录"浙里办"APP—"浙里人才之家"（金华）—"引进人才生活补助"线上提交申请；2. 系统自动比对身份、学历、社保等信息，5个工作日内完成审核（市属事业单位引进人才由市人社部门审核，企业、区属事业单位引进人才由所在区人社部门审核）；3. 在"浙里人才之家"公示专区公示5个工作日；4. 公示通过后将补助资金发放至申请人社保卡金融账户 |
| 申请平台 | "浙里办"APP—"浙里人才之家"（金华）—"引进人才生活补助" |
| 申请材料 | 通过"浙里办"—"浙里人才之家"线上申请，系统自动比对身份、学历学位、社保缴纳等信息；国（境）外学历需提供教育部留学服务中心出具的学历学位认证书 |
| 审核周期 | 5个工作日内完成审核，公示5个工作日 |
| 发放方式 | 分三年支付，发放至社保卡金融账户 |
| 生效时间 | 2020-05-04 |
| 失效时间 | null（持续执行） |
| 政策状态 | 有效 |
| 官方来源 | 金华本地宝（http://m.jh.bendibao.com/mip/20819.shtm） |
| 最后核实日期 | 2026-07-01 |

## 二、金华人才购房货币补助

| 项目 | 内容 |
|------|------|
| 政策名称 | 金华人才购房货币补助 |
| 政策文号 | 《"智选金华"大学生人才集聚工程若干意见的实施细则》 |
| 适用对象 | 博士、硕士、双一流本科、普通本科、专科毕业生 |
| 补贴金额/标准 | 企事业单位全职引进人才：博士最高40万元、硕士最高20万元、双一流本科最高15万元、普通本科最高8万元、普通专科最高4万元。市区政府机关单位（含参公事业单位）全职引进人才：博士最高10万元、硕士最高5万元、本科最高3万元。金华籍应届本科毕业生、在金高校应届本科毕业生可提档享受。夫妻双方共同持有房屋产权且均符合条件的，按一方全额、一方50%标准享受 |
| 学历要求 | 博士、硕士、双一流本科、普通本科、专科 |
| 年龄限制 | null |
| 社保要求 | 2022年5月8日起首次在大市范围内参保，在市区用人单位全职工作，由用人单位连续参保满6个月 |
| 申请条件 | 1. 2022年5月8日起首次在大市范围内参保；2. 在市区用人单位全职工作，由用人单位连续参保满6个月；3. 在市区首次购买住房；4. 自引进起3年内购房有效，所购住房满5年后方可上市交易 |
| 申请流程 | 1. 符合条件人才登录"浙里办"—"浙里人才之家"（金华）—"购房补助"提出申请；2. 系统比对身份、学历、社保、购房等信息并审核；3. 公示；4. 公示通过后发放至社保卡金融账户 |
| 申请平台 | "浙里办"—"浙里人才之家"（金华）—"购房补助" |
| 申请材料 | 通过"浙里办"—"浙里人才之家"线上申请，系统比对身份、学历学位、社保缴纳、购房等信息；国（境）外学历需提供教育部留学服务中心出具的学历学位认证书；须提供所购住房网签备案合同或不动产登记证书等购房材料 |
| 审核周期 | 线上审核，公示5个工作日 |
| 发放方式 | 一次性发放至社保卡金融账户 |
| 生效时间 | 2022-05-08 |
| 失效时间 | null（持续执行） |
| 政策状态 | 有效 |
| 官方来源 | 金华人社（http://m.toutiao.com/group/7259930845098951180/） |
| 最后核实日期 | 2026-07-01 |

## 三、金华出售型人才住房购买折扣

| 项目 | 内容 |
|------|------|
| 政策名称 | 金华出售型人才住房购买折扣 |
| 政策文号 | 《"智选金华"大学生人才集聚工程若干意见的实施细则》 |
| 适用对象 | 博士、硕士、双一流本科、其他本科、大专毕业生 |
| 补贴金额/标准 | 按90平方米配售面积享受出售型人才住房购买折扣优惠：博士7折、硕士及双一流本科8.5折、其他本科9折、大专9.5折 |
| 学历要求 | 博士、硕士、双一流本科、其他本科、大专 |
| 年龄限制 | null |
| 社保要求 | 以用人单位名义依法缴纳养老保险6个月以上 |
| 申请条件 | 1. 与我市企事业单位（不含参公事业单位）签订3年以上全职劳动合同；2. 以用人单位名义依法缴纳养老保险6个月以上；3. 本人及配偶、未成年子女均未在市区拥有除农房外的任何形式自有住房 |
| 申请流程 | 1. 登录"浙里办"—"浙里人才之家"（金华）—"人才住房预约"提交预约；2. 审核资格；3. 选房购房享受折扣 |
| 申请平台 | "浙里办"—"浙里人才之家"（金华）—"人才住房预约" |
| 申请材料 | 通过"浙里办"—"浙里人才之家"线上预约，系统比对身份、学历、社保、住房等信息；国（境）外学历需提供教育部学历学位认证书 |
| 审核周期 | 线上审核 |
| 发放方式 | 购房折扣优惠（非现金，直接在购房时享受） |
| 生效时间 | 待核实（依据"智选金华"细则，持续执行） |
| 失效时间 | null（持续执行） |
| 政策状态 | 有效 |
| 官方来源 | 金华人社（http://m.toutiao.com/group/7259930845098951180/） |
| 最后核实日期 | 2026-07-01 |

> 说明：购房货币补助与出售型人才住房购买折扣不可同时享受，二选一。

## 四、金华高校毕业生到中小微企业就业岗位补贴

| 项目 | 内容 |
|------|------|
| 政策名称 | 金华高校毕业生到中小微企业就业岗位补贴 |
| 政策文号 | 浙江省高校毕业生就业政策（金华市执行） |
| 适用对象 | 毕业2年以内高校毕业生 |
| 补贴金额/标准 | 每年3000元，补贴期限不超过3年 |
| 学历要求 | 高校毕业生（专科及以上） |
| 年龄限制 | null（毕业2年以内） |
| 社保要求 | 依法缴纳社会保险费 |
| 申请条件 | 1. 毕业2年以内高校毕业生到中小微企业就业；2. 签订1年及以上劳动合同；3. 依法缴纳社会保险费；4. 工资低于当地上年度非私营和私营单位就业人员加权平均工资 |
| 申请流程 | 1. 通过"浙里办"APP、支付宝"浙里办"小程序、微信"浙里办"小程序或浙江政务服务网搜索"高校毕业生就业补贴申领"办理；2. 系统审核；3. 公示后发放 |
| 申请平台 | "浙里办"APP / 浙江政务服务网，搜索"高校毕业生就业补贴申领" |
| 申请材料 | 通过"浙里办"线上申请，系统比对身份、学历、劳动合同、社保等信息；须提供劳动合同、社保缴费证明等 |
| 审核周期 | 线上审核 |
| 发放方式 | 按年发放，不超过3年 |
| 生效时间 | 待核实（浙江省政策，长期执行） |
| 失效时间 | null（持续执行） |
| 政策状态 | 有效 |
| 官方来源 | 金华本地宝（http://m.jh.bendibao.com/mip/28596.shtm） |
| 最后核实日期 | 2026-07-01 |

> 说明：该项补贴与到养老、家政服务和现代农业企业就业岗位补贴不重复享受。

## 六、金华学历提升补助

| 项目 | 内容 |
|------|------|
| 政策名称 | 金华学历提升补助 |
| 政策文号 | 《"智选金华"大学生人才集聚工程若干意见的实施细则》之学历提升补助实施细则 |
| 适用对象 | 在职攻读经统招统考的非全日制硕士、博士且获得学历学位证书的人员 |
| 补贴金额/标准 | 获得硕士学历学位证书给予3万元补助；获得博士学历学位证书给予5万元补助 |
| 学历要求 | 在职攻读非全日制硕士、博士（获取的学历学位须在原先基础上有提升） |
| 年龄限制 | null |
| 社保要求 | 在市（区）属企事业单位就业，由用人单位连续参加基本养老保险满6个月，且发放前一直处于基本养老保险参保状态 |
| 申请条件 | 1. 在职攻读经统招统考的非全日制硕士、博士且获得学历学位证书（学历学位须在原先基础上有提升）；2. 在市（区）属企事业单位就业；3. 由用人单位连续参加基本养老保险满6个月，且发放前一直处于参保状态 |
| 申请流程 | 1. 符合条件人才登录"浙里办"—"浙里人才之家"（金华）—"学历提升补助"提出申请，提交相关材料；2. 审核（市属事业单位引进人才由市人社部门审核，企业、区属事业单位引进人才由所在区人社部门审核）；3. 在"浙里人才之家"公示专区公示5个工作日；4. 公示通过后将补助资金发放至申请人社保卡金融账户 |
| 申请平台 | "浙里办"—"浙里人才之家"（金华）—"学历提升补助" |
| 申请材料 | 1.通过"浙里办"—"浙里人才之家"线上申请；2.须上传学信网出具的学历验证报告，或国（境）外学历学位认证书；3.系统比对社保等信息 |
| 审核周期 | 线上审核，公示5个工作日 |
| 发放方式 | 一次性发放至社保卡金融账户 |
| 生效时间 | 2023-10-10 |
| 失效时间 | null（持续执行） |
| 政策状态 | 有效 |
| 官方来源 | 金华市人力资源和社会保障局（http://rsj.jinhua.gov.cn/art/2023/10/10/art_1229165046_58716419.html） |
| 最后核实日期 | 2026-07-01 |
