# 临沂市人才补贴政策

> 最后核实日期：2026-07-01
>
> 政策依据：《临沂市青年人才津贴发放实施办法（试行）》（依据临发〔2018〕12号）、临沂市青年人才购房补贴政策、临建发〔2021〕33号，由临沂市人力资源和社会保障局、临沂市住房和城乡建设局等部门发布并执行。

---

## 一、临沂市青年人才津贴

| 项目 | 内容 |
|------|------|
| 政策名称 | 临沂市青年人才津贴 |
| 政策文号 | 《临沂市青年人才津贴发放实施办法（试行）》（依据临发〔2018〕12号）、《关于做好临沂市青年人才津贴申报审核工作的通知》 |
| 适用对象 | 博士毕业生、硕士毕业生、全日制"双一流"本科毕业生、全日制普通本科毕业生 |
| 补贴金额/标准 | 博士3000元/月、硕士2000元/月、全日制"双一流"本科1000元/月，补贴期3年；全日制普通本科1000元/月，补贴期2年。由市财政和用人单位分别承担50% |
| 学历要求 | 博士、硕士、全日制"双一流"本科、全日制普通本科 |
| 年龄限制 | null（要求在择业期内，即毕业后三个年度内） |
| 社保要求 | 在临沂市连续缴纳社会保险（以养老保险为准）满1年以上；自主创办企业人员须在临沂市登记注册企业且依法纳税满1年 |
| 申请条件 | 1. 择业期（毕业后三个年度）内在临沂市创办企业或到企业、其他"两新"组织就业；2. 与用人单位签订劳动合同（自主创办企业人员须为企业法定代表人）；3. 在临沂市连续缴纳社会保险（以养老保险为准）且满1年（自主创办企业人员须登记注册企业且依法纳税满1年）；4. 全日制普通本科毕业生须2022年5月14日（含）以后在临沂市首次缴纳社会保险（自主创办企业人员须2022年5月14日（含）以后登记注册企业且依法纳税满1年） |
| 申请流程 | 1. 采取"无形认证"+网上申报方式；2. 各级人社部门通过政务数据比对，按属地（纳税所在地）管理原则审核确认符合条件人员；3. 既往人员由用人单位负责申请并填报有关信息；4. 审核通过后公示、发放 |
| 申请平台 | 临沂市人力资源和社会保障局网上服务大厅（http://zwfw.rsj.linyi.gov.cn/smartsisp-dwwb/#/）；临沂人才工作网（http://linyi.rcsd.cn/） |
| 申请材料 | 1.全程网办，依托'临沂市青年人才津贴申报系统'无形认证，通过政务数据比对审核，原则上无需提交纸质材料；2.申报人在系统如实填报人才津贴申报信息（含身份、学历学位、劳动合同、社保缴纳、创办企业登记纳税等信息项）；3.既往人员由用人单位负责填报有关信息 |
| 审核周期 | 每年集中申报（往年约8月受理，具体以年度通知为准） |
| 发放方式 | 按月发放，博士/硕士/双一流本科期限3年，普通本科期限2年 |
| 生效时间 | 2018-06（临发〔2018〕12号） |
| 失效时间 | null（持续执行） |
| 政策状态 | 有效（2024年度仍在按月公示发放） |
| 官方来源 | 临沂本地宝（http://m.linyi.bendibao.com/live/18638.shtm） |
| 最后核实日期 | 2026-07-01 |

## 二、临沂市青年人才购房补贴

| 项目 | 内容 |
|------|------|
| 政策名称 | 临沂市青年人才购房补贴 |
| 政策文号 | 临沂市青年人才购房补贴政策（依据临发〔2018〕12号及2022年人才新政） |
| 适用对象 | 博士研究生、硕士研究生、全日制本科生 |
| 补贴金额/标准 | 博士30万元、硕士10万元、本科5万元，一次性发放至社保卡金融账户。夫妻双方均符合补贴条件的，可分别享受，但享受补贴总金额不得超过购房金额 |
| 学历要求 | 博士研究生、硕士研究生、全日制本科生 |
| 年龄限制 | null |
| 社保要求 | 在市内"两新"组织、创办企业正常缴纳12个月以上的社会保险（以养老保险为准） |
| 申请条件 | 1. 2022年5月14日（含）后新到临沂市"两新"组织就业或创办企业；2. 与就业单位签订3年及以上的劳动合同；初次自主创业人才须依法登记注册并担任法定代表人；3. 在市内"两新"组织、创办企业正常缴纳12个月以上社会保险（以养老保险为准），且购房时间不得早于劳动合同签订时间、创办企业注册时间；4. 申请人及其配偶、未成年子女在临沂市范围内无住房登记信息，且至购房之日前在临沂市无住房交易记录；5. 所购住房须为临沂市已取得预售许可证的新建商品住宅，或已取得不动产权证的二手住宅，产权应为申请人所有（已婚共有产权人应为申请人配偶） |
| 申请流程 | 1. 符合条件人才由用人单位按照属地（纳税所在地）管理原则，按年度统一向县区人社局提出申请；2. 县区人社部门将审核符合条件人员基本信息、所购住房情况、拟发放补贴金额通过网站公示5个工作日；3. 公示无异议后，县区人社局将补贴明细报同级财政部门和市人社局，市人社局汇总后报市财政局；4. 由县区人社局按程序将购房补贴发放至申请人社保卡金融账户 |
| 申请平台 | 用人单位按属地原则向县区人社局申请 |
| 申请材料 | 1.《人才购房补贴申请表》；2.《人才购房补贴申请汇总表》；3.申请人身份证，已婚的应提供结婚证及未成年子女信息；4.人才证明材料（博士、硕士、本科人员提供学历学位证明；海外留学归国人员提供教育部留学服务中心出具的国外学历学位认证书；港澳台及外籍大学生提供国家承认的学历学位有关证明）；5.人才所在单位营业执照（或组织机构代码证）副本；6.人才与用人单位签订的全职劳动合同（自主创业的提供公司营业执照、公司章程）；7.所购住房的不动产登记证书或经网签备案的商品房买卖合同（以共有产权住房申请的，共有人应为申请人配偶）；8.在用人单位显著位置公示人才基本情况（姓名、人才类别、毕业院校、引进时间、拟申请补贴金额等）的照片，公示期不得少于3个工作日。第3至第7项规定的证件、证明需提供原件 |
| 审核周期 | 按年度集中受理 |
| 发放方式 | 一次性发放至社保卡金融账户 |
| 生效时间 | 2022-05-14 |
| 失效时间 | null（持续执行） |
| 政策状态 | 有效 |
| 官方来源 | 临沂本地宝（https://m.linyi.bendibao.com/news/16874.shtm） |
| 最后核实日期 | 2026-07-01 |

## 三、临沂市新就业无房职工阶段性住房租赁补贴

| 项目 | 内容 |
|------|------|
| 政策名称 | 临沂市新就业无房职工阶段性住房租赁补贴 |
| 政策文号 | 临建发〔2021〕33号《临沂市新就业无房职工发放阶段性住房租赁补贴实施方案》 |
| 适用对象 | 全日制专科及以下、本科、硕士及以上学历新就业无房职工 |
| 补贴金额/标准 | 全日制专科及专科以下学历300元/月；全日制本科学历350元/月；全日制硕士研究生以上学历或学位400元/月。申请人实际承担的房屋租金低于核定发放补贴金额的，按实际发生金额发放。累计发放期限不超过36个月，且不超过房屋租赁期限 |
| 学历要求 | 全日制专科及以下、本科、硕士及以上 |
| 年龄限制 | null（要求全日制学校毕业未满5年） |
| 社保要求 | 正常缴纳社会保险和住房公积金（至少缴存至申请上月且个人账户状态正常，不存在冻结等情况） |
| 申请条件 | 1. 具有临沂市区（含兰山区、罗庄区、河东区、经济技术开发区、高新技术开发区）的常住人口；2. 全日制学校毕业未满5年；3. 2021年1月1日以后在市区范围内首次就业并办理就业登记；4. 正常缴纳社会保险和住房公积金；5. 在市区范围内以家庭为单位无自有住房，也未享受过其他保障性住房，需通过租赁住房方式解决居住困难；6. 与人才购房补贴、人才租房补贴、安家补贴等财政列支的其他住房补贴以及住房保障政策互斥，不得重复享受 |
| 申请流程 | 1. 申请人填写《临沂市市区新就业无房职工住房租赁补贴申请审批表》并备齐相关材料；2. 向单位所在区住建（住房保障）部门提出申请；3. 区住建部门联合相关单位对申请人资格审核；4. 审核通过后公示；5. 自公示合格的次月起按季度或年度发放 |
| 申请平台 | 各区住建（住房保障）部门（兰山区8069606、罗庄区2987573、河东区7116957、高新区7958667；临沂市供热和住房保障中心8208191） |
| 申请材料 | 1.《临沂市市区新就业无房职工住房租赁补贴申请审批表》；2. 身份证、户口本、婚姻证明等身份及家庭关系材料；3. 学历证明；4. 劳动合同、就业登记证明、社保及公积金缴存证明；5. 租房合同；6. 家庭住房情况核查授权 |
| 审核周期 | 每季度首月10日前受理 |
| 发放方式 | 按季度或年度发放，累计不超过36个月 |
| 生效时间 | 2021-10-09 |
| 失效时间 | null（持续执行，2026年第一季度仍在公示发放） |
| 政策状态 | 有效 |
| 官方来源 | 临沂市住房和城乡建设局（http://lyjs.linyi.gov.cn/info/1294/68717.htm）；琅琊新闻网（https://app.langya.cn/a/403763.html） |
| 最后核实日期 | 2026-07-01 |
