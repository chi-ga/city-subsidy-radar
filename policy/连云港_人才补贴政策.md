# 连云港人才补贴政策

> 最后核实日期：2026-07-01
>
> 政策依据：《关于大力实施花果山英才计划助推高质量发展的若干政策意见》（人才新政2.0），由连云港市委人才工作领导小组办公室、连云港市人力资源和社会保障局发布并执行。

## 一、连云港市人才生活补贴（人才新政2.0）

| 项目 | 内容 |
|------|------|
| 政策名称 | 连云港市人才生活补贴 |
| 政策文号 | 《关于大力实施花果山英才计划助推高质量发展的若干政策意见》（人才新政2.0） |
| 适用对象 | 企业引进的博士研究生；毕业后3年内初次到本市工作的全日制普通高校硕士研究生、本科生（须由主导产业规上工业企业、人才企业、科技型企业引进） |
| 补贴金额/标准 | 企业引进博士研究生3000元/月；全日制硕士2000元/月；全日制本科1000元/月。事业单位引进的博士研究生1000元/月 |
| 学历要求 | 博士研究生（不限全日制）；全日制普通高校硕士、本科毕业生 |
| 年龄限制 | null |
| 社保要求 | 引进人才参保后即可申报（以参保时间为准） |
| 申请条件 | 1. 从连云港市行政区域外引进到连云港市市区（不含赣榆区）；2. 博士为企业引进（不限毕业年限），硕士、本科须毕业后3年内初次到本市全职工作；3. 硕士、本科须由主导产业规上工业企业、人才企业、科技型企业引进（三类企业由主管部门审核认定）；4. 引进后在连参保；5. 在引进或晋升后1年内申报（逾期不再受理） |
| 申请流程 | 1. 引进人才参保后，登录"连云港市高层次人才一站式服务平台"注册；2. 在平台选择"人才生活补贴"模块填报信息并上传材料；3. 主管部门审核企业资质与人才资格；4. 审核通过后按月发放生活补贴 |
| 申请平台 | 连云港市高层次人才一站式服务平台（https://yzs.lygdj.gov.cn/），可通过"连云港党建网""花果山英才"微信公众号登录 |
| 申请材料 | null（线上平台申报，系统填报信息并按平台指引上传材料，具体材料清单以平台申报指引为准） |
| 审核周期 | null（不设集中申报期，实时申报） |
| 发放方式 | 按月发放（标准为元/月，人才在连工作时间需大于等于1个月/年） |
| 生效时间 | 人才新政2.0实施后（2023年起执行） |
| 失效时间 | null |
| 政策状态 | 有效。2025年持续受理申报。不设集中申报期，引进人才参保后即可实时申报。生活补贴申领有效期为1年（引进或晋升后1年内申报）。注：A/B/C类高层次人才及石化、生物医药、新材料三大重点产业引进的顶尖、领军、紧缺人才需经市委人才办遴选认定，不属于普惠政策范畴 |
| 官方来源 | 连云港本地宝（https://m.lyg.bendibao.com/live/16967.shtm） |
| 最后核实日期 | 2026-07-01 |

## 二、连云港市人才购房券（3.0版）

| 项目 | 内容 |
|------|------|
| 政策名称 | 连云港市人才购房券（3.0版） |
| 政策文号 | 3.0版人才购房券功能（依据《关于大力实施花果山英才计划助推高质量发展的若干政策意见》人才新政2.0），由连云港市委人才办、市住建局、市人社局联合实施 |
| 适用对象 | 从连云港市行政区域外引进到连云港市市区（不含赣榆区）的各类人才，覆盖从大专生到博士等8类群体 |
| 补贴金额/标准 | 企业全职引进：博士30万元、全日制硕士15万元、全日制"双一流"本科10万元、全日制其他高校本科5万元、全日制大专3万元；事业单位引进博士10万元。最高200万元（A类高层次人才） |
| 学历要求 | 博士研究生（不限全日制）；全日制普通高校硕士、本科、大专毕业生 |
| 年龄限制 | null |
| 社保要求 | 引进人才在连参保后即可申请 |
| 申请条件 | 1. 从连云港市行政区域外引进到连云港市市区（不含赣榆区）；2. 企业全职引进的博士、全日制硕士、"双一流"本科、其他高校本科、大专生，或事业单位引进的博士；3. 在连参保后即可申请；4. 须自引进之日起5年内申请发放和兑付，逾期不再受理；5. 每人在连云港市行政区域内仅可享受一次购房券政策，与原共有产权人才房政策互斥 |
| 申请流程 | 1. 引进人才在连参保后，登录"连云港市高层次人才一站式服务平台"；2. 按平台指引填写信息，上传《购房券发放申请表》申请购房券；3. 审核通过后获得购房券额度；4. 意向购房时，通过同一平台上传《购房券兑付申请表》申请抵扣首付款；5. 购房金额低于购房券额度的按实际购房金额兑付，购房券仅可兑付一次 |
| 申请平台 | 连云港市高层次人才一站式服务平台（https://yzs.lygdj.gov.cn/） |
| 申请材料 | 1. 《购房券发放申请表》（申请发放时）；2. 《购房券兑付申请表》（申请兑付抵扣首付款时） |
| 审核周期 | null（不设集中申请期，不限申请人名下是否有房产，参保后即可申请） |
| 发放方式 | 一次性兑付（用于抵扣购房首付款，购房券实行实名制管理，仅限申请人本人使用，夫妻双方均持有购房券可累加使用） |
| 生效时间 | 2025年10月30日（3.0版功能上线），适用于2025年5月16日及之后来连参保的人才 |
| 失效时间 | null |
| 政策状态 | 有效。3.0版于2025年10月30日上线，覆盖8类人才群体。申请兑付"零门槛"，全流程线上申请。公积金贷款额度同步提额：C类及以上人才140万元、博士120万元、硕士及本科80万元、大专60万元，夫妻双方可叠加使用。注：A/B/C类高层次人才需经人才分类认定，属于专业通道层；企业/事业单位引进的博士、硕士、本科、大专为普惠层 |
| 官方来源 | 连云港发布/澎湃新闻（https://m.thepaper.cn/baijiahao_31870763）、连云港本地宝（https://m.lyg.bendibao.com/live/16212.shtm） |

### 区级政策互斥关系说明

> 最后核实日期：2026-07-01

连云港市人才补贴政策采用**市级统一政策**模式，生活补贴和购房券均由市级统一制定标准，在市区（不含赣榆区）范围内统一执行。具体情况如下：

1. **无独立区级政策**：海州区、连云区、赣榆区（赣榆区除外）、东海县、灌云县、灌南县等不设独立的差异化人才补贴政策，统一执行市级生活补贴和购房券政策。

2. **购房券适用范围**：3.0版购房券适用于连云港市市区（不含赣榆区），引进人才在市区参保后即可申请。赣榆区执行单独政策。

3. **无区级互斥关系**：由于各区执行同一市级政策、标准统一，**不存在区级政策互斥问题**。市级生活补贴与市级购房券为两项独立政策，分别针对租房居住阶段和购房阶段，可叠加享受。

4. **生活补贴与购房券关系**：生活补贴为按月发放的阶段性补贴，购房券为一次性购房支持券，二者适用阶段不同，可叠加享受。购房券每人仅可享受一次，与原共有产权人才房政策互斥。
