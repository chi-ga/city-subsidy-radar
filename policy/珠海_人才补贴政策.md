#### 1. 珠海市企业新引进青年人才租房和生活补贴

| 字段 | 内容 |
|------|------|
| **政策名称** | 珠海市企业新引进青年人才租房和生活补贴 |
| **政策文号** | 珠人社〔2019〕97号、珠人社规〔2023〕4号 |
| **适用对象** | 从市外全职新引进到珠海市企业、新型研发机构、社会团体和民办非企业等用人单位的青年人才 |
| **补贴金额/标准** | 全日制本科：2.6万元，分2年两次等额发放（每次1.3万元）；全日制硕士研究生（含中级职称、高级技师）：3.8万元，分2年两次等额发放（每次1.9万元） |
| **学历要求** | 普通高等教育全日制本科或硕士研究生 |
| **年龄限制** | 本科：引进时30周岁以下（含30周岁）；硕士：引进时35周岁以下（含35周岁） |
| **社保要求** | 首次申请时已在珠海连续缴纳社会保险费及工资薪金个人所得税6个月以上；续发时距上一笔补贴间隔12个月以上，期间仍在珠海连续缴纳社保，因变更单位中断累计不超过3个月 |
| **申请条件** | 1. 2018年1月1日（含）后从市外全职新引进；2. 与珠海市用人单位签订2年以上（本科）或3年以上（硕士）劳动或聘用合同且全职在珠海工作；3. 依法在珠海缴纳社会保险费及工资薪金个人所得税；4. 须取得珠海市户籍（港澳台及外籍人才、正副高级职称除外）；5. 用人单位注册地不在横琴粤澳深度合作区范围内；6. 珠海市高校不列入申报范围 |
| **申请流程** | 1. 登录珠海市人力资源和社会保障网上服务平台注册并填报；2. 用人单位网上审核；3. 所在区经办机构初审；4. 市人才资源和就业服务中心复核；5. 公示5个工作日；6. 市人社局印发补贴发放方案，各区拨付至个人账户 |
| **申请平台** | 珠海市人力资源和社会保障网上服务平台 https://wsfw.zhrsj.zhuhai.gov.cn/zhrsClient/login.jsp |
| **申请材料** | 1. 《个人申请表》；2. 全日制学历证书（系统自动比对成功则无需上传）；3. 工资薪金缴纳个税纳税清单；4. 在珠海实习经历的《在校生实习说明》（如有）；5. 合同单位与参保单位不一致的情况说明（如有）；6. 国（境）外学历需上传教育部留学服务中心《国（境）外学历学位认证书》及学历学位证书 |
| **审核周期** | 复核15个工作日，特殊情况不超过20个工作日；公示5个工作日 |
| **发放方式** | 分2年两次等额发放，拨付至申请人银行账户 |
| **生效时间** | 2019年（珠人社〔2019〕97号） |
| **失效时间** | 珠人社〔2019〕97号已到期；目前暂停受理新引进申请，续发按原标准执行；后续新政待定 |
| **政策状态** | 原政策到期，新引进申请暂停受理，续发申领仍按原标准执行 |
| **官方来源** | https://www.zhuhai.gov.cn/sjb/xw/yw/content/post_3708120.html |
| **最后核实日期** | 2026-06-30 |

| **申请条件** | 1. 从市外全职新引进；2. 与珠海市用人单位签订3年以上劳动或聘用合同且全职在珠海工作；3. 年收入要求：副高级18万元以上，正高级20万元以上；4. 无须取得珠海市户籍；5. 用人单位注册地不在横琴粤澳深度合作区范围内 |
| **申请流程** | 1. 登录珠海市人力资源和社会保障网上服务平台注册并填报；2. 用人单位网上审核；3. 所在区经办机构初审；4. 市人才资源和就业服务中心复核；5. 公示5个工作日；6. 市人社局印发补贴发放方案，各区拨付至个人账户 |
| **申请平台** | 珠海市人力资源和社会保障网上服务平台 https://wsfw.zhrsj.zhuhai.gov.cn/zhrsClient/login.jsp |
| **申请材料** | 1. 《个人申请表》；2. 职称证书；3. 个人所得税纳税记录和纳税申报明细材料（加盖单位公章）；4. 劳动合同；5. 工资薪金缴纳个税纳税清单 |
| **审核周期** | 复核15个工作日，特殊情况不超过20个工作日；公示5个工作日 |

| **生效时间** | 2019年（珠人社〔2019〕97号） |

| **政策状态** | 原政策到期，新引进申请暂停受理，续发申领仍按原标准执行；需高级职称，属专业通道 |
| **官方来源** | https://www.zhuhai.gov.cn/sjb/xw/yw/content/post_3708120.html |

#### 1. 珠海高新区新引进青年人才租房（生活）补贴

| 字段 | 内容 |
|------|------|
| **政策名称** | 珠海高新区新引进青年人才租房（生活）补贴 |
| **政策文号** | 珠高党群〔2023〕147号 |
| **适用对象** | 2022年1月6日（含）后新引进到高新区主园区企业（含企业化运作的新型研发机构，不含国有企业），首次在高新区缴纳社会保险的非珠海市户籍青年人才 |
| **补贴金额/标准** | 全日制本科：800元/月，最长24个月，合计1.92万元；全日制硕士研究生/中级职称：1000元/月，最长24个月，合计2.4万元 |
| **学历要求** | 普通高等教育全日制本科及以上学历（教育部认可的海外同等学历），或具有中级职称专业技术资格 |

| **社保要求** | 与高新区企业签订劳动合同并依法缴纳社会保险费；申报时仍在区内企业正常就职 |
| **申请条件** | 1. 2022年1月6日（含）后新引进到高新区主园区；2. 首次在高新区缴纳社保；3. 非珠海市户籍；4. 本人在珠海市范围内未入选高新区共有产权房配额名单；5. 须在高新区首次购买社保后两年内提交申请；6. 不重复享受市、区其他租（住）房补贴与安居类补贴 |
| **申请流程** | 1. 高新区发布申报通告；2. 登录申报系统个人注册并填报；3. 区党群工作部系统审核+人工审核；4. 第三方机构全面核实；5. 公示5个工作日；6. 区管委会审批后由区财政拨付至个人账户 |
| **申请平台** | 珠海高新区人才申报系统（具体网址以当年度申报指南为准） |
| **申请材料** | 1. 身份证明（国（境）外人士上传身份证、通行证或护照，国（境）内人士无需上传）；2. 国（境）外学历需上传毕业证书及教育部留学服务中心《国（境）外学历学位认证书》，国（境）内学历无需上传；3. 中级职称人才另需提供中级职称证书、职称评审表（或考核认定表）、考试报名表（或成绩查询网页截图）、学历证书 |

| **发放方式** | 按月发放，最长24个月 |
| **生效时间** | 2023-08-01 |
| **失效时间** | 细则有效期3年，至2026年7月31日左右，具体以官方最新通知为准 |
| **政策状态** | 有效（2023年修订版施行中） |

### 横琴粤澳深度合作区

> 最后核实日期：2026-06-30
> 数据来源：横琴粤澳深度合作区官网《关于开展横琴粤澳深度合作区2026年度引进学历型、专业技术和技能人才生活补贴申报工作的通知》、珠海本地宝

> **区级政策互斥说明：** 横琴粤澳深度合作区实行独立的人才生活补贴政策，用人单位须在合作区依法登记注册且实质性运营。该补贴与珠海市其他区级补贴及市级租房和生活补贴按“从新、从优、从高、不重复”原则执行，用户只能根据实际就业区域选择其一。

#### 1. 横琴粤澳深度合作区引进学历型、专业技术人才生活补贴

| 字段 | 内容 |
|------|------|
| **政策名称** | 横琴粤澳深度合作区引进学历型、专业技术人才生活补贴 |
| **政策文号** | 粤澳深合执字〔2024〕19号、粤澳深合经发通〔2024〕4号 |
| **适用对象** | 在合作区依法登记注册且实质性运营的企业或机构等用人单位实际工作的全日制本科/硕士/博士毕业生，或具有助理级/中级/副高级及以上专业技术资格的人才 |
| **补贴金额/标准** | 本科/助理级：6万元，分3年等额发放（每年2万元）；硕士/中级：9万元，分3年等额发放（每年3万元）；博士/副高级以上：27万元，分3年等额发放（每年9万元） |
| **学历要求** | 普通高等教育全日制本科/硕士/博士并取得学位（国（境）外学历需教育部认证） |
| **年龄限制** | 本科/助理级：不超过40周岁（含40周岁）；硕士/中级：不超过45周岁（含45周岁）；博士/副高级以上：不超过50周岁（含50周岁） |
| **社保要求** | 首次申请须在合作区连续缴纳个人所得税和社会保险6个月以上；申请第二笔须累计缴纳24个月以上；申请第三笔须累计缴纳36个月以上；社保断缴累计不超过3个月可累计计算 |
| **申请条件** | 1. 在合作区实质性运营的企业或机构工作（党政机关、参公事业单位在编人员除外）；2. 与合作区用人单位累计签订3年以上劳动合同，申报时仍存在劳动关系且在合作区实际工作；3. 未申领完毕原横琴新区或珠海市同类政策补贴；4. 已享受合作区保障性住房或其他同类人才补贴的不得重复申领 |
| **申请流程** | 1. 用人单位登录横琴粤澳深度合作区惠企利民服务平台进行单位资格审核；2. 申请人登录平台填报个人信息并上传材料；3. 用人单位审核；4. 受理机构复核；5. 公示5个工作日；6. 财政资金拨付至申请人一类银行账户 |
| **申请平台** | 横琴粤澳深度合作区惠企利民服务平台 https://ycfz.hengqin.gov.cn/#/home |
| **申请材料** | 1. 身份证正反面扫描件；2. 学历学位证书（国（境）外学历附教育部认证报告）；3. 专业技术资格证书及评审文件（如有）；4. 劳动合同；5. 社保缴纳证明、个税缴纳证明；6. 《个人承诺书》《用人单位承诺书》《用人单位实质性运营自评承诺表》 |
| **审核周期** | 以官方通知为准；公示5个工作日 |
| **发放方式** | 分3年等额发放 |
| **生效时间** | 2024年（粤澳深合经发通〔2024〕4号） |
| **失效时间** | 政策有效期内（截至2025年12月31日止）符合申报条件的申请人可申报2026年度补贴，后续以官方通知为准 |
| **政策状态** | 有效（2026年度申报已启动，申报期2026年1月19日至3月6日） |
| **官方来源** | https://www.hengqin.gov.cn/macao_zh_hans/zwgk/tzgg/gg/content/post_3872566.html |
| **最后核实日期** | 2026-06-30 |

### 其他区说明

> 最后核实日期：2026-06-30

- **香洲区**：现行“香山人才计划”（香山创业英才、香山创新英才、香山青年人才、香洲工匠等）均需经评审认定或企业推荐，不属于仅凭学历、年龄、社保等基础信息即可判断的普惠性补贴，故不纳入本系统自动匹配范围。

- **斗门区**：新来珠青年人才住房“一免两减”政策属于实物化住房租金减免（首年免租、次年3折、第三年5折），非货币性现金补贴，故不纳入本系统自动匹配范围。
- **万山区**：未检索到面向普通高校毕业生、博士/硕士等群体的普惠性现金人才补贴政策。
