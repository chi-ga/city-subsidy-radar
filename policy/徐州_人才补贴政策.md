#### 1. 徐州市青年人才生活补贴（高校毕业生）

| 字段 | 内容 |
|------|------|
| **政策名称** | 徐州市青年人才生活补贴（高校毕业生） |
| **政策文号** | 《青年人才生活补贴实施细则》（徐人社发〔2023〕99号），依据《深入实施“彭城英才计划”加快推进新时代人才强市建设的若干政策》（徐委发〔2023〕25号） |
| **适用对象** | 2023年11月1日起引进的毕业五年内的博士及全日制普通高校硕士、本科毕业生（含经教育部留学服务中心国（境）外学历学位认证的留学归国人员），在我市重大科技平台、市属企业或鼓楼区、云龙区、泉山区（港务区）、铜山区、贾汪区、徐州经开区、徐州高新区的企业就业 |
| **补贴金额/标准** | 博士3000元/月、硕士1500元/月、本科1000元/月，最长领取3年（36个月）。另：非徐州籍毕业生给予每年2000元探亲交通补贴；驻徐高校毕业生给予每月300元专项补贴。以上补贴最长领取3年 |
| **学历要求** | 博士（不限全日制）；全日制普通高校硕士、本科毕业生 |
| **年龄限制** | null |
| **社保要求** | 引进后首次在我市参加职工基本养老保险且正常缴费连续满6个月及以上 |
| **申请条件** | 1. 2023年11月1日起引进；2. 毕业五年内的博士及全日制普通高校硕士、本科毕业生（含留学归国人员）；3. 在重大科技平台、市属企业或鼓楼区/云龙区/泉山区（港务区）/铜山区/贾汪区/徐州经开区/徐州高新区企业就业，且用人单位纳税应在上述区域；4. 引进后首次在徐参加职工基本养老保险且正常缴费连续满6个月及以上；5. 与用人单位签订一年及以上期限劳动合同；6. 引进后所有工作履历涉及单位均须在市区；7. “毕业5年内”指毕业后至引进徐州就业当年不超过5年 |
| **申请流程** | 1. 引才单位登录徐州市人力资源和社会保障网“一网通办”人才综合服务系统注册；2. 企业为人才申请“彭城英才卡”，人才收到审核通过短信后携本人身份证到指定银行现场办卡；3. 企业再次登录系统申领生活补贴；4. 市重大科技平台/市属企业材料由市人社局直接审核，区级企业由区人社局会同区财政局初审后提交市人社局复审；5. 补贴建议名单在徐州市人社局官网公示5个工作日；6. 公示无异议后发放至“彭城英才卡”，每次发放1个年度补贴；7. 申领次年补贴时，引才单位需在上年度补贴审核通过且职工基本养老保险再次缴满12个月后提交 |
| **申请平台** | 徐州市人力资源和社会保障网“一网通办”人才综合服务系统（http://hrss.xz.gov.cn/） |
| **申请材料** | 1. 身份证原件；2. 毕业证书原件（取得国（境）外学历学位的还需提供教育部留学服务中心出具的学历学位认证书）；3. 申报单位的营业执照原件；4. 与人才签订的1年（含）以上劳动合同原件；5. 单位纳税证明原件（申报当月的无欠税证明和近6个月内任1个月的企业完税证明）；6. 通过人力资源服务机构引进人才的，还需提供单位与人力资源服务机构签订的人才派遣合同和派遣人员名册原件；7. 申报探亲交通补贴的，还需提供非徐州籍生源证明（即高考前原户口簿“户主页”和“本人页”） |
| **审核周期** | 区人社局初审+市人社局复审，公示5个工作日 |
| **发放方式** | 经公示无异议后发放，每次发放1个年度的生活补贴、探亲交通补贴和专项补贴，发放至“彭城英才卡”。补贴发放期内人才离职但仍在徐州工作的中止补贴，新工作单位符合条件需重新申请；调离徐州的未领取部分不再发放 |
| **生效时间** | 2023年11月1日 |
| **失效时间** | 待核实 |
| **政策状态** | 有效。2025年、2026年均在持续审核发放补贴名单（2025年第一批1926人、2026年第二批765人）。本细则替代原《徐州市“555”引才工程实施方案》（徐委发〔2021〕15号）中的生活补贴、租房补贴政策，2023年11月1日前引进的人才按原政策执行 |
| **官方来源** | http://hrss.xz.gov.cn/govxxgk/014052557/2023-12-29/9f662afb-0157-4af3-8d68-b57e9185f97c.html |

### 区级政策互斥关系说明

> 最后核实日期：2026-07-01

徐州市人才补贴政策采用**市级统一政策、各区按细则执行**的模式，不存在各区独立的差异化人才补贴政策。具体情况如下：

1. **无独立区级政策**：鼓楼区、云龙区、泉山区（港务区）、铜山区（含高新区）、贾汪区、徐州经开区统一执行市级《青年人才生活补贴实施细则》（徐人社发〔2023〕99号）和《徐州市区青年人才购房券补贴实施细则》，各区补贴标准完全一致，无金额或条件的差异化。

2. **经费分担机制**：市重大科技平台和市属企业补贴经费由市级财政全额承担；鼓楼区、云龙区、泉山区（港务区）企业补贴经费由市级财政给予50%补助；铜山区（含高新区）、贾汪区、徐州经开区按照本细则执行，自行做好辖区内补贴的受理、发放和解释工作。

3. **无区级互斥关系**：由于各区执行同一市级政策、标准统一，**不存在区级政策互斥问题**。人才在上述任一区域企业就业参保均可享受同标准补贴，无需在区与区之间做选择。市级生活补贴与市级购房券补贴为两项独立政策，分别针对租房居住阶段和购房阶段，**可同时享受**，不互斥。

4. **购房券通用范围**：购房券在铜山区（高新区）、贾汪区、鼓楼区、云龙区、泉山区（港务区）、徐州经开区等城区通用，不受具体区域限制。

5. **生活补贴与购房券关系**：生活补贴为按月发放的阶段性补贴（最长3年），购房券为一次性购房支持券，二者适用阶段不同，可叠加享受。原“555”引才工程的生活补贴+租房补贴政策已被现行生活补贴政策替代，原购房补贴（徐委办〔2019〕175号）已被现行购房券政策替代，新旧政策不重复享受。
