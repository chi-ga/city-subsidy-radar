#### 1. 南昌市青年人才生活补贴（新“人才10条”）

| 字段 | 内容 |
|------|------|
| **政策名称** | 南昌市青年人才生活补贴（新“人才10条”） |
| **政策文号** | 洪人社发〔2023〕14号（南昌新“人才10条”）、洪人社发〔2024〕51号（实施细则） |
| **适用对象** | 新“人才10条”印发之日（2023年10月24日）起，首次来昌工作的全日制博士、全日制硕士及高级技师、全日制本科及技师 |
| **补贴金额/标准** | 全日制博士每人10万元；全日制硕士及高级技师每人3万元；全日制本科及技师每人1万元。生活补贴分两笔发放，每笔50%；养老保险和同期工伤保险累计缴费满6个月发放第一笔，满12个月发放第二笔 |
| **学历要求** | 全日制博士、全日制硕士、全日制本科；相应技能等级（高级技师、技师） |
| **年龄限制** | null（新实施细则未列明年龄上限） |
| **社保要求** | 首次在昌进行社会保险参保登记并同时缴纳城镇职工基本养老保险和工伤保险，首次缴费实际到账时间为在昌参保缴费起始时间；累计缴费满6个月/12个月分别对应两笔发放 |
| **申请条件** | 1. 新“人才10条”印发之日起首次来昌工作；2. 具有符合要求的全日制学历或技能等级；3. 首次在昌同时缴纳养老保险和工伤保险；4. 通过用人单位在“昌通码”/南昌市人才“港、网、窗”一体化服务平台申报；5. 申领后须在昌就业满3年，否则需退回剩余补贴 |
| **申请流程** | 1. 个人使用社保卡在“昌通码”小程序或南昌市人才“港、网、窗”一体化服务平台注册并补充信息；2. 向当前社保缴纳单位提出申请；3. 单位审核并推送至系统平台；4. 系统智能审批；5. 审批通过后自动拨付至个人社保卡金融账户 |
| **申请平台** | “昌通码”小程序、昌通码APP、南昌市人才“港、网、窗”一体化服务平台 |
| **申请材料** | 1. 申请人有效身份证件；2. 有效期内学信网验证码（留学回国人员提供教育部留学服务中心学历学位认证书）或技能等级证书；3. 与驻昌企业签订的劳动（聘用）合同或行政机关/事业单位录用证明（就业人才），或创办企业等经济实体佐证材料（创业人才）；4. 社会保障卡银行账号（需激活金融功能） |
| **审核周期** | 系统智能审批，即时反馈 |
| **发放方式** | 分两笔发放至个人社保卡金融账户，每笔50% |
| **生效时间** | 2023年10月24日 |
| **失效时间** | 待核实 |
| **政策状态** | 有效 |
| **官方来源** | https://rsj.nc.gov.cn/ncsrsj/tzgg/202404/1ebcb69c35384162bcba12dc4089b2dc.shtml |
| **最后核实日期** | 2026-07-01 |

#### 2. 南昌市青年人才租房补贴（新“人才10条”）

| 字段 | 内容 |
|------|------|
| **政策名称** | 南昌市青年人才租房补贴（新“人才10条”） |
| **政策文号** | 洪人社发〔2023〕14号（南昌新“人才10条”） |
| **适用对象** | 来昌工作并首次在昌缴纳社保的全日制博士、全日制硕士及高级技师、全日制本科及技师（不含机关事业单位在编人员） |
| **补贴金额/标准** | 每人1万元，分两笔发放，每笔50%；养老保险和同期工伤保险累计缴费满6个月发放第一笔，满12个月发放第二笔 |
| **学历要求** | 全日制博士、全日制硕士、全日制本科；或相应技能等级 |
| **年龄限制** | null |
| **社保要求** | 首次在昌同时缴纳养老保险和工伤保险，累计满6个月/12个月分别对应两笔发放 |
| **申请条件** | 1. 已按新“人才10条”规定成功领取生活补贴；2. 非机关事业单位在编人员；3. 承租房具有政策印发之日起的有效商品房租赁备案登记凭证，且承租人与申报人为同一人 |
| **申请流程** | 1. 个人在“昌通码”/南昌市人才“港、网、窗”一体化服务平台提交申请；2. 社保缴纳单位审核并推送；3. 系统智能审批；4. 拨付至个人社保卡金融账户 |
| **申请平台** | “昌通码”小程序、昌通码APP、南昌市人才“港、网、窗”一体化服务平台 |
| **申请材料** | 1. 申请人有效身份证件；2. 社会保障卡；3. 有效商品房租赁备案登记凭证；4. 已领取生活补贴的系统记录（平台自动核验） |
| **审核周期** | 系统智能审批 |
| **发放方式** | 分两笔发放至个人社保卡金融账户，每笔50% |
| **生效时间** | 2023年10月24日 |
| **失效时间** | 待核实 |
| **政策状态** | 有效 |
| **官方来源** | https://m.nc.bendibao.com/live/87535.shtm |
| **最后核实日期** | 2026-07-01 |

#### 3. 南昌市青年人才购房补贴（新“人才10条”）

| 字段 | 内容 |
|------|------|
| **政策名称** | 南昌市青年人才购房补贴（新“人才10条”） |
| **政策文号** | 洪人社发〔2023〕14号（南昌新“人才10条”） |
| **适用对象** | 全日制博士研究生、全日制硕士研究生（含高级技师），不含机关事业单位在编人员 |
| **补贴金额/标准** | 全日制博士研究生每人10万元；全日制硕士研究生（含高级技师）每人6万元。一次性发放 |
| **学历要求** | 全日制博士、全日制硕士；或高级技师 |
| **年龄限制** | null |
| **社保要求** | 养老保险和同期工伤保险累计缴费满12个月 |
| **申请条件** | 1. 已按新“人才10条”规定成功领取两笔生活补贴；2. 非机关事业单位在编人员；3. 新“人才10条”印发之日起在南昌市购买商品住房；4. 购房补贴一人仅享受一次 |
| **申请流程** | 1. 个人在“昌通码”/南昌市人才“港、网、窗”一体化服务平台注册并提交购房补贴申请；2. 社保缴纳单位核实并推送；3. 系统智能审批；4. 拨付至个人社保卡金融账户 |
| **申请平台** | “昌通码”小程序、昌通码APP、南昌市人才“港、网、窗”一体化服务平台 |
| **申请材料** | 1. 申请人有效身份证件；2. 有效期内的学信网验证码或教育部留学服务中心学历学位认证书，或技能等级证书；3. 与用人单位签订的劳动合同及用人企业工商营业执照扫描件（就业人才），或创办企业等经济实体佐证材料（创业人才）；4. 个人银行账号/社会保障卡；5. 购房合同备案或不动产登记证明 |
| **审核周期** | 系统智能审批 |
| **发放方式** | 一次性发放至个人社保卡金融账户 |
| **生效时间** | 2023年10月24日 |
| **失效时间** | 待核实 |
| **政策状态** | 有效 |

### 区级政策说明

> 最后核实日期：2026-07-01
> 数据来源：南昌市各区县人民政府官网、区人社局公开信息、南昌本地宝

经公开渠道检索，南昌市各县区（东湖区、西湖区、青云谱区、青山湖区、新建区、红谷滩区、南昌县、进贤县、安义县、湾里管理局等）目前主要执行市级“人才10条”政策，通过“昌通码”/南昌市人才“港、网、窗”一体化服务平台统一兑现。未检索到各区县独立发布、与市级政策互斥的普惠性区级人才生活/租房/购房补贴。各区县人才政策多为提供人才驿站、人才公寓、就业服务配套或针对特定产业/企业的专项扶持，不满足本系统“普通毕业生凭学历+年龄+社保即可判断”的普惠性标准，暂不纳入。
