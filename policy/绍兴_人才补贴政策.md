## 绍兴市

### 市级政策

> 最后核实日期：2026-06-30
> 数据来源：绍兴市人社局、绍兴本地宝

绍兴高校毕业生人才补贴（房票、安家、租赁）目前主要由各区、县（市）按属地政策兑现。市级层面主要提供政策框架与“绍兴人才码”“越快兑”统一受理平台，普惠性高校毕业生补贴以各区县政策为准。以下按区分别收录。

#### 3. 越城区高校毕业生房票补贴

| 字段 | 内容 |
|------|------|
| **政策名称** | 越城区高校毕业生房票补贴 |
| **政策文号** | 待核实 |
| **适用对象** | 2023年1月1日之后首次在越城区工作，与越城区注册且纳税企业签订3年以上劳动合同，承诺在越工作5年，在该企业缴纳社保，未享受过经济适用房或房改政策，在越工作后首次购房的高校毕业生、副高职称人员及高级技师、技师 |
| **补贴金额/标准** | 博士研究生50万元；副高职称人员、高级技师、硕士研究生、“双一流”高校本科毕业生20万元；其他普通高校本科毕业生、技师15万元；专科、高职毕业生（仅限紧缺专业）6万元。房票凭证自发放之日起两年内购房有效。 |
| **学历要求** | 全日制专科（高职）及以上 |
| **年龄限制** | null |
| **社保要求** | 在该企业缴纳社保满1年（实际以政策规定为准） |
| **申请条件** | 1. 2023年1月1日后首次在越工作；2. 与越城区注册纳税企业签订3年以上劳动合同；3. 承诺在越工作5年；4. 在该企业缴纳社保；5. 未享受经济适用房或房改政策；6. 在越工作后首次购房；7. 房票适用范围不包括房地产及金融、通信、电力、石化、烟草等国有企业；8. 申请房票时需减除已享受的租房补贴总额（符合绍市建设〔2020〕84号文件规定的新就业职工除外） |
| **申请流程** | 1. 购房后登录浙里办申领“绍兴人才码”完成人才认定；2. 在“房票补贴申请”模块填写信息提交申请；3. 系统自动审批，公示7天；4. 核发《绍兴市越城区人才购房房票凭证》；5. 持房票在越城区购买商品住房；6. 房票兑现资金划入所购房屋对应监管账户 |
| **申请平台** | 浙里办APP“绍兴人才码”、越快兑 https://reward.sx.gov.cn/ |
| **申请材料** | 1. 《越城区人才房票补贴申请表》；2. 申请人及配偶身份证；3. 毕业证、学位证、学信网学籍/学历在线验证报告；4. 劳动合同或事业单位聘用合同；5. 企业工商营业执照及企业当年度纳税证明；6. 来越工作5年承诺书；7. 不动产权证（二手房/现房）或合同备案证明；8. 结婚证或户口本（有共同购房人的） |
| **审核周期** | 一个季度联审一次（3/6/9/12月末审核），审核通过后1-2个月拨款 |
| **发放方式** | 房票形式一次性抵扣购房款 |
| **生效时间** | 2023-01-01 |
| **失效时间** | 待核实 |
| **政策状态** | 有效。人才所购住房满5年后方可上市交易。 |
| **官方来源** | http://m.sx.bendibao.com/live/22697.shtm |

## 互斥关系汇总

绍兴市下辖越城区、柯桥区、上虞区、诸暨市、嵊州市、新昌县的人才补贴均按税收/社保属地原则兑现。高校毕业生在同一时间段内只能在其中一个区（县市）就业参保，因此上述各区县级的安家补贴、房票/购房补贴、租赁补贴属于互斥关系。用户应根据实际工作单位注册地选择对应区县政策，系统计算总补贴金额时同一互斥组只取最高金额。
