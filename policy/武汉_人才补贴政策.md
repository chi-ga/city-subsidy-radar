#### 1. 武汉市高校毕业生租住人才租赁房租金减免

| 字段 | 内容 |
|------|------|
| **政策名称** | 武汉市高校毕业生租住人才租赁房租金减免 |
| **政策文号** | 待核实（武人才通〔2022〕8号等相关文件） |
| **适用对象** | 取得普通高校（含教育部认可的境外高等院校）全日制大专以上学历、毕业6年以内、2022年6月7日起在汉就业创业并正常缴纳社会保险、家庭在汉无自有住房的高校毕业生 |
| **补贴金额/标准** | 租住人才租赁房的，按不高于市场租金的70%缴纳房租，累计减免期限不超过3年。其中全日制博士研究生免缴2年租金（每月不超过2000元），全日制硕士研究生免缴1年租金（每月不超过1500元） |
| **学历要求** | 全日制大专及以上 |
| **年龄限制** | null |
| **社保要求** | 2022年6月7日起在汉就业创业并正常缴纳社会保险 |
| **申请条件** | 1.取得全日制大专以上学历；2.毕业6年以内；3.2022年6月7日起在汉就业创业并正常缴纳社保；4.家庭在汉无自有住房（未婚指本人及父母，已婚指夫妻双方及未成年子女） |
| **申请流程** | 1.登录武汉人才服务网或向意向区政务服务中心大学生服务窗口提交申请；2.各区人社部门2个工作日内完成学历资格审核，房管部门2个工作日内完成住房状况核查；3.审核通过后查询房源；4.选房配租并签订租赁合同；5.入住后按市场租金缴交，各区定期给予租金补贴 |
| **申请平台** | 武汉人才服务网（https://www.whzc.gov.cn）或各区政务服务中心大学生服务窗口 |
| **申请材料** | 1.《武汉市人才租赁房申请表》；2.毕业证和学历证明（学信网教育部学历证书电子注册备案表或教育部留学服务中心学历学位认证书）；3.本人及家庭成员身份证 |
| **审核周期** | 学历审核2个工作日，住房核查2个工作日 |
| **发放方式** | 入住后按市场租金缴交，各区定期给予租金补贴，优惠政策累计不超过3年 |
| **生效时间** | 2022-06-07 |
| **失效时间** | 待核实 |
| **政策状态** | 有效（2026年仍在执行，各区持续配租） |
| **官方来源** | https://m.wh.bendibao.com/live/142750.shtm |
| **最后核实日期** | 2026-06-30 |
