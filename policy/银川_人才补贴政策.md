### 一、银川市创新型大学生宜居工程生活补贴

| 项目 | 内容 |
|------|------|
| 政策名称 | 银川市创新型大学生宜居工程生活补贴 |
| 政策文号 | 银党办〔2018〕36号 |
| 适用对象 | 2017年及以后毕业的全日制普通高校（含海外全日制高校）学士/硕士/博士 |
| 补贴金额/标准 | 按年发放，补贴期限最高3年。非"双一流"博士1.8万元/年、硕士0.96万元/年、学士0.72万元/年；"双一流"博士2.4万元/年、硕士1.8万元/年、学士1.2万元/年 |
| 学历要求 | 全日制本科、硕士、博士 |
| 年龄限制 | null |
| 社保要求 | 按规定缴纳企业职工社会保险 |
| 申请条件 | 1. 2017年及以后毕业的全日制普通高校（含海外全日制高校）学士/硕士/博士；2. 在银企业就业签一年以上劳动合同（或自主创业领营业执照）；3. 按规定缴纳企业职工社会保险；4. 所学专业须符合银川市创新型人才需求目录；5. 生活补贴与购房补贴、人才公寓、共有产权房四项政策只能享受其一；6. 申请人终止在银企业服务的，生活补贴次月停发 |
| 申请流程 | 1. 填写《银川市创新型大学生宜居工程申报表》；2. 准备证明材料；3. 所在单位核查条件并加具推荐意见；4. 报送各区/园区或行业主管部门审核；5. 加具推荐意见后报送银川人才工作服务局审核；6. 审核通过后发放 |
| 申请平台 | "银川人才云"（www.ycrcy.gov.cn）；材料报送至银川人才工作服务局人才服务窗口 |
| 申请材料 | 1. 《银川市创新型大学生宜居工程申报表》；2. 本人有效身份证；3. 毕业证书；4. 学位证书；5. 学信网学籍证明；6. 就业人员：与公司签订的就业劳动合同；7. 创业人员：营业执照、纳税证明；8. 在银缴纳企业职工养老保险证明 |
| 审核周期 | 须自毕业证书签发之日起2年内提出申请 |
| 发放方式 | 按年发放，补贴期限最高3年 |
| 生效时间 | 2018-10-29 |
| 失效时间 | null（持续执行） |
| 政策状态 | 有效 |
| 官方来源 | 银川本地宝（http://m.yc.bendibao.com/mip/44555.shtm） |
| 最后核实日期 | 2026-07-01 |

### 二、银川市创新型大学生宜居工程购房补贴

| 项目 | 内容 |
|------|------|
| 政策名称 | 银川市创新型大学生宜居工程购房补贴 |
| 政策文号 | 银党办〔2018〕36号 |
| 适用对象 | 2017年及以后毕业的全日制普通高校（含海外全日制高校）学士/硕士/博士 |
| 补贴金额/标准 | 在企业服务期满三年后一次性发放。非"双一流"博士10万元、硕士6万元、学士4万元；"双一流"博士12万元、硕士8万元、学士6万元 |
| 学历要求 | 全日制本科、硕士、博士 |
| 年龄限制 | null |
| 社保要求 | 按规定缴纳企业职工社会保险 |
| 申请条件 | 1. 条件同生活补贴基本要求；2. 须提供购房合同或房产证；3. 须在企业服务满三年后领取；4. 生活补贴与购房补贴、人才公寓、共有产权房四项政策只能享受其一 |
| 申请流程 | 1. 填写《银川市创新型大学生宜居工程申报表》；2. 准备证明材料（含购房合同/房产证）；3. 所在单位核查条件并加具推荐意见；4. 报送各区/园区或行业主管部门审核；5. 加具推荐意见后报送银川人才工作服务局审核；6. 审核通过后发放（须在企业服务满三年后） |
| 申请平台 | "银川人才云"（www.ycrcy.gov.cn） |
| 申请材料 | 1. 《银川市创新型大学生宜居工程申报表》；2. 本人有效身份证；3. 毕业证书；4. 学位证书；5. 学信网学籍证明；6. 就业人员：与公司签订的就业劳动合同；7. 创业人员：营业执照、纳税证明；8. 在银缴纳企业职工养老保险证明；9. 购房合同或房产证原件及复印件；10. 已婚需提供结婚证原件及复印件 |
| 审核周期 | 须自毕业证书签发之日起2年内提出申请，企业服务满三年后领取 |
| 发放方式 | 在企业服务期满三年后一次性发放 |
| 生效时间 | 2018-10-29 |
| 失效时间 | null（持续执行） |
| 政策状态 | 有效 |
| 官方来源 | 银川本地宝（http://m.yc.bendibao.com/mip/44555.shtm） |
| 最后核实日期 | 2026-07-01 |

### 三、银川市西夏区购房补贴

| 项目 | 内容 |
|------|------|
| 政策名称 | 银川市西夏区购房补贴 |
| 政策文号 | 银西政规发〔2026〕2号 |
| 适用对象 | 应届全日制专科、统招专升本、全日制本科、硕士、博士毕业生 |
| 补贴金额/标准 | 应届全日制专科、统招专升本、全日制本科5000元/套，硕士20000元/套，博士30000元/套（学信网认定最高学历）。各项补贴可叠加享受，每户最高不超过5万元 |
| 学历要求 | 全日制专科、本科、硕士、博士 |
| 年龄限制 | null |
| 社保要求 | null |
| 申请条件 | 1. 购买西夏区行政区划范围内新建商品房住宅（不含二手房、商业、公寓等）；2. 补贴资金总计300万元，先到先得；3. 失效时间2026年6月30日 |
| 申请流程 | 1. 认购西夏区新建商品住宅，签订《商品房买卖合同》，完成网签备案；2. 购房人向西夏区住房城乡建设和交通局提交补贴申请资料；3. 资料审核、登记信息；4. 社会公示（≥5个工作日）；5. 公示无异议后1个月内发放 |
| 申请平台 | 银川市西夏区市民服务大厅（二楼9号窗口） |
| 申请材料 | 1. 购房人身份证、户口簿复印件（多个购房人需全部身份证复印件）；2. 《商品房买卖合同》正本原件及首页至签字页复印件；3. 首付款缴纳凭证（刷卡小票/转账凭证/发票）原件及复印件；4. 购房人名下有效储蓄卡复印件；5. 学历学位证件及学信网学历证明文件（在读生提供院校在读证明）；6. 存在多个购房人的需提交授权委托书指定收款人；7. 审核部门要求的其他材料 |
| 审核周期 | 2026年1月22日至2026年6月30日（补贴资金300万元补完为止） |
| 发放方式 | 一次性发放 |
| 生效时间 | 2026-01-22 |
| 失效时间 | 2026-06-30 |
| 政策状态 | 有效（时限性政策） |
| 官方来源 | 银川市西夏区人民政府官网（http://www.ycxixia.gov.cn/zwgk/bmxxgkmn/xxqzfbgs/fdzdgknr/gfxwj_9807/qzf_72541/202601/t20260122_5144933.html） |
| 最后核实日期 | 2026-07-01 |
