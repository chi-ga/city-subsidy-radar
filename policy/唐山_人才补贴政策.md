# 唐山市人才补贴政策汇编

> 数据来源：唐山市人力资源和社会保障局、唐山人才综合服务平台、唐山高新区管委会、唐山本地宝
> 最后核实日期：2026-07-01
> 说明：唐山市人才政策统称"凤凰英才"计划（3.0 升级版），核心文件为《唐山市青年人才奖补实施细则（修订版）》（唐人社〔2024〕30号）及《关于调整明确青年人才奖补政策范围的通知》（唐才办〔2024〕6号）。本汇编保留面向普通高校毕业生、条件可量化的补贴政策，排除"一事一议"、顶尖人才认定类政策。

## 一、市级政策（凤凰英才 3.0 版 · 青年人才奖补）

### 政策 1：唐山市博士研究生生活补贴

| 字段 | 内容 |
| --- | --- |
| 政策名称 | 唐山市博士研究生生活补贴 |
| 政策文号 | 《唐山市青年人才奖补实施细则（修订版）》（唐人社〔2024〕30号）、《关于调整明确青年人才奖补政策范围的通知》（唐才办〔2024〕6号） |
| 适用对象 | 2021年10月21日以来，全职引进或培养的在唐工作博士研究生（同时具有毕业证和学位证） |
| 补贴金额/标准 | 每月 3000 元，发放 60 个月（5 年），合计最高 18 万元 |
| 学历要求 | 博士研究生 |
| 年龄限制 | 50 周岁及以下 |
| 社保要求 | 在唐工作 5 年合同期内正常缴纳社会保险 |
| 申请条件 | ①全日制/统招博士研究生，同时具有学历证和学位证；②与在唐用人单位签订 5 年及以上劳动合同；③全职引进或培养；④正常缴纳社会保险；⑤在唐最低服务 5 年。生活补贴与购房补贴二者只能选择一项 |
| 申请流程 | ①用人单位在唐山人才综合服务平台 PC 端注册，提交材料到唐山市民服务中心 B 区 611 室线下备案；②个人通过"唐山人才综合服务平台"微信公众号注册并提出补贴备案申请；③平台审核备案（满一年后方可申领）；④个人登录平台上传材料提交补贴申领；⑤平台初审→复审→报同级财政部门核定；⑥公示 7 天后发放 |
| 申请平台 | 唐山人才综合服务平台 PC 端 https://rcpc.hfclass.com ；"唐山人才综合服务平台"微信公众号 |
| 申请材料 | 身份证；学历证书；学位证书；劳动合同；社会保险缴纳证明；用人单位营业执照（或事业单位法人证书） |
| 审核周期 | 备案满一年后方可申领；每年一、三季度集中申领；公示 7 天 |
| 发放方式 | 每满半年发放一次，拨付至申领人个人银行账户 |
| 生效时间 | 2021-10-21 |
| 失效时间 | null |
| 政策状态 | 现行有效 |
| 官方来源 | 唐山人才综合服务平台、唐山市人力资源和社会保障局、唐山本地宝 |
| 最后核实日期 | 2026-07-01 |

## 三、区级政策互斥关系说明

1. **区级政策之间互斥**：唐山市各区（含高新区）自行制定的人才补贴政策属于互斥关系，同一申请人不得同时享受两个及以上区级同类人才补贴。本汇编收录的唐山高新区"高学历人才生活补贴"（政策 7）与"高学历人才购房支持"（政策 8）为同一区级方案下的不同补贴类型；若唐山市其他区（如路北区、路南区、丰润区、丰南区、曹妃甸区等）后续出台同类区级人才补贴，亦与高新区政策互斥，只能择一享受。

2. **市级与区级同类补贴不重复享受**：唐山高新区政策明确，非首次来唐的申请人须"未享受唐山市'凤凰英才计划'政策"方可申领区级补贴，即市级青年人才奖补（政策 1-6）与高新区高学历人才生活补贴（政策 7）属同类生活补贴，原则上不重复享受，按"就高从优、不重复"原则执行。

3. **市级政策内部互斥**：
   - 博士生活补贴（政策 1）与博士购房补贴（政策 2）二者只能选择一项；
   - 硕士生活补贴（政策 3）与硕士购房补贴（政策 4）二者只能选择一项；
   - 同一申请人按就业单位类型（企业/其他单位、国企/民企）对应适用相应标准，不可跨类叠加。

4. **服务年限约束**：市级政策要求在唐最低服务 5 年；区级政策要求合同期内不得提前解除劳动合同离区，否则终止补贴发放。
