# 盐城人才补贴政策

> 最后核实日期：2026-07-01
>
> 政策依据：《关于深入实施"黄海明珠人才计划"的若干政策》（盐发〔2024〕2号），由中共盐城市委、盐城市人民政府发布，盐城市人力资源和社会保障局牵头执行；配套细则《黄海明珠人才计划2.0版人才卡发放实施细则（试行）》（2025年3月5日发布）。

## 一、盐城市黄海明珠人才计划生活补贴

| 项目 | 内容 |
|------|------|
| 政策名称 | 盐城市黄海明珠人才计划生活补贴（普通高校毕业生） |
| 政策文号 | 《关于深入实施"黄海明珠人才计划"的若干政策》（盐发〔2024〕2号）；《黄海明珠人才计划2.0版人才卡发放实施细则（试行）》 |
| 适用对象 | 持三星级黄海明珠人才卡、符合普通高校毕业生条件的全日制博士研究生、硕士研究生和本科生（技师班毕业生），被盐城市企业和社会组织引进或来盐自主创业 |
| 补贴金额/标准 | 全日制博士研究生3000元/月；全日制硕士研究生1500元/月；全日制本科生（技师班毕业生）1000元/月；发放36个月 |
| 学历要求 | 全日制普通高校本科及以上（含技工院校技师班毕业生）；海外学子同等享受 |
| 年龄限制 | 35周岁以下（以毕业后在盐首次参保缴费年度年龄为准） |
| 社保要求 | 首次在盐参加各项社会保险且正常缴费（除学生医保外，在盐无其他社会保险参保缴费记录） |
| 申请条件 | 1.被盐城市企业和社会组织引进或来盐自主创业；2.首次在盐参加各项社会保险且正常缴费；3.年龄35周岁以下；4.全日制普通高校本科及以上学历 |
| 申请流程 | 1.下载"我的盐城"APP或登录盐城市人力资源和社会保障网黄海明珠人才专栏；2.点击"人才申报"，根据自身情况申请三星级人才绿卡；3.签署信用承诺，上传材料；4.黄海明珠人才服务窗口审核年龄、首次参保时间、企业注册、证书真实性；5.审核通过后由合作银行制作人才卡并邮寄；6.持卡后在系统内申请生活补贴 |
| 申请平台 | "我的盐城"APP、江苏政务服务网盐城旗舰店黄海明珠人才专栏、盐城市人力资源和社会保障网（https://jsychrss.yancheng.gov.cn） |
| 申请材料 | 1.本人身份证；2.毕业证书；3.教育部电子学历注册备案表（本科以上需提供学位证书）；4.单位营业执照或注册登记证书；5.海外留学人员另需提供教育部留学服务中心出具的《国外学历学位认证书》 |
| 审核周期 | 材料审核后制卡发放，补贴自参保次月起按月到账 |
| 发放方式 | 按月发放，连续计算36个月（与参保缴费相对应，中途中断不向后延期） |
| 生效时间 | 2024-01-01 |
| 失效时间 | null |
| 政策状态 | 有效。黄海明珠人才计划2.0版持续执行中 |
| 官方来源 | 盐城市人力资源和社会保障局（https://jsychrss.yancheng.gov.cn/art/2025/3/5/art_31881_4297030.html）；盐城本地宝（http://m.yancheng.bendibao.com/news/13077.shtm） |
| 最后核实日期 | 2026-07-01 |

### 区级政策互斥关系说明

盐城市黄海明珠人才计划适用于大丰区、盐都区、亭湖区、盐城经济技术开发区、盐南高新区和市本级，市区范围内执行统一标准，不因行政区划不同而互斥。各县（市）参照本细则执行，但具体标准可能存在差异，人才流动到市区企业的须重新申请人才卡。上述五项补贴中，生活补贴、租房补贴、购房补贴、探亲交通补贴可分别申请（生活/租房/探亲为不同类型可叠加；购房补贴与生活/租房补贴为不同类型可叠加享受）。
