#### 1. 成都市新引进急需紧缺人才安家补贴

| 字段 | 内容 |
|------|------|
| **政策名称** | 成都市新引进急需紧缺人才安家补贴 |
| **政策文号** | 《成都市引进急需紧缺人才实施办法》（2026年1月1日生效） |
| **适用对象** | **2000元/月档**：取得全日制博士学历学位证书后一年内引进到成都市企业工作的人才；或符合《成都市人才开发指引》中紧缺度3星岗位要求的人才。**3000元/月档**：满足《成都市人才分类目录》A、B、C类条件之一的人才 |
| **补贴金额/标准** | 2000元/人·月或3000元/人·月，从引进次月起发放3年，合计7.2万元或10.8万元 |
| **学历要求** | 2000元档：全日制博士或紧缺度3星岗位不限学历；3000元档：不限学历（以成都市人才认定证书为准） |
| **年龄限制** | null |
| **社保要求** | 在成都市企业就业并缴纳社保，劳动合同签订单位与社保缴纳单位一致 |
| **申请条件** | 1. 取得全日制博士学历学位证书后一年内引进到成都市企业就业；或符合引进当年《成都市人才开发指引》紧缺度3星岗位要求（2000元档）；或满足《成都市人才分类目录》A、B、C类条件之一（3000元档）；2. 须由企业引进，个人不可单独申报；3. 企业按属地原则（以企业注册地为准）申报 |
| **申请流程** | 1. 企业登录"成都市急需紧缺人才服务管理系统"；2. 填写人才信息并上传材料；3. 企业按属地原则提交申请；4. 审核通过后按月发放 |
| **申请平台** | 成都市急需紧缺人才服务管理系统 |
| **申请材料** | 1. 《成都市新引进人才安家补贴申请表》；2. 企业营业执照；3. 引进人才有效身份证件复印件、劳动合同、个人近6个月连续缴纳社保证明；4. 《承诺书》；5. 学历学位证书、6个月有效期学信网学历认证报告（国（境）外高校学历学位证书的，提供教育部留学服务中心认证报告）；6. 成都市人才认定证书（3000元档）；7. 其他根据审核需要须补充提供的材料 |
| **审核周期** | 每年5月至7月集中申报，受理上一年度补贴 |
| **发放方式** | 从引进的次月起3年内按月发放 |
| **生效时间** | 2026-01-01 |
| **失效时间** | 待核实 |
| **政策状态** | 有效。2026年度集中申报至2026年7月31日前 |
| **官方来源** | https://m.cd.bendibao.com/fangchan/210204.shtm |
| **最后核实日期** | 2026-06-30 |

#### 2. 成都县市新城新引进人才生活补贴

| 字段 | 内容 |
|------|------|

| **申请平台** | 成都市急需紧缺人才服务管理系统 |
| **申请材料** | 1. 《成都市新引进人才安家补贴（县市新城新引进人才生活补贴）申请表》；2. 企业营业执照；3. 引进人才有效身份证件复印件、劳动合同、个人近6个月连续缴纳社保证明；4. 《承诺书》；5. 学历学位证书、6个月有效期学信网学历认证报告（国（境）外高校学历学位证书的，提供教育部留学服务中心认证报告）；6. 其他根据审核需要须补充提供的材料 |
| **审核周期** | 每年5月至7月集中申报，受理上一年度补贴 |
| **发放方式** | 从引进的次月起3年内按月发放，补贴为税后资金 |
| **生效时间** | 2026-01-01 |
| **失效时间** | 待核实 |

| **审核周期** | 按项目运营单位审核周期 |

| **生效时间** | 2026-04-01 |
| **失效时间** | 2031-04-01 |
| **政策状态** | 有效。该细则在征集意见阶段，2026年4月1日起正式实施。需先取得成都市人才认定资格 |
