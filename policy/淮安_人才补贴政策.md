# 淮安人才补贴政策

> 最后核实日期：2026-07-01
>
> 政策依据：《"淮安市333产业人才"集聚行动方案（试行）》（淮办发〔2021〕11号）、《关于加强淮安市人才安居保障工作的实施意见》（淮政发〔2022〕10号），由淮安市委人才工作领导小组办公室、淮安市人力资源和社会保障局发布并执行。

## 一、淮安市人才购房补贴

| 项目 | 内容 |
|------|------|
| 政策名称 | 淮安市人才购房补贴 |
| 政策文号 | 《关于组织申报2025年度淮安市人才购房补贴的通知》（淮人社发〔2025〕34号），依据《"淮安市333产业人才"集聚行动方案（试行）》（淮办发〔2021〕11号）、《关于加强淮安市人才安居保障工作的实施意见》（淮政发〔2022〕10号）、《关于印发<"淮安市333产业人才"集聚行动方案（试行）>实施细则的通知》（淮人才办〔2021〕12号） |
| 适用对象 | 到淮安市"333产业"等实体经济企业就业的全日制本科及以上学历大学生（2021年4月22日及之后来淮就业人员） |
| 补贴金额/标准 | "333产业"企业博士30万元、硕士15万元、本科生5-10万元（双一流高校享受更高标准）；驻淮高校优秀博士15万元。一次性发放至社保卡金融账户，补贴金额不高于购房交易金额 |
| 学历要求 | 全日制本科及以上学历（全日制技工院校技师班/预备技师班毕业生视同本科；留学生需教育部留学服务中心认证） |
| 年龄限制 | null |
| 社保要求 | 首次在淮安缴纳基本养老保险等社会保险，须在来淮留淮就业5年内提出有效申请（从首次在淮安缴纳基本养老保险等社会保险之日起算） |
| 申请条件 | 1. 2021年4月22日（含）之后来淮就业；2. 到淮安市"333产业"等实体经济企业就业的全日制本科及以上学历大学生；3. 首次在淮安缴纳基本养老保险等社会保险；4. 在淮安区域内购买规划用途为住宅的首套商品房并取得不动产权证（购买前在淮安区域内无其它商品房不动产权登记、商品房交易和《商品房买卖合同》网签记录）；5. 购房时间以网签《商品房买卖合同》时间为准；6. 须在来淮留淮就业5年内提出有效申请 |
| 申请流程 | 1. 登录淮安政务服务网（https://ha.jszwfw.gov.cn），选择"财政奖补"→"人才奖励"对应栏目进行申报；2. 同意"享受购房补贴的所购商品房5年内不上市交易"承诺；3. 按要求上传附件材料（原件扫描件，以材料名称命名）；4. 各县区（园区）负责受理在本县区（园区）纳税单位引进人才申请并核查信息，市人才中心负责受理在市库纳税单位引进人才申请；5. 经市委人才办主任会议研究通过后，拟发放名单在淮安市人社局网站公示5个工作日；6. 公示无异议后，购房补贴发放至人才本人社会保障卡金融账户 |
| 申请平台 | 淮安政务服务网（https://ha.jszwfw.gov.cn），"财政奖补"→"人才奖励"栏目 |
| 申请材料 | 1. 《淮安市人才购房补贴申请表》或《淮安市人才"购房券"兑现申请表》；2. 《不动产权证书》（期房提供已备案的《商品房买卖合同》）；3. 身份证（身份证明）、学历学位证书（职业资格证书）、入选人才项目文件；4. 与就职单位签订的有效劳动合同；5. 社会保险权益记录单 |
| 审核周期 | 申报时间2025年12月1日—12月31日，审核时间2025年12月1日—2026年1月15日，公示5个工作日 |
| 发放方式 | 一次性发放至人才本人社会保障卡金融账户（市库纳税单位由市人才中心发放，县区/园区纳税企业由县区/园区本级发放） |
| 生效时间 | 2021年4月22日（2021年4月22日之前来淮就业人员按原政策执行） |
| 失效时间 | null |
| 政策状态 | 有效。2025年度申报正在进行（2025年12月1日—12月31日）。人才须在来淮5年内提出有效申请，5年内未提出视为自动放弃。所购商品房5年内不得上市交易。未满5年服务期限离淮的，由用人单位负责追缴退还，按实际服务年限每不满1年以20%比例退还。近3年ARWU世界大学学术排名前200名国（境）外大学毕业生享受国内"双一流"高校（学科）相应学历补贴标准 |

### 区级政策互斥关系说明

> 最后核实日期：2026-07-01

淮安市人才购房补贴政策采用**市级统一政策、各区按属地受理**的模式，不存在各区独立的差异化人才补贴政策。具体情况如下：

1. **无独立区级政策**：清江浦区、淮安区、淮阴区、洪泽区、涟水县、盱眙县、金湖县、淮安经济技术开发区、工业园区、生态文旅区等不设独立的差异化人才购房补贴政策，统一执行市级补贴标准。

2. **经费分担机制**：市库纳税单位引进人才购房补贴由市人才中心发放；县区（园区）纳税企业引进人才购房补贴由县区（园区）本级发放。各区受理在本区纳税单位引进人才的申请，但补贴标准全市统一。

3. **无区级互斥关系**：由于各区执行同一市级政策、标准统一，**不存在区级政策互斥问题**。人才在淮安市任一区域企业就业参保并购房均可享受同标准补贴，无需在区与区之间做选择。

4. **新旧政策衔接**：2021年4月22日之前来淮就业创业人才按原政策执行；2021年4月22日（含）之后来淮就业人员适用现行"333产业"人才购房补贴政策。新旧政策不重复享受。
