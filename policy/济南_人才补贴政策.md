#### 1. 济南市新就业职工租赁住房补贴

| 字段 | 内容 |
|------|------|
| **政策名称** | 济南市新就业职工租赁住房补贴 |
| **政策文号** | 济建发〔2024〕50号 |
| **适用对象** | 在济南市历下区、市中区、槐荫区、天桥区、历城区、长清区、章丘区、济阳区、莱芜区、钢城区及济南高新区、市南部山区、济南新旧动能转换起步区（相关区）依法注册的企业、民办非企业单位、社会团体，以及经省科技厅备案的新型研发机构、其他经济和社会组织全职就业的新就业职工 |
| **补贴金额/标准** | 博士1500元/月、硕士1000元/月、本科700元/月、专科500元/月；补贴期限累计最长不超过3年，最短不低于1个月 |
| **学历要求** | 全日制本科、专科及以上学历（博士、硕士由相关企事业单位引进的可不受部分单位性质限制） |
| **年龄限制** | null |
| **社保要求** | 至申请之日，在济连续缴纳养老保险6个月（补缴的不计算连续）且处于在缴状态 |
| **申请条件** | 1. 具有本市户籍或持有合法有效的《济南市居住证》；2. 至申请之日，毕业不超过5年，且在相关区依法登记注册的各类企业、民办非企业单位、社会团体稳定就业（确定1年及以上期限的稳定劳动关系），驻济科研院所、医疗机构和市属事业单位引进的全日制博士研究生可不受单位性质限制；3. 在济连续缴纳养老保险6个月且处于在缴状态；4. 在相关区无自有住房且租赁住房居住；5. 未正在享受经济适用住房、公共租赁住房、企业新引进研究生租房和生活补贴等政策 |
| **申请流程** | 1. 申请人向用人单位提出申请并填写《新就业职工租赁住房补贴申请表》；2. 用人单位核实确认后，提交至单位注册地区县住房城乡建设部门；3. 区县住建部门会同民政、人社、社保、住房交易、不动产登记等系统信息比对及人工核查；4. 审核通过后按月或按季度发放至申请人银行账户 |
| **申请平台** | 济南人才网（https://rc.jinan.gov.cn/）租赁住房补贴申报系统，或用人单位线下提交至区县住建部门 |
| **申请材料** | 1. 申请人身份证件；2. 户口簿本人单页或公安机关出具的户籍证明，或有效期内《济南市居住证》；3. 毕业证、学位证；4. 劳动合同或聘用合同；5. 社会保险缴纳证明；6. 租赁住房合同或相关居住证明；7. 授权书和承诺书；8. 家庭住房情况相关材料（已婚者需提供配偶及未成年子女住房信息） |
| **审核周期** | 信息自动比对+人工核查，具体周期以各区县住建部门为准 |
| **发放方式** | 按月或按季度发放至申请人银行账户 |
| **生效时间** | 2024年（济建发〔2024〕50号） |
| **失效时间** | 待核实 |
| **政策状态** | 有效。适用于济南市相关区，平阴县、商河县可参照执行 |
| **官方来源** | http://jnhrss.jinan.gov.cn/ |
| **最后核实日期** | 2026-06-30 |

#### 2. 济南市高层次人才生活和租房补贴

| 字段 | 内容 |
|------|------|
| **政策名称** | 济南市高层次人才生活和租房补贴 |
| **政策文号** | 《济南市人才服务支持政策（30条）》《济南市人才发展环境政策（30条）》实施细则（2025版）人才住房保障实施细则（济人才办发〔2025〕4号） |
| **适用对象** | 1. 驻济高校新引进的全球TOP200高校毕业的博士研究生；2. 驻济科研院所、医疗机构和市属及各区县事业单位新引进的全日制博士研究生；3. 驻济企业新引进的全日制博士、硕士研究生（A、B、C、D类高层次人才按另行标准执行，本系统不作普惠匹配） |
| **补贴金额/标准** | 博士研究生1500元/月、硕士研究生1000元/月；最长补贴3年 |
| **学历要求** | 全日制博士、硕士研究生 |
| **年龄限制** | null |
| **社保要求** | 与用人单位签订3年及以上劳动合同（聘用合同），并在济正常缴纳社会保险 |
| **申请条件** | 1. 首次提报至区县时，毕业时间不超过3年；2. 与用人单位签订3年及以上劳动合同（聘用合同），并在济正常缴纳社会保险；3. 高层次人才生活补贴与人才公寓、购房补贴、租赁住房补贴、保障性租赁住房不同时享受；已申请并享受高层次人才购房补贴政策的，不再享受生活补贴 |
| **申请流程** | 1. 用人单位在济南人才网注册单位账户，为申报人开通申报权限；2. 申报人通过生活和租房补贴申报系统网上申报，填报个人信息；3. 用人单位核实确认后，按属地原则提交至区县人社部门；4. 区县及市级审核，公示后拨付 |
| **申请平台** | 济南人才网（https://rc.jinan.gov.cn/）生活和租房补贴申报系统 |
| **申请材料** | 1. 申请人身份证件；2. 毕业证、学位证（留学回国人员提供教育部学历学位认证书）；3. 劳动合同或聘用合同；4. 社会保险缴纳证明；5. 用人单位出具的在职证明；6. 个人承诺书、授权书；7. 房屋租赁合同（A、B、C、D类人才提供，全日制博士/硕士一般不需要） |
| **审核周期** | 常态化申报，审核周期以区县人社部门为准 |
| **发放方式** | 自补贴申报审核通过次月起按月发放 |
| **生效时间** | 2025年（济人才办发〔2025〕4号） |
| **失效时间** | 待核实 |
| **政策状态** | 有效。本政策中A、B、C、D类高层次人才通道需人才认定，不纳入系统自动匹配；全日制博士/硕士研究生通道按学历条件直接匹配 |
| **官方来源** | https://rc.jinan.gov.cn/ |
| **最后核实日期** | 2026-06-30 |

#### 3. 济南市高层次人才购房补贴

| 字段 | 内容 |
|------|------|
| **政策名称** | 济南市高层次人才购房补贴 |
| **政策文号** | 《济南市人才服务支持政策（30条）》《济南市人才发展环境政策（30条）》实施细则（2025版）人才住房保障实施细则（济人才办发〔2025〕4号） |
| **适用对象** | 1. 驻济企业就职且经市委人才办认定的A、B、C、D类高层次人才；2. 纳入E类人才认定范围且符合就职要求的全日制硕博研究生（驻济企业新引进/驻济高校全球TOP200博士/驻济科研院所医疗机构及事业单位新引进博士） |
| **补贴金额/标准** | A类人才"一人一策、一事一议"；B类人才购房金额50%最高100万元；C类人才购房金额50%最高70万元；D类人才购房金额50%最高40万元；博士研究生15万元一次性；硕士研究生10万元一次性 |
| **学历要求** | 全日制博士、硕士研究生（需纳入E类人才认定） |
| **年龄限制** | null |
| **社保要求** | 在济正常缴纳社会保险 |
| **申请条件** | 1. 企业申请人签订3年及以上劳动合同，事业单位申请人签订聘用合同或具有组织部门任命文件，均在济正常缴纳社会保险；2. 申请家庭购房网签备案时间需在2019年9月（含）以后，且所购住房网签备案时间在申请人毕业5年内；3. 所购住房为申请家庭在济拥有的唯一住房（不含购买直系亲属住房）；4. 购房之日前3年内申请家庭在济无其他住房登记信息和住房交易记录；5. 已申请并享受高层次人才购房补贴政策的，不再享受生活补贴 |
| **申请流程** | 1. 用人单位通过济南人才网购房补贴系统注册账户；2. 申请人填报信息并填写个人承诺书、授权书，提交用人单位核实；3. 用人单位提交至单位注册地区县住房城乡建设部门；4. 区县住建部门信息比对及人工核查；5. 市保障性住房服务中心汇总审核通过名单并在济南人才网公示5个工作日；6. 公示期满无异议后发放至申请人社会保障卡金融账户 |
| **申请平台** | 济南人才网（https://rc.jinan.gov.cn/）购房补贴系统 |
| **申请材料** | 1. 申请人身份证件；2. 毕业证、学位证；3. 劳动合同或聘用合同/任命文件；4. 社会保险缴纳证明；5. 购房网签备案材料；6. 家庭住房情况证明；7. 个人承诺书、授权书；8. 社会保障卡金融账户信息 |
| **审核周期** | 人工核查10个工作日，公示5个工作日 |
| **发放方式** | D类及以上分3年发放（首年40%，第2、3年各30%）；博士、硕士研究生一次性发放 |
| **生效时间** | 2019年9月（网签备案时间要求） |
| **失效时间** | 待核实 |
| **政策状态** | 有效。本政策需纳入E类及以上人才认定，不纳入系统自动匹配，仅作政策库展示 |
| **官方来源** | https://rc.jinan.gov.cn/ |
| **最后核实日期** | 2026-06-30 |

### 区级政策说明

> 最后核实日期：2026-06-30
> 数据来源：济南市历下区、市中区、历城区等区人民政府官网及人社局公开信息

经公开渠道检索，济南市目前各区（历下区、市中区、槐荫区、天桥区、历城区、长清区、章丘区、济阳区、莱芜区、钢城区及高新区、南部山区、新旧动能转换起步区）主要执行市级人才住房保障政策，未检索到各区独立发布的、与市级政策互斥的普惠性区级人才生活/租房/购房补贴。上述市级政策中的租赁住房补贴和高层次人才生活补贴均按单位注册地或属地原则由各区县受理审核，同一申请人不能重复享受市级同类补贴。
