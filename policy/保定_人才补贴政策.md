# 保定市人才补贴政策

> 最后核实日期：2026-07-01
>
> 政策依据：《关于支持高校毕业生和技能人才在保就业创业的若干措施》（"保定人才十条"）、《关于鼓励高校毕业生来保留保发展的若干扶持政策》（保办发〔2021〕6号）及其《有关事项操作指引》（保才办〔2022〕1号），由保定市人力资源和社会保障局、市公安局、市住建局等部门发布并执行。

---

## 一、保定市高校毕业生落户补贴

| 项目 | 内容 |
|------|------|
| 政策名称 | 保定市高校毕业生落户补贴 |
| 政策文号 | 保才办〔2022〕1号《〈关于鼓励高校毕业生来保留保发展的若干扶持政策〉有关事项操作指引》（依据保办发〔2021〕6号、"保定人才十条"） |
| 适用对象 | 在我市落户的非保定籍高校毕业生 |
| 补贴金额/标准 | 1000元/人，一次性发放 |
| 学历要求 | 高校全日制专科及以上毕业生 |
| 年龄限制 | null（要求2022年1月1日后毕业） |
| 社保要求 | 提交申请时，前1个月的我市城镇职工养老保险缴费证明 |
| 申请条件 | 1. 2022年1月1日后毕业；2. 在我市企业、民办非企业就业或自主创业；3. 高校全日制专科及以上毕业生；4. 非保定籍高校毕业生并在我市落户 |
| 申请流程 | 1. 用人单位审核汇总申请材料后，报营业执照住所地县（市、区）受理部门；2. 受理部门会同同级公安部门对申请人户籍信息进行审核；3. 审核通过人员由受理部门公示，公示期不少于5个工作日；4. 公示无异议的，市辖七区将名单报市人社局，市人社局审核汇总后报市财政局，市财政局将资金拨付至市辖七区后由受理部门发放给申请人；省财政直管县（市）由本级财政负担，参照办理 |
| 申请平台 | 各县（市、区）人社局受理部门（市人社局官网 https://rsj.baoding.gov.cn 下载申请表） |
| 申请材料 | 1.《落户补贴申请表》；2.申请人居民身份证及户口簿（户主页及本人页）；3.申请人毕业证书；4.中国高等教育学生信息网验证打印的《教育部学历证书电子注册备案表》，取得国（境）外学历学位的须提供教育部留学服务中心《国（境）外学历学位认证书》；5.自主创业的申请人须提供在我市登记注册的营业执照，共同创业（合伙创业）的还须提供市场监管部门留存备案的公司章程（合伙协议）；6.提交申请时，前1个月的我市城镇职工养老保险缴费证明；7.本地社会保障卡（已激活金融功能）。以上材料均需提供原件及加盖单位公章的复印件 |
| 审核周期 | 法定工作日即来即办，公示不少于5个工作日 |
| 发放方式 | 一次性发放至社保卡金融账户 |
| 生效时间 | 2022-01-01（保才办〔2022〕1号） |
| 失效时间 | null（持续执行，2025年仍有公示发放） |
| 政策状态 | 有效 |
| 官方来源 | 保定就业（保定市人社局）（https://rsj.baoding.gov.cn/bdjy/wap/index.php?a=show&c=article&id=65） |
| 最后核实日期 | 2026-07-01 |

## 二、保定市高校毕业生租房补贴

| 项目 | 内容 |
|------|------|
| 政策名称 | 保定市高校毕业生租房补贴 |
| 政策文号 | 保才办〔2022〕1号《〈关于鼓励高校毕业生来保留保发展的若干扶持政策〉有关事项操作指引》（依据保办发〔2021〕6号、"保定人才十条"） |
| 适用对象 | 在我市企业、民办非企业就业或自主创业的高校全日制专科及以上毕业生 |
| 补贴金额/标准 | 博士2000元/月、硕士1000元/月、本科500元/月、大专300元/月，最长不超过36个月。租房补贴与购房补贴不能同时享受 |
| 学历要求 | 高校全日制专科、本科、硕士、博士 |
| 年龄限制 | null（要求2022年1月1日后毕业） |
| 社保要求 | 首次申请须提交在我市1个月以上的城镇职工养老保险缴费证明；之后申请须提交享受补贴期间在我市连续缴纳城镇职工养老保险证明。间断或停止缴纳的，后续补贴停止发放且不再受理 |
| 申请条件 | 1. 2022年1月1日后毕业；2. 在我市企业、民办非企业就业或自主创业；3. 高校全日制专科及以上毕业生；4. 享受补贴期间须在我市连续缴纳城镇职工养老保险；5. 租房补贴与购房补贴不能同时享受；6. 已享受租房补贴的，在申请购房补贴时扣除已发放的租房补贴 |
| 申请流程 | 1. 用人单位审核汇总申请材料后，报营业执照住所地县（市、区）受理部门，每季度第一个月的工作日集中受理；2. 受理部门对申请资格进行核验，通过人员由受理部门公示，公示期不少于5个工作日；3. 公示无异议的，由受理部门汇总后报市人社局，市人社局审核汇总后形成拨款申请报市财政局，市财政局将资金拨付至市辖七区后由受理部门发放给申请人；省财政直管县（市）由本级财政负担，参照办理 |
| 申请平台 | 各县（市、区）人社局受理部门（市人社局官网 https://rsj.baoding.gov.cn 下载申请表） |
| 申请材料 | 1.《租房补贴申请表》；2.申请人及配偶居民身份证及户口簿；3.申请人毕业证书；4.中国高等教育学生信息网验证打印的《教育部学历证书电子注册备案表》，取得国（境）外学历学位的须提供教育部留学服务中心《国（境）外学历学位认证书》；5.自主创业的申请人须提供在我市登记注册的营业执照，共同创业（合伙创业）的还须提供市场监管部门留存备案的公司章程（合伙协议）；6.首次申请须提交在我市1个月以上的城镇职工养老保险缴费证明，之后申请须提交享受补贴期间在我市连续缴纳城镇职工养老保险证明；7.本地社会保障卡（已激活金融功能）。以上材料均须提供原件及加盖单位公章的复印件 |
| 审核周期 | 每季度第一个月工作日集中受理，公示不少于5个工作日 |
| 发放方式 | 按季度发放，累计不超过36个月 |
| 生效时间 | 2022-01-01（保才办〔2022〕1号） |
| 失效时间 | null（持续执行，2026年1月仍有公示发放） |
| 政策状态 | 有效 |
| 官方来源 | 保定就业（保定市人社局）（https://rsj.baoding.gov.cn/bdjy/wap/index.php?c=article&a=show&id=67） |
| 最后核实日期 | 2026-07-01 |

> 说明：租房补贴与购房补贴（安家费）不能同时享受，二者互斥。

## 三、保定市高校毕业生购房补贴（安家费）

| 项目 | 内容 |
|------|------|
| 政策名称 | 保定市高校毕业生购房补贴（安家费） |
| 政策文号 | 《关于支持高校毕业生和技能人才在保就业创业的若干措施》（"保定人才十条"）；保办发〔2021〕6号《关于鼓励高校毕业生来保留保发展的若干扶持政策》；保才办〔2022〕1号操作指引 |
| 适用对象 | 在我市企业就业或自主创业的本科及以上学历学位高校毕业生 |
| 补贴金额/标准 | 博士15万元、硕士6万元、本科2万元；"双一流"高校全日制本科毕业生6万元。夫妻双方均符合条件的，可分别享受 |
| 学历要求 | 本科、硕士、博士（本科及以上学历学位；双一流本科按6万元） |
| 年龄限制 | 35周岁以下（含35周岁，指全日制本科及大专毕业生）；博士、硕士不限年龄 |
| 社保要求 | 在我市企业就业的须提供与用人单位签订的劳动合同和社保缴费证明 |
| 申请条件 | 1. 首次在我市企业就业或自主创业（2020年11月6日后引进，保才办〔2022〕1号操作指引明确2022年1月1日后毕业的高校毕业生申请范围）；2. 本科及以上学历学位高校毕业生；3. 全日制本科及大专毕业生35周岁以下；4. 在我市主城区购买住房（含新建商品住房及二手住房），且所购住房属申请人家庭在我市首套住房；5. 自主创业的须提供在我市登记注册的营业执照 |
| 申请流程 | 1. 市本级企业就业或自主创业的，购买预售商品住房的，申请人在办理房屋买卖合同网签备案、取得《不动产权证书》后分两次将申报材料报市人社局；购买现售商品住房和二手住房的，申请人在取得《不动产权证书》后将申报材料报市人社局。县（市、区）引进的人才报所在地县（市、区）受理部门；2. 市人社局会同市住建局集中审核（人社局负责就业创业信息、学历学位审核，住建局负责房屋网签备案信息核对），县（市、区）初审后报市人社局复审/备案；3. 复审（备案）通过人员名单在市人社局官网公示5个工作日；4. 市人社局向市财政局提出资金申请，市财政局拨付后由市人社局发放给申请人 |
| 申请平台 | 市人社局受理地址：东二环人才大厦1楼（0312-5063229）；县（市、区）引进人才报所在地县（市、区）受理部门 |
| 申请材料 | 1.《在保就业高校毕业生（技能人才）安家费申请表》或《在保创业高校毕业生（技能人才）安家费申请表》；2.申请人居民身份证、户口簿；3.申请人毕业证书、学位证书，以及中国高等教育学生信息网验证打印的《教育部学历证书电子注册备案表》、中国学位与研究生教育信息网在线打印的查询结果或认证报告，取得国（境）外学历学位的须提供教育部留学服务中心《国（境）外学历学位认证书》；4.技能人才须提供职业资格证书或职业技能等级证书及查询结果；5.在我市企业就业的申请人须提供与用人单位签订的劳动合同和社保缴费证明，自主创业的申请人须提供在我市登记注册的营业执照，共同创业（合伙创业）的还须提供市场监管部门留存备案的公司章程（或合伙协议）；6.房屋买卖网签备案合同；7.《不动产权证书》及完税证明；8.申请人持有的本地社会保障卡（已激活金融功能）。以上材料均须提供原件及加盖单位公章的复印件 |
| 审核周期 | 法定工作日即来即办，公示5个工作日 |
| 发放方式 | 购买预售商品住房的，于办理房屋网签备案和取得《不动产权证书》后分两次发放，每次发放标准安家费的50%；购买现售商品住房和二手住房的，自取得《不动产权证书》后一次性发放 |
| 生效时间 | 2020-11-06（"保定人才十条"） |
| 失效时间 | null（持续执行） |
| 政策状态 | 有效 |
| 官方来源 | 保定本地宝（https://m.bd.bendibao.com/mip/14050.shtm）；保定就业（保定市人社局） |
| 最后核实日期 | 2026-07-01 |

> 说明：购房补贴（安家费）与租房补贴不能同时享受，二者互斥。已享受租房补贴的，在申请购房补贴时扣除已发放的租房补贴。
