#### 1. 惠州市新引进人才安家补贴

| 字段 | 内容 |
|------|------|
| **政策名称** | 惠州市新引进人才安家补贴 |
| **政策文号** | 惠市人社规〔2022〕5号《惠州市人力资源和社会保障局 中共惠州市委组织部关于新引进人才安家落户补贴发放实施细则》 |
| **适用对象** | 非行政机关、非参公管理单位、非个体工商户新引进的45周岁以下的博士研究生、博士后，50周岁以下具有正高级职称的专业技术人才 |
| **补贴金额/标准** | 博士研究生20万元安家补贴，分3年按6万、7万、7万发放；博士后、正高级职称专业技术人才30万元安家补贴，分3年等额发放。补贴为税前金额，按偶然所得适用20%税率计征个人所得税，发放金额为税后金额 |
| **学历要求** | 博士研究生、博士后（或正高级职称） |
| **年龄限制** | 博士研究生、博士后45周岁以下；正高级职称专业技术人才50周岁以下（按首次在惠签订劳动或聘用合同时间计算） |
| **社保要求** | 在本市连续参加社会保险和缴纳个人所得税均满12个月且当前正常参保状态。本市是社会保险和工资薪金所得个人所得税缴纳唯一地，参保单位、扣缴义务人与用人单位一致 |
| **申请条件** | 1.2022年1月1日至2025年12月31日期间引进，且首次在惠参加工作并参保；2.非行政机关、非参公管理单位、非个体工商户新引进；3.已与本市用人单位签订3年及以上劳动或聘用合同，全职在本市工作；4.在本市连续参加社保和缴纳个税均满12个月且当前正常参保；5.引进前已持有相关证书或佐证材料；6.具有本市户籍（国境外人才不受此项限制）；7.须在新引进到本市工作3年以内提出申请 |
| **申请流程** | 通过"粤财扶助"平台网上申报→各县（区）人社部门受理、初审（市直单位直接由市人社局受理、审批）→市人社局复核审批→公示5个工作日→核准发放名单→补贴优先发至人才在本市的社保卡金融账户。续领第二年及第三年补贴须提供连续12个月社保参保证明和个税纳税记录 |
| **申请平台** | 粤财扶助平台（https://czbt.czt.gd.gov.cn/#/home） |
| **申请材料** | 1.安家补贴申请表；2.申请人身份证或有效证件；3.学历学位证书、博士后证书（国境外人才另提供学历学位认证书）；4.劳动或聘用合同（或非参公事业单位增人计划卡）；5.社会保障卡；6.本市户籍证明（户口簿，国境外人才除外）；7.连续12个月社会保险个人参保证明；8.连续12个月个人所得税纳税记录；9.职称证书（正高级职称人才提供）；10.用人单位营业执照或法人证书。续领另需提供更新至申请前的社保参保证明和个税纳税记录 |
| **审核周期** | null（经受理、初审、复核审批、公示5个工作日等环节） |
| **发放方式** | 博士分3年按6万/7万/7万发放；博士后及正高级分3年等额发放。优先发至社保卡金融账户 |
| **生效时间** | 2022年（惠市人社规〔2022〕5号印发） |
| **失效时间** | 2026年10月31日（本次申报截止日期，引进时间为2022年1月1日至2025年12月31日） |
| **政策状态** | 有效。博士20万、博士后及正高级30万安家补贴，分3年发放。须3年以内提出申请，仍在领取领军人才补贴及在站博士后不予申报。续领须提供连续12个月社保和个税证明 |
| **官方来源** | http://m.toutiao.com/group/7634493195460428297/ |
| **最后核实日期** | 2026-07-01 |

#### 2. 惠州市新引进人才落户补贴

| 字段 | 内容 |
|------|------|
| **政策名称** | 惠州市新引进人才落户补贴 |
| **政策文号** | 惠市人社规〔2022〕5号《惠州市人力资源和社会保障局 中共惠州市委组织部关于新引进人才安家落户补贴发放实施细则》 |
| **适用对象** | 非行政机关、非参公管理单位、非个体工商户新引进的高校应届硕士毕业生、45周岁以下高级职称专业技术人才、45周岁以下高级技师；民营企业（含外资企业）新引进的高校应届本科毕业生、35周岁以下中级职称专业技术人才、35周岁以下技师 |
| **补贴金额/标准** | 应届硕士毕业生、高级职称专业技术人才、高级技师享受一次性1万元落户补贴；应届本科毕业生、中级职称专业技术人才、技师享受一次性最高5000元落户补贴（"双一流"院校应届本科或中级职称/技师按每人5000元发放，其他院校应届本科按每人3000元发放）。补贴为税前金额，按偶然所得20%税率计征个人所得税，发放金额为税后金额 |
| **学历要求** | 本科、硕士（或中级职称、高级职称、技师、高级技师） |
| **年龄限制** | 高级职称及高级技师45周岁以下；中级职称及技师35周岁以下（按首次在惠签订劳动或聘用合同时间计算） |
| **社保要求** | 在本市连续参加社会保险和缴纳个人所得税均满12个月且当前正常参保状态。本市是社会保险和工资薪金所得个人所得税缴纳唯一地，参保单位、扣缴义务人与用人单位一致 |
| **申请条件** | 1.2022年1月1日至2025年12月31日期间引进，且首次在惠参加工作并参保；2.非行政机关、非参公管理单位、非个体工商户新引进；3.民营企业（含外资企业）新引进的应届本科毕业生可申请；4.已与本市用人单位签订3年及以上劳动或聘用合同，全职在本市工作；5.在本市连续参加社保和缴纳个税均满12个月且当前正常参保；6.引进前已持有相关证书或佐证材料；7.具有本市户籍（国境外人才不受此项限制）；8.须在新引进到本市工作3年以内提出申请；9.一次性申报项目，不得重复申领 |
| **申请流程** | 通过"粤财扶助"平台网上申报→各县（区）人社部门受理、初审（市直单位直接由市人社局受理、审批）→市人社局复核审批→公示5个工作日→核准发放名单→补贴优先发至人才在本市的社保卡金融账户 |
| **申请平台** | 粤财扶助平台（https://czbt.czt.gd.gov.cn/#/home） |
| **申请材料** | 1.落户补贴申请表；2.申请人身份证或有效证件；3.学历学位证书（国境外人才另提供学历学位认证书）；4.劳动或聘用合同（或非参公事业单位增人计划卡）；5.社会保障卡；6.本市户籍证明（户口簿，国境外人才除外）；7.自引进到本市工作之日起至申请之日止连续12个月社保个人参保证明；8.连续12个月个人所得税纳税记录；9.职称证书或职业资格证书（按相应条件提供）；10.用人单位营业执照或法人证书。"双一流"院校应届本科需提供毕业当年双一流院校名单佐证 |
| **审核周期** | null（经受理、初审、复核审批、公示5个工作日等环节） |
| **发放方式** | 一次性发放，优先发至社保卡金融账户 |
| **生效时间** | 2022年（惠市人社规〔2022〕5号印发） |
| **失效时间** | 2026年10月31日（本次申报截止日期，引进时间为2022年1月1日至2025年12月31日） |
| **政策状态** | 有效。应届硕士1万元、应届本科5000元（双一流）或3000元（其他）一次性落户补贴。须3年以内提出申请，一次性申报不得重复申领。仍在领取领军人才补贴及在站博士后不予申报 |
