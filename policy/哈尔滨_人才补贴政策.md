#### 1. 哈尔滨市引进高校毕业生申领安家费生活补助

| 字段 | 内容 |
|------|------|
| **政策名称** | 哈尔滨市引进高校毕业生申领安家费生活补助 |
| **政策文号** | 哈人社规〔2023〕9号（修订），依据《人才新政30条》（哈发〔2023〕3号）第九条 |
| **适用对象** | 在哈企业及市县乡三级事业单位全职新引进的统招高校毕业生；技工院校预备技师班、高级工班毕业生；经住院医师规范化培训合格的本科学历临床医师；通过教育部认证的港澳台地区或海外留学人员 |
| **补贴金额/标准** | 博士：生活补助3000元/月（连发3年）+安家费10万元；硕士（毕业5年内·企业）：生活补助2000元/月（连发3年）+安家费3万元；本科及技工院校预备技师/高级工（毕业5年内·企业）：生活补助1500元/月（连发3年）。生活补助按月发放，安家费按月平均计算额度与生活补助同步发放，连发36个月 |
| **学历要求** | 统招博士、统招硕士、统招本科；技工院校预备技师班、高级工班毕业生；住院医师规培合格的本科学历临床医师享受医学类硕士待遇 |
| **年龄限制** | null（博士无年龄限制；硕士、本科要求毕业5年内） |
| **社保要求** | 须与用人单位签订不少于3年劳动（聘用）合同，并在聘用单位缴纳社会保险 |
| **申请条件** | 1.全职新引进（哈发〔2023〕3号施行后首次签订劳动合同、首次缴纳社保，不含就业见习人员）；2.博士适用于在哈企业及市县乡三级事业单位；硕士、本科仅适用于在哈企业且毕业5年内；3.签订不少于3年劳动（聘用）合同并缴纳社保；4.与其他人才补贴政策重复交叉的，按就高、不重复原则享受；5.政策实施后取得统招更高层级学历的可按更高层级申请 |
| **申请流程** | 1.用人单位对本单位申请人申请资格进行初审；2.初审合格后登录哈尔滨人社综合服务平台注册并录入申报信息，或到区、县（市）人社经办机构窗口提交电子版材料；3.经办机构受理后通过相关数据平台对注册信息、学历、劳动关系、社会保险等信息逐一核实比对；4.各区、县（市）经办机构确认通过后于当月底形成资金申请汇总表，报同级财政申请拨付资金；5.经办机构将资金直接发放给申请人，次月初报市人力资源服务中心备案 |
| **申请平台** | 哈尔滨人社综合服务平台 http://www.hrbrsj.org.cn ；或各区、县（市）人社经办机构窗口 |
| **申请材料** | 1.用人单位书面申请；2.《哈尔滨市引进高校毕业生申领安家费生活补助申请表》；3.企业营业执照或事业单位法人证书（通过数据平台比对核实）；4.申请人毕业证；5.劳动合同备案册或劳动合同（通过数据平台比对核实）；6.社会保险缴费信息（通过数据平台比对核实）；7.劳务派遣类申请人另需提供：派遣单位与在哈被派遣单位合作协议、申请人在被派遣单位员工名册（双方加盖公章）、派遣单位支付给申请人的银行工资流水等 |
| **审核周期** | 按月审核，当月底汇总报财政拨付（约1个月） |
| **发放方式** | 生活补助按月发放；安家费按月平均计算额度与生活补助同步发放，连发36个月 |
| **生效时间** | 2023-02-20（与《人才新政30条》同步） |
| **失效时间** | 待核实（政策有效期与《人才新政30条》同步，截至核实日仍有效） |

#### 2. 哈尔滨市人才购房补贴

| 字段 | 内容 |
|------|------|
| **政策名称** | 哈尔滨市人才购房补贴（《人才新政30条》第九条） |
| **政策文号** | 哈发〔2023〕3号第九条（2026年办理流程由哈市住建局发布；此前哈住建发〔2024〕339号已于2025-12-31到期） |
| **适用对象** | 与在哈用人单位签订聘用合同或服务协议、缴纳6个月社保的各类人才，以及自主创业、年龄在40周岁以下的各类人才（无户籍限制）；不含公务员、参照公务员法管理的群团机关和事业单位人员 |
| **补贴金额/标准** | 博士10万元；硕士5万元；本科及技工院校预备技师班、高级工班毕业生3万元；经住院医师规范化培训合格的本科学历临床医师享受医学类硕士同等待遇；港澳台地区或海外留学人员（通过教育部认证）享受与国内高校毕业生同等待遇。一次性发放 |
| **学历要求** | 本科以上学历毕业生，或技工院校预备技师班、高级工班毕业生；住院医师规培合格的本科学历临床医师享受医学类硕士待遇；港澳台/海外留学人员（教育部认证）享受同等待遇 |
| **年龄限制** | 40周岁以下（未满41周岁） |
| **社保要求** | 购房前（以商品房买卖合同网签备案时间为节点）已连续缴纳6个月以上社保，社保缴纳在哈市行政区域范围内，并纳入社保系统管理且处于存续状态；自主创业的须在哈市市场监管局注册登记、营业执照有效且法定代表人为购房人本人 |
| **申请条件** | 1.自2026年1月1日起在哈市9区购买首套或第二套新建商品住房（以网签备案时间为节点）；2.与在哈用人单位签订聘用合同或服务协议，购房前已连续缴纳6个月以上社保（或自主创业且营业执照有效）；3.年龄40周岁以下（未满41周岁）；4.具有本科以上学历或技工院校预备技师班、高级工班毕业生（无户籍限制）；5.以家庭为单位，夫妻双方不能同时享受，只能一方（产权人）申请；6.在取得《不动产权证书》后即可申请，申请时间不受文件有效期影响；7.不含公务员及参公管理人员 |
| **申请流程** | 1.线上申领表格：在"哈尔滨市住建"微信公众号下载《购房补贴申请表》；2.线下提出申请：携带申报材料到哈尔滨市住房保障中心（民安街184号）现场申请；3.审批合格后按资金流程及申报顺序下发补贴资金，由发放部门代扣代缴个人所得税 |
| **申请平台** | "哈尔滨市住建"微信公众号（下载申请表）；哈尔滨市住房保障中心（民安街184号）现场办理，咨询电话0451-88370807 |
| **申请材料** | 1.购房补贴申请表；2.申请人本人或共有人双方身份证原件；3.结婚证或离婚证（或离婚判决）原件和户口原件，丧偶应提供死亡证明；4.本人名下哈尔滨银行一类借记卡原件；5.毕业证等学历证明材料原件；6.网签备案《商品房买卖合同》原件，或不动产登记部门出具加盖公章的网签备案《商品房买卖合同》复印件；7.新房《不动产权证书》原件；8.电子印章为红色彩打的有单位及个人信息社保凭证，或注册有效期内的营业执照；9.产权为共有关系的，双方需同时到场，只能一方申请，另一方签订放弃保证书；10.经住院医师规范化培训合格的本科学历临床医师的培训合格证或相关证明；11.港澳台地区或海外留学人员提供教育部认证的学历证明；12.与在哈用人单位签订的聘用合同或服务协议 |

| **发放方式** | 一次性发放（由发放部门代扣代缴个人所得税） |

### 区级政策互斥关系说明

> 最后核实日期：2026-07-01

经核实，哈尔滨市人才补贴政策以市级统一政策为主，由哈尔滨市人社局、市住建局统一制定，全市9区（道里、道外、南岗、香坊、平房、松北、呼兰、阿城、双城）9县（市）均适用同一市级政策，仅财政负担主体不同（主城6区由市本级财政负担，其余3区及9县市由本级财政负担）。本次调研未检索到各区独立设置、与市级政策并行且金额/条件不同的区级人才补贴政策，故暂无区级互斥关系。

说明：
1. 上述3条政策均为市级政策，`exclusiveGroup` 留空。
2. 根据《哈尔滨市引进高校毕业生申领安家费生活补助实施细则》，该政策"与我市现有其他人才补贴政策存在重复、交叉的，按照就高、不重复原则享受"，即同一申请人就同类补贴不可重复享受。
3. 若后续哈尔滨各区出台独立的人才补贴政策（与市级并行），则同一人才在同一时段仅可享受其中一项区级补贴，届时区级政策将归入 `exclusiveGroup: "harbin-district"` 互斥组。
4. 安家费生活补助（生活补助+安家费）与人才购房补贴分属不同类别（生活类/购房类），购房补贴针对在哈购置新建商品住房，两者用途不同；是否可叠加享受以官方审核为准，建议申请前通过市人社局（0451-84656008）或市住房保障中心（0451-88370807）核实。
