#### 1. 深圳市新引进博士人才生活补贴

| 字段 | 内容 |
|------|------|
| **政策名称** | 深圳市新引进博士人才生活补贴 |
| **政策文号** | 深人社规〔2022〕3号 |
| **适用对象** | 2021年9月1日起经市、区人力资源部门首次引进的具有博士学历的应届毕业生、在职人才和留学回国人员（国内新引进），以及首次在深圳就业的具有博士学历（学位）的港澳台和外籍人才（海外新引进） |
| **补贴金额/标准** | 100,000元/人，分两次发放：首次30%（30,000元），续发70%（70,000元） |
| **学历要求** | 博士研究生学历（学位） |
| **年龄限制** | 35岁（含）以下（以引进时间为判断时点） |
| **社保要求** | 首发申请时：引进后通过本市工作单位缴纳社会保险累计3个月以上（不含补缴）；续发申请时：自首发申请之日起连续缴纳社会保险满12个月。社保含养老、医疗、失业、生育、工伤5个险种 |
| **申请条件** | 国内新引进博士人才需取得深圳户籍；海外新引进博士人才无户籍要求，但需为港澳永久居民/赴港澳定居内地居民/台湾居民/外国国籍人士。机关事业单位在编人员不可申请。已领取高层次人才奖励补贴、博士后生活补贴、原租房和生活补贴的不可重复申领 |
| **申请条件（详细-国内）** | ①取得深圳户籍；②具有博士研究生学历；③首发申请时在深圳全职工作，引进后通过本市工作单位缴纳社会保险累计3个月以上（不含补缴）；④续发申请时仍具有深圳户籍、在深圳全职工作且自首发申请之日起通过本市工作单位连续缴纳社会保险满12个月。学历、引进身份等条件以办理引进手续时核准的信息为准 |
| **申请条件（详细-海外）** | ①具有以下身份之一：香港/澳门永久性居民（以永久性港澳居民身份证和通行证为准）；赴港澳定居的内地居民（已注销内地户籍，以港澳居民身份证和通行证为准）；台湾地区居民（以通行证为准）；外国国籍人士（以护照或外国人永久居留证为准）；②具有博士研究生学历（学位），国（境）外学历以教育部留学服务中心出具的认证书为准；③首发申请时在深圳全职工作，引进后通过本市工作单位缴纳社会保险累计3个月以上；④续发申请时仍在深圳全职工作且自首发申请之日起连续缴纳社会保险满12个月。申请人身份信息应与申报缴纳个税、社保所使用的身份信息保持一致 |
| **申请材料** | 1.深圳市新引进博士人才生活补贴申报承诺书（签名+单位公章）；2.全国版个人所得税纳税记录/清单；3.海外新引进博士人才还需提供身份信息材料（港澳居民身份证+通行证/台湾居民通行证/护照或永居证）；4.海外新引进博士人才还需提供学历信息材料（国内学历：《教育部学历证书电子注册备案表》；境外学历：教育部留学服务中心出具的《学历学位认证书》） |
| **申请流程** | 线上办理：登录"广东政务服务网"搜索"深圳市新引进博士人才生活补贴"→在线办理→填报信息并上传材料→提交所在单位审核→单位审核后提交市人社部门→受理→审核（15个工作日）→次月公示5个工作日→公示结束后20个工作日内拨付至单位账户→单位30日内发放至个人 |
| **申请渠道** | 广东政务服务网：https://www.gdzwfw.gov.cn/portal/v3/guide/114403006955832485344211108Y00001 |

| **失效时间** | 2027年4月14日（有效期5年） |
