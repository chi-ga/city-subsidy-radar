# 太原市人才补贴政策

> 最后核实日期：2026-07-01
>
> 政策依据：《中共太原市委关于深化人才发展体制机制改革加快推进创新驱动转型升级的实施意见》（并发〔2018〕2号），配套《高校毕业生和高技能人才来并工作补助（贴）发放实施办法》《本土人才能力素质提升补助发放办法》，由太原市人力资源和社会保障局于2022年9月9日发布申报通知并持续执行。截至2026年，各区/开发区（小店区、尖草坪区、综改示范区等）仍按季度公示审核通过名单，政策处于有效执行状态。

---

## 一、高校毕业生和高技能人才来并工作生活补助

| 项目 | 内容 |
|------|------|
| 政策名称 | 高校毕业生和高技能人才来并工作生活补助 |
| 政策文号 | 并发〔2018〕2号；《高校毕业生和高技能人才来并工作补助（贴）发放实施办法》 |
| 适用对象 | 全日制博士研究生；世界排名前200名大学（不含境内）、教育部公布的"双一流"建设高校及建设学科的全日制硕士研究生、本科生 |
| 补贴金额/标准 | 博士研究生5000元/月、硕士研究生3000元/月、本科生1500元/月；补贴时间为5年。已获省级相应财政补助的，按照"就高不重复"原则予以补齐 |
| 学历要求 | 全日制博士研究生、硕士研究生、本科生（硕士、本科需为世界排名前200名大学或"双一流"建设高校及建设学科） |
| 年龄限制 | null（高校毕业生需在国家规定的择业期内参加工作） |
| 社保要求 | 在用人单位按规定缴纳城镇职工基本养老保险 |
| 申请条件 | 1. 2018年1月1日后首次在并参加工作；2. 与用人单位签订不少于3年的劳动（聘用）合同（服务基层项目人员可放宽到不少于1年），或已累计签订并履行3年及以上劳动合同；3. 具有太原市户籍；4. 用人单位注册地在太原市域内且诚实守信、合法经营；5. 用人单位为各类企业、市县所属机关事业单位、中央/省属驻并高校/科研院所/公立医院（不含参公单位）、民办非企业单位、民办高校、律师事务所等 |
| 申请流程 | 1. 申请人登录太原市人社局官网进入"太原市人才补贴申报系统"录入信息并打印申请表；2. 用人单位核实材料真实性并公示5个工作日；3. 用人单位向所在地政务大厅人才服务窗口申报；4. 人才服务窗口审核；5. 审定名单公示5个工作日；6. 市财政局拨付资金，委托银行代发到人才联名卡 |
| 申请平台 | 太原市人力资源和社会保障局官网"太原市人才补贴申报系统"（http://rsj.taiyuan.gov.cn） |
| 申请材料 | 1. 申请表（系统录入生成打印，原件）；2. 用人单位花名表（原件）；3. 身份证、户口簿本人单页、人才联名卡（原件及复印件1份）；4. 毕业证、学位证、《教育部学历证书电子注册备案表》（留学回国人员提供《国（境）外学历学位认证书》）（原件及复印件1份）；5. 劳动合同或事业单位聘用文件及合同（劳务派遣人员另需劳务派遣协议及员工花名表）（原件及复印件1份）；6. 所在单位营业执照/法人登记证书/统一社会信用代码证书（律师事务所提供执业许可证）（原件及复印件1份）；7. 企业单位基本养老保险证明材料（原件，机关事业单位在编在岗人员不需提供）；8. 中央/省属驻并高校/医院/科研院所一线岗位证明材料花名表（原件）。复印件需加盖单位公章 |
| 审核周期 | 全年申报，每人每年申报一次，补助按季度发放 |
| 发放方式 | 委托银行代发，直接发放到申请人人才联名卡 |
| 生效时间 | 2018年（申报系统2022年9月9日开放） |
| 失效时间 | null（持续执行，无明确失效时间） |
| 政策状态 | 有效 |
| 官方来源 | 太原市人力资源和社会保障局官网（http://rsj.taiyuan.gov.cn）；太原本地宝（ty.bendibao.com） |
| 最后核实日期 | 2026-07-01 |

## 五、本土人才能力素质提升补助

| 项目 | 内容 |
|------|------|
| 政策名称 | 本土人才能力素质提升补助 |
| 政策文号 | 并发〔2018〕2号；《本土人才能力素质提升补助发放办法》 |
| 适用对象 | 取得上一级学历学位（硕士研究生、博士研究生）的在职（聘）人员；取得上一级职业资格证书（技师、高级技师）的在职（聘）人员 |
| 补贴金额/标准 | 取得硕士研究生学历学位每年给予2万元补助；取得博士研究生学历学位每年给予3万元补助；取得二级（技师）职业资格证书每年给予1.5万元补助；取得一级（高级技师）职业资格证书每年给予2万元补助。补助发放时间均为2年 |
| 学历要求 | 在职期间取得更高学历学位（硕士研究生、博士研究生）或职业资格证书（技师、高级技师） |
| 年龄限制 | null |

## 区级政策互斥关系说明

太原市人才补贴政策为**市级统一政策**，由太原市委人才工作领导小组统一制定，各区（小店区、迎泽区、杏花岭区、尖草坪区、万柏林区、晋源区）及开发区（山西转型综改示范区、中北高新区等）设有人才服务窗口负责受理审核和按季度公示，**各区/开发区不设独立的人才补贴政策**，执行的均为上述市级统一标准。

**市级政策内部互斥关系：**

1. **生活补助与租房补贴互斥**：生活补助与租房补贴只可申请一项，不得同时申请。其中，生活补助面向全日制博士及"双一流"/世界名校的硕士、本科；租房补贴面向其他普通高校的硕士、本科及高技能人才。两类补贴的适用对象本身互不重叠，且不可同时申领。

2. **就高不重复原则**：已获省级相应财政补助的，按照"就高不重复"的原则予以补齐，不重复享受。

3. **按最高学历申领**：高校毕业生所有补贴均按最高学历申领。

4. **学费补贴限制**：由财政部门承担其学费、教材费、住宿费并给予生活费补助的高校毕业生不能申请学费补贴。

> 说明：尖草坪区出台的"黄金十条"为招商引资政策（面向落户企业及投资项目），非面向普通高校毕业生的人才补贴，故未纳入本政策清单。
