#### 1. 常州市青年人才生活居住双资助—生活资助

| 字段 | 内容 |
|------|------|
| **政策名称** | 常州市青年人才生活居住双资助—生活资助 |
| **政策文号** | 常政办发〔2021〕21号《"青年人才生活居住双资助"实施细则》 |
| **适用对象** | 毕业两年内，由常州市企业引进的博士研究生、35周岁以下硕士研究生、全日制本科生以及高级技师和技师 |
| **补贴金额/标准** | 博士研究生800元/月；硕士研究生和高级技师500元/月；全日制本科生和技师300元/月。资助期限3年（36个月） |
| **学历要求** | 全日制本科、硕士、博士（技师、高级技师参照本科、硕士标准） |
| **年龄限制** | 硕士研究生35周岁以下（含35周岁）；博士、本科无明确年龄限制 |
| **社保要求** | 在常州首次参保并连续缴纳社保6个月以上（由常州就业企业缴纳） |
| **申请条件** | 1.毕业两年内由常州市企业引进；2.在常州首次参保并连续缴纳社保6个月以上；3.博士研究生不限年龄，硕士研究生须35周岁以下，本科生须全日制；4.2021年1月1日后企业新引进（含境外取得学士学位以上及离开常州3年以上再回常州工作的人才） |
| **申请流程** | 1.该项政策采取免申报方式，系统自动比对社保、学历信息生成发放名单→2.从人才参保第7个月起按月发放→3.符合条件的人才也可登录"我的常州"APP提出申请→4.系统无法比对数据的，由申请人根据提示补充上传相应资料 |
| **申请平台** | "我的常州"APP（免申报，系统自动比对；也可主动申请） |
| **申请材料** | 1.身份证（系统无法比对时补充上传）；2.学历学位证书（系统无法比对时补充上传） |
| **审核周期** | 系统自动比对，从参保第7个月起按月发放 |
| **发放方式** | 按月发放，资助期限3年 |
| **生效时间** | 2021-03-19（2023年1月1日后新引进人才适用3年资助期限） |
| **失效时间** | 待核实 |
| **政策状态** | 有效。2023年1月1日后企业新引进人才资助期限由2年延长至3年 |

### 区级政策互斥关系说明

常州市各区级人才补贴政策属于**互斥关系**——同一人才在同一城市只能选择一个区享受区级人才补贴政策（因就业参保地、购房地等条件绑定特定区域，无法跨区重复享受）。

目前纳入系统的区级政策：

| 区名 | 政策名称 | exclusiveGroup |
|------|---------|----------------|
| 常州经开区 | 常州经开区人才购房补贴 | changzhou-district |

**互斥规则：**
1. 同一城市的所有区级政策统一设置 `exclusiveGroup: "changzhou-district"`
2. 市级政策（青年人才生活居住双资助）不设置互斥组，可与区级政策叠加享受
3. 常州经开区购房补贴明确要求"在享受市级购房补贴的基础上"给予额外补贴，属于市级政策的补充叠加，但与其他区级补贴互斥
4. 随着更多区级政策纳入，新政策将加入同一互斥组

> **注意：** 常州市天宁区"十大人才政策"（最高30万元生活补助）需经评定择优，属于人才认定型政策，未纳入本系统。新北区、武进区、金坛区、钟楼区暂未检索到面向普通高校毕业生、无需特殊人才认定的独立区级补贴政策，如后续出台将补充录入。
