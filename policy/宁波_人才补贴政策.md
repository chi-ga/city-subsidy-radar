#### 1. 毕业一年内来甬人才安居补贴

| 字段 | 内容 |
|------|------|
| **政策名称** | 毕业一年内来甬人才安居补贴 |
| **政策文号** | 待核实 |
| **适用对象** | 新引进毕业一年内来甬工作的硕士毕业生、全日制本科毕业生、全日制技工学校技师班（预备技师班）毕业生 |
| **补贴金额/标准** | 硕士：1,000元/月，最长60个月，总额最高60,000元；全日制本科/技师班：1,000元/月，最长30个月，总额最高30,000元 |
| **学历要求** | 硕士研究生、全日制本科、全日制技工学校技师班（预备技师班） |
| **年龄限制** | null |
| **社保要求** | 在甬实际工作并依法缴纳社保，以首次在甬缴纳社保时间为新引进时间；社保变更或中断须重新申报 |
| **申请条件** | ①毕业一年内来甬工作（国内高校以毕业证书日期后一年内，海外高校以学历认证书日期后一年内）；②在甬缴纳社保；③首次在甬缴纳社保；④未享受青年人才租房补贴、硕士和博士生活补贴、高层次人才安家补助；⑤公务员（含参公人员）不享受 |
| **申请流程** | ①在甬缴纳社保次月到账后登录宁波市企业综合服务平台；②填写信息并上传材料；③工作单位审核同意（灵活就业人员由社保缴纳地人社部门直接审核）；④市及各区（县、市）人社部门审核；⑤按月发放至申请人社保卡金融账户 |
| **申请平台** | 宁波市企业综合服务平台：https://qf.ningbo.gov.cn/qykj/projectDetail/52581 |
| **申请材料** | 1. 身份证；2. 学历学位证书；3. 《教育部学历证书电子注册备案表》（国内高校）或《国（境）外学历学位认证书》（海外高校）；4. 劳动合同或聘用合同（灵活就业及社保缴纳单位非人力资源企业者可免）；5. 社保卡（需开通金融功能） |
| **审核周期** | 待核实 |
| **发放方式** | 按月发放至申请人本人社保卡金融账户 |
| **生效时间** | 2025-12-01 |
| **失效时间** | 待核实 |
| **政策状态** | 有效（2025年12月起实施） |
| **官方来源** | https://qf.ningbo.gov.cn/qykj/publicDetail/1668 |
| **最后核实日期** | 2026-06-30 |

#### 2. 硕士和博士生活补助

| 字段 | 内容 |
|------|------|
| **政策名称** | 硕士和博士生活补助 |
| **政策文号** | 甬人社发〔2022〕23号 |
| **适用对象** | 新引进在宁波工作或灵活就业的硕士、博士毕业生 |
| **补贴金额/标准** | 硕士：30,000元/人；世界排名前100高校硕士：80,000元/人；世界排名前200高校博士：100,000元/人 |
| **学历要求** | 硕士、博士 |
| **年龄限制** | 硕士35周岁以下；博士40周岁以下 |
| **社保要求** | 在甬累计缴纳社保满6个月 |
| **申请条件** | ①新引进35周岁以下硕士或40周岁以下世界排名前200高校博士；②在甬累计缴纳社保满6个月；③本人及配偶在宁波市行政区域内无房；④工作单位须在宁波市内税务登记注册 |
| **申请流程** | ①登录宁波市人才服务申报系统PC端或宁波"人才码"移动端；②填报信息并上传材料；③市服务总窗或区（县、市）服务联盟窗口初审；④市及各区（县、市）人社部门复核；⑤发放至申请人社保卡金融账户 |
| **申请平台** | 宁波市人才服务申报系统：https://hrs.nbrc.com.cn/；宁波"人才码"移动端 |

| **审核周期** | 待核实 |
| **发放方式** | 一次性发放至申请人本人社保卡金融账户 |
| **生效时间** | 2022-07-25 |
| **失效时间** | 待核实 |
| **政策状态** | 有效（常态化受理） |
| **官方来源** | https://hrs.nbrc.com.cn/download/sshbsshbzzcsm.pdf |

#### 3. 青年人才租房补贴

| 字段 | 内容 |
|------|------|
| **政策名称** | 青年人才租房补贴 |
| **政策文号** | 甬人社发〔2021〕17号 |
| **适用对象** | 新引进35周岁以下的全日制应届本科、硕士毕业生（含非全日制应届毕业硕士研究生）和具有技师及以上职业资格（技能等级）的青年人才 |
| **补贴金额/标准** | 10,000元/年，最长3年，总额最高30,000元 |
| **学历要求** | 全日制应届本科、硕士（含非全日制应届硕士）；技师及以上职业资格（技能等级） |
| **年龄限制** | 35周岁以下 |
| **社保要求** | 应届毕业生累计缴纳社保满6个月、20个月、30个月后可分别申领；技师及以上职业资格人才累计缴纳社保满10个月、20个月、30个月后可分别申领 |
| **申请条件** | ①2021年4月20日起新引进；②35周岁以下；③全日制应届本科/硕士或具有技师及以上职业资格（技能等级）；④在甬依法缴纳社保达到一定期限；⑤本人及配偶在宁波市行政区域内无房；⑥政策享受周期为自新引进之月起满3年（36个月） |
| **申请流程** | ①登录宁波市人才服务申报系统PC端或宁波"人才码"移动端；②填报信息并上传材料；③市及区（县、市）高层次人才创业创新服务联盟窗口匡算；④复核通过后发放至申请人社保卡金融账户 |
| **申请平台** | 宁波市人才服务申报系统：https://hrs.nbrc.com.cn/；宁波"人才码"移动端 |
| **申请材料** | 1. 租房合同；2. 《国（境）外学历学位认证书》（海外学历人才提供）；3. 劳动合同或聘用合同（社保缴纳单位为人力资源企业者须提供）；4. 职业资格证书或技能等级证书（技能人才提供）；5. 部省属企业人员需提供《浙江省基本养老保险历年参保证明》及《浙江省社会保险参保证明（个人专用）》（查询范围48个月）；6. 社保卡（需开通金融功能） |
| **审核周期** | 待核实 |
| **发放方式** | 按年发放至申请人本人社保卡金融账户 |
| **生效时间** | 2021-04-20 |
| **失效时间** | 待核实 |
| **政策状态** | 有效（每月21日至月底申报） |

| **年龄限制** | null |

| **申请平台** | 浙江政务服务网：https://www.zjzwfw.gov.cn/zjservice-fe/#/workguide?localInnerCode=7c3a8d34-573a-42d8-a416-2f1285676cb0；"浙里办"APP |
| **申请材料** | 1. 《高校毕业生就业岗位补贴申请表》；2. 居民身份证；3. 学历证明（首次申请时提供）；4. 劳动合同；5. 中小微企业就业相关证明（必要时提供） |
| **审核周期** | 23个工作日（不含公示期） |

| **生效时间** | 2023年（以《宁波市贯彻落实稳就业政策实施办法》印发时间为准） |

| **政策状态** | 有效 |
