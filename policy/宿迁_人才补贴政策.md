# 宿迁人才补贴政策

>
> 政策依据：《关于强化高质量发展人才引领的实施意见》（宿办发〔2018〕78号），由宿迁市委人才工作领导小组办公室、宿迁市人力资源和社会保障局发布并执行。

## 一、宿迁市新引进高校毕业生租房补贴

| 项目 | 内容 |
|------|------|
| 政策名称 | 宿迁市新引进高校毕业生租房补贴 |
| 政策文号 | 《关于印发<宿迁市新引进高校毕业生租房补贴申领暂行办法>的通知》（宿组通〔2020〕38号），依据《关于强化高质量发展人才引领的实施意见》（宿办发〔2018〕78号） |
| 适用对象 | 上一年度新引进到宿迁市工业及电子商务企业工作或开展自主创业的博士研究生、硕士研究生或国家"双一流"建设高校本科学历高校毕业生（企业注册地在乡镇工业园区的学历放宽至普通高校本科） |
| 补贴金额/标准 | 博士研究生2500元/月；硕士研究生2000元/月；国家"双一流"建设高校本科毕业生1500元/月；其他普通高校本科毕业生1000元/月。补贴期限为2年，资金按年度发放，主要用于新引进高校毕业生租房、生活等支出 |
| 学历要求 | 博士研究生、硕士研究生、国家"双一流"建设高校本科（企业注册地在乡镇工业园区的放宽至普通高校本科） |
| 年龄限制 | 不超过35周岁 |
| 社保要求 | 企业应在我市正常纳税、足额缴纳社会保险；企业引进人才须缴纳社保 |
| 申请条件 | 1. 上一年度新引进到我市工业及电子商务企业工作或开展自主创业；2. 具有博士研究生、硕士研究生或国家"双一流"建设高校本科学历（企业注册地在乡镇工业园区的学历放宽至普通高校本科）；3. 年龄不超过35周岁；4. 企业应在我市正常纳税、足额缴纳社会保险；5. 自主创业的引进人才，在所创办企业的实收资本中货币投资不少于30万元，且已完成注册登记、社保等相关手续；6. 企业引进的人才，已与我市企业签订工作合同，缴纳社保，且承诺自接受资助当年起全职在企业工作2年以上；7. 已入选县级以上引才计划的各类人才不再列入资助范围；事业单位引进人才不列入资助范围 |
| 申请流程 | 1. 人才先提出申请；2. 由所在企业将申报材料统一报送至企业所在县（区）、市各功能区人社部门；3. 各县（区）、市各功能区人社部门受理申报材料并进行初审，不符合申报条件的及时退回；4. 审核无误后报市人社局、市委组织部进行复审；5. 复审后将拟补贴人员名单进行公示；6. 无异议后有序拨付 |
| 申请平台 | 企业所在县（区）、市各功能区人社部门（线下申报，由企业统一报送） |
| 申请材料 | 1. 引进人才的身份证或护照；2. 毕业证书和学位证书（境外学历学位须提供由教育部留学服务中心出具的认证报告）；3. 学信网学历证明；4. 人才与所在单位签订的劳动（聘用）合同；5. 社保缴费证明；6. 企业营业执照 |
| 审核周期 | 县区/功能区人社部门初审→市人社局、市委组织部复审→公示→拨付 |
| 发放方式 | 按年度发放，补贴期限2年 |
| 生效时间 | 2020年6月17日 |
| 失效时间 | null |
| 政策状态 | 有效。2025年持续受理申报，每年针对上一年度新引进人才开展申领工作。补贴期限2年，按年度发放。已入选县级以上引才计划的人才及事业单位引进人才不列入资助范围。咨询电话：0527-84353007 |
| 官方来源 | 宿迁市人民政府（http://www.suqian.gov.cn/cnsq/yszcjd/202007/3dfcfa4678e5479b98e4a4ee53320d14.shtml） |
| 最后核实日期 | 2026-07-01 |

## 二、宿迁市人才购房券

| 项目 | 内容 |
|------|------|
| 政策名称 | 宿迁市人才购房券 |
| 政策文号 | 《宿迁市人才"购房券"管理和使用暂行办法》（宿人才发〔2016〕2号），依据《关于强化高质量发展人才引领的实施意见》（宿办发〔2018〕78号） |
| 适用对象 | 当年度新引进到宿迁市中心城区全职工作或自主创业的各类人才。其中普惠部分为：全职在企业或民办非企业单位工作的紧缺急需高校毕业生 |
| 补贴金额/标准 | 全日制博士30万元、全日制硕士研究生15万元、"双一流"建设高校本科毕业生10万元、普通本科高校毕业生5万元。按4:3:3比例分三次兑现到位，初次申兑成功后需间隔1年以上方可再次申兑。另有高级技师10万元、技师5万元；院士200万、国家重点工程专家100万等高层次人才标准（需人才认定） |
| 学历要求 | 全日制博士研究生、全日制硕士研究生、"双一流"建设高校本科、普通高校本科毕业生 |
| 年龄限制 | null |
| 社保要求 | 人才应自在宿迁市中心城区缴纳社保之日（创业人才应自企业创办之日）起1年内申请 |
| 申请条件 | 1. 当年度新引进到宿迁市中心城区全职工作或自主创业；2. 全职在企业或民办非企业单位工作；3. 具备相应学历层次（全日制博士/硕士/双一流本科/普通本科）；4. 自在中心城区缴纳社保之日起1年内申请（逾期不予受理）；5. 申领后须在宿迁工作满1年并购房后方可申请兑现；6. 实施范围为中心城区（市本级、宿豫区、宿城区、宿迁经济技术开发区、湖滨新区、洋河新区、苏州宿迁工业园区） |
| 申请流程 | 1. 申领：自在中心城区缴纳社保之日起1年内，通过手机端关注"宿迁英才"微信公众号或电脑端登录宿迁党建网、宿迁市人力资源和社会保障网，点击宿迁人才服务"云平台"专栏进行在线申请；2. 兑现：工作满1年并购房后，向用人单位递交申兑材料，由用人单位统一报送所在地人社部门；3. 市人社局汇总审核，市委组织部人才处复审，确定拟兑付补贴名单并公示；4. 公示无异议后按4:3:3比例分三次发放"购房券"补贴 |
| 申请平台 | "宿迁英才"微信公众号、宿迁党建网（http://www.sqdj.gov.cn/）、宿迁市人力资源和社会保障网（http://sqhrss.suqian.gov.cn/rlzyj/index.shtml），点击宿迁人才服务"云平台"专栏 |
| 申请材料 | 1. 《人才"购房券"兑现申请表(全职工作类)》或《人才"购房券"兑现申请表(自主创业类)》（兑现时）；2. 购房证明材料（购房合同、购房全款发票、契税发票）；3. 自主创业类人才另须提供企业纳税证明。上述材料需同时提供原件和复印件 |

| 发放方式 | 按4:3:3比例分三次兑现到位，初次申兑成功后需间隔1年以上方可再次申兑。享受"购房券"补贴的房屋5年内不得上市销售 |
| 生效时间 | 2016年（宿人才发〔2016〕2号印发） |

### 区级政策互斥关系说明

> 最后核实日期：2026-07-01

宿迁市人才补贴政策采用**市级统一政策**模式，租房补贴和购房券均由市级统一制定标准，在全市（购房券限中心城区）范围内统一执行。具体情况如下：

1. **无独立区级政策**：宿豫区、宿城区、宿迁经济技术开发区、湖滨新区、洋河新区、苏州宿迁工业园区、沭阳县、泗阳县、泗洪县等不设独立的差异化人才补贴政策，统一执行市级租房补贴和购房券政策。

2. **购房券适用范围**：购房券实施范围为中心城区（市本级、宿豫区、宿城区、宿迁经济技术开发区、湖滨新区、洋河新区、苏州宿迁工业园区），沭阳县、泗阳县、泗洪县执行县级相关政策。

3. **区级配套说明**：宿豫区对获批申兑市级"人才购房券"且房屋产权在宿豫的人才，按市级"人才购房券"区级配套资金的1.2倍标准额外给予购房资金扶持，属区级叠加配套，非独立区级政策。

4. **无区级互斥关系**：由于各区执行同一市级政策、标准统一，**不存在区级政策互斥问题**。市级租房补贴与市级购房券为两项独立政策，分别针对租房居住阶段和购房阶段，可叠加享受。

5. **租房补贴与购房券关系**：租房补贴为按月发放的阶段性补贴（期限2年），购房券为分期兑现的购房支持券，二者适用阶段不同，可叠加享受。购房券按4:3:3比例分三次兑现，享受补贴房屋5年内不得上市销售。
