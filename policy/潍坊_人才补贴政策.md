# 潍坊市人才补贴政策

> 最后核实日期：2026-07-01
>
> 政策依据：潍委人组办发〔2023〕4号，由潍坊市委人才工作领导小组办公室发布并执行。

---

## 一、潍坊市青年人才生活补助

| 项目 | 内容 |
|------|------|
| 政策名称 | 潍坊市青年人才生活补助 |
| 政策文号 | 潍委人组办发〔2023〕4号 |
| 适用对象 | 全日制专科、本科、硕士、博士毕业生 |
| 补贴金额/标准 | 博士6000元/月、硕士2000元/月、本科1000元/月、大专500元/月，期限3年。全球TOP200和国内"双一流"建设高校(学科)的本/硕/博期限5年。2017年后国家统招非全日制研究生纳入。预备技师(技师)参照本科、高级工参照大专 |
| 学历要求 | 全日制专科、本科、硕士、博士（2017年后国家统招非全日制研究生纳入；预备技师(技师)参照本科、高级工参照大专） |
| 年龄限制 | 博士≤45岁、硕士≤40岁、本科≤35岁、大专需在毕业5年内 |
| 社保要求 | 2022年4月27日后初次来潍就业创业并缴纳社会保险，签≥3年合同 |
| 申请条件 | 1. 2022年4月27日后初次来潍就业创业并缴纳社会保险；2. 与用人单位签≥3年合同；3. 博士≤45岁、硕士≤40岁、本科≤35岁、大专需在毕业5年内 |
| 申请流程 | 1. 符合条件大学生由用人单位向人社部门申请；2. 纳税用人单位→注册纳税地同级人社部门；3. 中央/省属驻潍及市属学校、医院、科研院所→潍坊市人社局；4. 县级及以下所属学校、医院、科研院所→本地人社部门；5. 同时在本单位公示，各级人社部门同步审核，审核结束后发放 |
| 申请平台 | 用人单位向人社部门申请 |
| 申请材料 | 1. 《潍坊市大学生生活补助申请表》1份；2. 申请人身份证、学历(毕业)证、学位证(本科及以上)、教育部学历证书电子注册备案表(大专及以上)；3. 技工院校毕业生加职业资格(技能等级)证书；4. 海外留学归国加教育部国外学历学位认证书；5. 《潍坊市大学生生活补助申请汇总表》1份；6. 单位营业执照(或统一社会信用代码证)原件及复印件；7. 就业人员全职劳动(聘用)合同（机关事业可提供录用/入职证明）原件及复印件；8. 自主创业人员公司营业执照或相关材料原件及复印件 |
| 审核周期 | 常态化受理 |
| 发放方式 | 按月发放，期限3年（双一流/全球TOP200高校期限5年） |
| 生效时间 | 2022-04-27 |
| 失效时间 | null（持续执行） |
| 政策状态 | 有效 |
| 官方来源 | 潍坊本地宝（http://m.wf.bendibao.com/news/22716.shtm） |
| 最后核实日期 | 2026-07-01 |

## 二、潍坊市市区人才安居工程购房补贴

| 项目 | 内容 |
|------|------|
| 政策名称 | 潍坊市市区人才安居工程购房补贴 |
| 政策文号 | 潍坊市市区人才安居工程实施细则（试行） |
| 适用对象 | 全日制专科、本科、硕士、博士毕业生 |
| 补贴金额/标准 | 博士30万元、硕士12万元、本科8万元、专科4万元，一次性发放。技工院校预备技师(技师)参照本科8万元、高级工参照专科4万元 |
| 学历要求 | 全日制专科、本科、硕士、博士（预备技师(技师)参照本科、高级工参照专科） |
| 年龄限制 | 博士≤45周岁、硕士≤40周岁、本科/专科≤35周岁 |
| 社保要求 | 在市区累计缴纳社保≥12个月 |
| 申请条件 | 1. 初次来市区全职就业/创业；2. 在市区累计缴纳社保≥12个月，签≥3年合同；3. 博士≤45周岁、硕士≤40周岁、本科/专科≤35周岁；4. 购买首套国有土地上住宅，家庭在市区无住房登记，购房前3年内无住房交易记录；5. 市区范围含市本级、奎文区、潍城区、坊子区、寒亭区、高新区、滨海区、保税区、峡山区、经济区 |
| 申请流程 | 1. 每季度首月10日前由用人单位线下/线上向审核部门提报；2. 集中受理后分别发相关部门信息核查；3. 网站公示5个工作日；4. 财政到位后直接发至申请人社保卡银行账户 |
| 申请平台 | 用人单位向各级人社部门提报（中央/省属驻潍及市属机关事业→市人社局；纳税用人单位→纳税地县级审核部门；其他→注册地县级审核部门） |
| 申请材料 | 1. 《潍坊市市区青年人才购房补贴申请表》；2. 《申请汇总表》；3. 申请人身份证（已婚加结婚证）；4. 人才证明材料（学历学位证明/专科学历证明/国外学历认证书/技工院校学历+职业资格证）；5. 单位营业执照(或统一社会信用代码证)副本；6. 全职劳动(聘用)合同（机关事业可提供录用/入职证明；创业提供营业执照或登记材料）；7. 二手房提供不动产登记证书，商品房提供网签备案买卖合同；8. 仅缴医保者提供医保缴纳佐证；9. 人才基本情况在单位显著位置公示（≥3个工作日）的照片 |
| 审核周期 | 每季度首月10日前集中受理 |
| 发放方式 | 一次性发放至社保卡银行账户 |
| 生效时间 | 2022-04-27 |
| 失效时间 | null（持续执行） |
| 政策状态 | 有效 |
| 官方来源 | 潍坊本地宝（http://m.wf.bendibao.com/news/22239.shtm） |
| 最后核实日期 | 2026-07-01 |

## 三、潍坊市市区阶段性住房租赁补贴

| 项目 | 内容 |
|------|------|
| 政策名称 | 潍坊市市区阶段性住房租赁补贴 |
| 政策文号 | 潍坊市市区阶段性住房租赁补贴政策明白纸 |
| 适用对象 | 专科、本科、硕士、博士毕业生 |
| 补贴金额/标准 | 博士600元/月、硕士400元/月、本科350元/月、专科及以下300元/月，累计不超过36个月。实际租金低于核定补贴额的按实际金额发放 |
| 学历要求 | 专科、本科、硕士、博士 |
| 年龄限制 | null |
| 社保要求 | 在市区缴社保 |
| 申请条件 | 1. 2023年5月1日后在潍坊市区首次就业；2. 申请与就业登记间隔≤1年；3. 在市区缴社保；4. 在潍坊市区无住房登记且3年内无房产交易记录；5. 与人才购房补贴、公租房补贴等互斥 |
| 申请流程 | 1. 申请人填表备料；2. 所在单位初步审核；3. 统一向单位所在地区/开发区住房保障部门申请；4. 区/开发区住房保障等部门内部流转审查；5. 通过后公示；6. 进入保障，按季度发放 |
| 申请平台 | 单位所在地区/开发区住房保障部门（线下） |
| 申请材料 | 1. 申请表；2. 身份证、户口本、婚姻证明；3. 学历证明；4. 劳动合同、就业登记证明、社保缴费证明；5. 租房合同；6. 家庭住房情况核查授权 |
| 审核周期 | 每季度末月审核材料，每季度首月发放上季度补贴 |
| 发放方式 | 按季度发放，累计不超过36个月 |
| 生效时间 | 2023-05-01 |
| 失效时间 | null（持续执行） |
| 政策状态 | 有效 |
| 官方来源 | 待核实 |
| 最后核实日期 | 2026-07-01 |
