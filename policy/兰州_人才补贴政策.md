# 兰州市人才补贴政策

> 最后核实日期：2026-07-01
>
> 政策依据：《兰州市大学生留兰倍增计划房票补贴实施细则》《兰州市急需紧缺人才引进实施办法》，由兰州市住建局、兰州市人社局发布并执行。

---

## 一、兰州市大学生留兰倍增计划房票补贴（博士）

| 项目 | 内容 |
|------|------|
| 政策名称 | 兰州市大学生留兰倍增计划房票补贴（博士） |
| 政策文号 | 《兰州市大学生留兰倍增计划房票补贴实施细则》 |
| 适用对象 | 全日制博士研究生 |
| 补贴金额/标准 | 博士30万元，一次性发放房票 |
| 学历要求 | 全日制博士研究生 |
| 年龄限制 | null |
| 社保要求 | 在兰州市连续缴纳三个月社会保险或个人所得税 |
| 申请条件 | 1. 2024年5月13日后新引进；2. 与兰州市用人企业签订全职正式聘用服务合同（或自主创业）；3. 已在兰连续三个月缴纳社保或个税；4. 本人在兰州市范围内无其他住房；5. 所购房产满5年方可转让 |
| 申请流程 | 1. 个人申请（填表+提交资料）；2. 资格联审（市住建局牵头市人社局、市市场监管局等部门，5个工作日内完成核查）；3. 名单确定（市住建局拟定名单报市委人才办复核）；4. 发放房票（复核通过后5个工作日内发放）；5. 住房选购（房票持有人12个月内持房票购买新建商品住房）；6. 房票结算（网签备案后，房企凭房票和合同向市住建局申请结算） |
| 申请平台 | 兰州市政务服务中心市住建局窗口（城关区庆阳路民安大厦5楼512窗口）或兰州市人才综合服务平台 |
| 申请材料 | 1. 兰州市大学生留兰倍增计划房票补贴申请表；2. 身份证原件及复印件；3. 学历证书、学位证书原件及复印件；4. 从业材料（全职人员：劳动合同/聘用合同；创业人员：兰州市营业执照+连续三个月完税证明）；5. 在兰州市连续缴纳三个月社会保险或个人所得税证明；6. 兰州市不动产登记信息查询证明；7. 特殊情况需要的其他材料 |
| 审核周期 | 常态化受理，资格联审5个工作日 |
| 发放方式 | 房票形式发放，持票人12个月内购房抵用 |
| 生效时间 | 2024-05-13 |
| 失效时间 | null（持续执行） |
| 政策状态 | 有效。2025年起支持"人才房票+公积金贷款"组合贷 |
| 官方来源 | 兰州市住建局官网（https://zjj.lanzhou.gov.cn/art/2025/6/10/art_11369_1510988.html） |
| 最后核实日期 | 2026-07-01 |

## 三、兰州市企事业单位引进"双一流"高校毕业生租房补贴

| 项目 | 内容 |
|------|------|
| 政策名称 | 兰州市企事业单位引进"双一流"高校毕业生租房补贴 |
| 政策文号 | 《兰州市急需紧缺人才引进实施办法》 |
| 适用对象 | "双一流"建设高校全日制本科、硕士、博士毕业生 |
| 补贴金额/标准 | 博士2000元/月、硕士1500元/月、本科1000元/月，补贴期3年（36个月）。或可申请入住人才公寓（二选一） |
| 学历要求 | 全日制本科、硕士、博士（"双一流"建设高校） |
| 年龄限制 | 引进公告规定本科≤30周岁、硕士≤35周岁（医疗卫生岗位可放宽） |
| 社保要求 | 通过兰州市企事业单位集中引进程序被录用后缴纳社保 |
| 申请条件 | 1. 通过兰州市企事业单位集中引进急需紧缺人才程序被录用；2. "双一流"建设高校全日制毕业生；3. 与引进单位签订聘用合同 |
| 申请流程 | 1. 通过兰州市人社局"兰州市考生公众服务系统"报名引进；2. 线上资格审查；3. 线下资格复审；4. 面试考核；5. 体检；6. 考察；7. 公示聘用；8. 享受租房补贴 |
| 申请平台 | 兰州市人力资源和社会保障局官网"兰州市考生公众服务系统"（http://rsj.lanzhou.gov.cn/） |
| 申请材料 | 1. 报名登记表；2. 身份证；3. 简历；4. 毕业证/学位证/就业推荐表；5. 职业资格证（如有） |
| 审核周期 | 按年度引进公告集中报名 |
| 发放方式 | 按月发放，补贴期3年 |
| 生效时间 | 待核实 |
| 失效时间 | null（持续执行，2024年、2025年均发布引进公告） |
| 政策状态 | 有效 |
| 官方来源 | 兰州市人力资源和社会保障局官网（http://rsj.lanzhou.gov.cn/） |
| 最后核实日期 | 2026-07-01 |
