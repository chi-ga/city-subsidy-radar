#### 1. 大连市高校毕业生住房补贴（兴连英才计划）

| 字段 | 内容 |
|------|------|
| **政策名称** | 大连市高校毕业生住房补贴（兴连英才计划） |
| **政策文号** | 《兴连英才计划》高校毕业生住房补贴实施细则（市委人才办组织实施，自2023年1月起执行） |
| **适用对象** | 毕业后2年内、在连新就业或自主创业、取得大连市户籍的全日制本科及以上学历高校毕业生 |
| **补贴金额/标准** | 博士2500元/月、硕士1500元/月、本科1000元/月，按月发放，保障期最多36个月；国内"双一流"建设高校毕业生住房补贴标准上浮10%（即博士2750元/月、硕士1650元/月、本科1100元/月） |
| **学历要求** | 全日制本科及以上学历 |
| **年龄限制** | 无明确年龄限制，以"毕业后2年内"为时限（毕业时间以毕业证书或国(境)外学位学历认证书所注日期为准） |
| **社保要求** | 须由用人单位在连缴纳社保；在连连续缴纳社保满6个月（不含补缴）后次月起按月享受补贴；社保断缴后续缴的视同重新申报，重新计算连续缴纳社保时长。在连缴纳社保1个月即可申报 |
| **申请条件** | 1.毕业后2年内；2.具有全日制本科及以上学历；3.取得大连市户籍；4.已由用人单位在连缴纳社保的新就业或自主创业高校毕业生；5.在连连续缴纳社保满6个月（不含补缴）后次月起按月享受补贴；6.社保卡已开通金融功能 |
| **申请流程** | 1.注册登录：访问大连人才网（https://www.dl-rc.com）或"大连智慧人才"微信公众号/微信小程序、辽事通App进行实名注册→2.提交申请：进入"高校毕业生住房补贴"模块，阅读办理须知并授权查阅户籍、社保和学信网信息，在线填报学历、户籍、社保等信息→3.资格审核：每月第1-2个工作日市公安局核验户籍、学信网核验学历，市人才大数据综合服务平台进行资格确认→4.生成名单：每月第3个工作日核验连续缴纳社保满6个月情况，生成发放名单→5.资金发放：每月第4-5个工作日将补贴发放至个人社保卡账户 |
| **申请平台** | 大连人才网（https://www.dl-rc.com）；"大连智慧人才"微信公众号/微信小程序；辽事通App"e大连频道" |
| **申请材料** | 1.身份信息（系统实名认证）；2.学历信息（境内高校毕业生填写毕业学校、学历层次，系统自动核验；国(境)外高校毕业生填写教育部留学服务中心颁发的留学人员认证书编号）；3.户籍信息（选择是否大连户籍，系统比对）；4.社保信息（系统自动匹配在连缴纳社保及社保卡账户）；5.已开通金融功能的社会保障卡 |
| **审核周期** | 每月集中审核，资格确认通过后次月发放（约1个月） |
| **发放方式** | 按月发放，发放至个人社保卡账户（每月第4-5个工作日） |
| **生效时间** | 2023年1月（由市委人才办组织实施） |
| **失效时间** | 待核实（政策持续执行中） |
| **政策状态** | 有效，政策持续执行中 |
| **官方来源** | https://m.dl.bendibao.com/news/75130.shtm ； https://m.dl.bendibao.com/mip/75132.shtm |
| **最后核实日期** | 2026-07-01 |

#### 2. 大连市高校毕业生学费补助

| 字段 | 内容 |
|------|------|
| **政策名称** | 大连市高校毕业生学费补助 |
| **政策文号** | 《关于组织开展2025年度高校毕业生学费补助项目申报工作的通知》（大连市人社局） |
| **适用对象** | 2022届及以后的国内全日制普通高校毕业生，毕业3年内在连稳定就业满2年（累计缴纳24个月及以上社保） |
| **补贴金额/标准** | 按博士、硕士、本科、专科4个层次一次性给予3万元、2万元、1万元、0.8万元学费补助；国内一流建设高校毕业生上浮40%；国内一流学科建设高校毕业生上浮20%；在"两市一县"（瓦房店市、庄河市、长海县）就业的国内高校毕业生上浮20%（同时符合两个及以上标准的按"就高不重复"原则执行，只能申领一次） |
| **学历要求** | 国内全日制普通高校博士、硕士、本科、专科毕业生 |
| **年龄限制** | 无明确年龄限制，以"2022届及以后、毕业3年内"为时限 |
| **社保要求** | 毕业后已由用人单位在连累计缴纳24个月及以上社会保险 |
| **申请条件** | 1.2022届及以后的国内全日制普通高校毕业生；2.毕业3年内已由用人单位在连累计缴纳24个月及以上社会保险；3.同时符合下列条件之一：（1）在我市稳定就业的在连高校毕业生（学籍地在大连市），（2）在我市稳定就业的国内"双一流"高校毕业生，（3）在"两市一县"稳定就业且在"两市一县"累计缴纳24个月及以上社会保险的国内高校毕业生 |
| **申请流程** | 1.在线提交：访问大连人才网（https://www.dl-rc.com）或"大连智慧人才"微信公众号/微信小程序，进入"高校毕业生学费补助"模块，通过身份验证后在线提交学历信息→2.自动比对：平台系统按申请顺序分批次自动进行学历信息、社保缴纳信息比对→3.区划确认：市人社部门会同各地区人社部门对申请人所在单位进行区划确认→4.社会公示：市人社部门将符合条件人员名单及拟发补贴额度在市人社局官网公示5个工作日→5.资金发放：公示无异议后，市人社部门向市财政部门提出资金分配方案，补助资金拨付至申请人开通金融功能的大连市社保卡账户 |
| **申请平台** | 大连人才网（https://www.dl-rc.com）；"大连智慧人才"微信公众号/微信小程序 |
| **申请材料** | 1.身份信息（系统实名认证）；2.学历信息（在线提交，系统通过学信网自动比对核验）；3.已开通金融功能的大连市社会保障卡（用于接收补助资金） |
| **审核周期** | 集中受理期（如2025年度为11月18日至12月1日），审核公示后拨付资金 |
| **发放方式** | 一次性发放，拨付至大连市社保卡账户 |
| **生效时间** | 2022届毕业生起执行 |
| **失效时间** | 待核实（政策持续执行中） |
| **政策状态** | 有效，每年集中受理一次 |
| **官方来源** | https://m.dl.bendibao.com/live/87568.shtm ； https://m.dl.bendibao.com/live/87569.shtm ； https://www.rsj.dl.gov.cn/ |
| **最后核实日期** | 2026-07-01 |

| **审核周期** | 现场受理审核后公示7天，公示无异议后拨付资金 |

| **生效时间** | 2022年5月26日 |
| **失效时间** | 待核实（政策持续执行中，按年度集中受理） |
| **政策状态** | 有效，按年度集中受理 |
| **官方来源** | https://m.dl.bendibao.com/news/7512389676028690985 ； 咨询电话39997553 |

## 区级政策互斥关系说明

大连市各区级人才补贴政策属于**互斥关系**，即同一申请人只能选择在一个区落户/就业并享受该区的人才补贴政策，不可同时在多个区重复申领区级人才补贴。

- **互斥组标识**：`dalian-district`
- **纳入互斥组的区级政策**：金普新区"名校优才"生活补贴
- **说明**：用户在大连市只能选择一个区（如金普新区）作为就业/创业所在地并享受该区区级人才补贴。市级政策（如住房补贴、学费补助、购房补贴等）与区级政策之间不构成互斥关系，符合条件者可叠加享受（具体叠加规则以各政策条款为准，如购房券与一次性购房补贴不可叠加）。
- **注意**：若后续检索到其他区（如高新区、旅顺口区、中山区、西岗区、沙河口区、甘井子区等）的区级人才补贴政策，应一并纳入 `dalian-district` 互斥组。本次调研中，除金普新区外，其他区暂未检索到面向普通高校毕业生、条件可量化且独立的区级人才补贴政策（部分区政策为高层次人才认定型或房交会期间临时性补贴，已按规则排除）。

## 政策筛选说明（排除政策）

以下政策因不符合纳入条件而未收录：

1. **引进高层次人才安家费（兴连英才计划）**：面向尖端人才、领军人才、高端人才、青年才俊，需经高层次人才认定（500万/260万/120万/30万元），属"需要特殊人才认定"型政策，普通毕业生无法仅凭学历+年龄+社保判断资格，已排除。
2. **高校毕业生一次性求职补贴**：面向低保家庭、零就业家庭、特困人员、残疾、烈士子女、孤儿及获得国家助学贷款的毕业学年高校毕业生，需提供低保证、残疾证、烈士证明等特殊证件，非面向普通毕业生的普惠型人才补贴，已排除。
3. **"大连工匠"奖励**：需经市人社局新选树为"大连工匠"（一次性12万元），属特殊认定型，已排除。
4. **高新区房交会期间人才购房补贴**：为2025年春季房交会（2025年3月14日-31日）期间临时性补贴，已过期且非常态化政策，已排除。
