#### 1. 青岛市高校毕业生住房补贴

| 字段 | 内容 |
|------|------|
| **政策名称** | 青岛市高校毕业生住房补贴 |
| **政策文号** | 青人社发〔2024〕6号（《关于实施高校毕业生住房补贴政策有关问题的通知》） |
| **适用对象** | 毕业三年内的国内普通高校统招全日制本科及以上学历毕业生；毕业三年内的国（境）外高等院校本科学历留学人员；三年内来青就业创业的国（境）外高等院校研究生学历留学人员；2022年5月5日及以后来青就业创业的毕业三年内外籍及港澳台本科及以上学历高校毕业生 |
| **补贴金额/标准** | 本科毕业生每人每月500元、硕士研究生每人每月800元、博士研究生每人每月1200元。自首次享受补贴月份起，最长可享受36个月 |
| **学历要求** | 全日制本科及以上学历，并获得相应学士、硕士、博士学位；国（境）外高校毕业生须取得教育部留学服务中心学历学位认证 |
| **年龄限制** | null |
| **社保要求** | 在青岛市就业创业，并由青岛市用人单位或创业实体（含个体工商户）按规定为其缴纳城镇职工社会保险，包括以灵活就业身份缴纳城镇职工社会保险的无雇工个体工商户负责人 |
| **申请条件** | 1. 毕业三年内的国内普通高校统招全日制本科及以上学历毕业生，或符合条件的国（境）外留学人员、外籍及港澳台高校毕业生；2. 申领补贴时须具有青岛市户籍（在"两区"就业创业的外籍及港澳台高校毕业生、2022年5月5日及以后来青就业创业的外籍及港澳台高校毕业生不受户籍限制）；3. 在青岛市就业创业并按规定缴纳城镇职工社会保险；4. 未享受本市引进优秀高层次人才购房安家补贴、博士后聚青资助、博士后生活资助、青年人才在青创新创业一次性安家费政策（已申领住房补贴的，申请上述补贴时按就高原则扣减） |
| **申请流程** | 1. 登录青岛人才网进入"青岛人社·学历汇"完成学历确认；2. 登录青岛人才网"高校毕业生住房补贴申报"模块录入信息，或到用人单位或创业实体注册登记地所在区（市）人力资源社会保障部门现场办理；3. 区（市）人力资源社会保障部门每月两次通过系统线上汇总核实；4. 审核通过后报同级财政部门申请资金；5. 资金到账后5个工作日内发放到申请人 |
| **申请平台** | 青岛人才网（https://rc.qingdao.gov.cn/）或各区（市）人力资源社会保障部门窗口 |
| **申请材料** | 1. 身份证；2. 《教育部学历证书电子注册备案表》、毕业证、学位证、就业报到证（2023年1月1日及以后毕业的无需提供就业报到证）；3. 国（境）外高校毕业生提供《国（境）外学历学位认证书》；4. 在青岛行政区域外缴纳社会保险的，需提交当地社保部门出具的社会保险缴纳情况相关材料 |
| **审核周期** | 每月两次线上汇总核实，系统无法核实的按审核时间要求现场审核 |
| **发放方式** | 按月发放至申请人银行账户 |
| **生效时间** | 2024年4月1日 |
| **失效时间** | 2027年3月31日 |
| **政策状态** | 有效。此前按青人社规〔2021〕1号申领但未享受期满的人员，可按本通知领取剩余期限补贴 |

#### 2. 青岛市青年人才在青创新创业一次性安家费

| 字段 | 内容 |
|------|------|

| **政策文号** | 青人社发〔2024〕7号（《青岛市人力资源和社会保障局等5部门关于实施青年人才在青创新创业一次性安家费政策有关问题的通知》） |
| **适用对象** | 2018年6月6日及以后在青岛行政区域内首次就业创业的国内普通高校统招全日制研究生、经教育部留学服务中心国（境）外学历学位认证的留学回国研究生；2022年5月5日及以后来青就业创业的外籍及港澳台研究生 |
| **补贴金额/标准** | 博士研究生每人15万元，硕士研究生每人10万元，一次性发放 |
| **学历要求** | 全日制研究生毕业并获得相应硕士或博士学位；国（境）外高校毕业生须取得教育部留学服务中心学历学位认证 |
| **年龄限制** | 博士研究生年龄40周岁（不含）以下，硕士研究生年龄35周岁（不含）以下，以申报时实际年龄为准 |
| **社保要求** | 申请人须由青岛市用人单位或创业实体按规定缴纳城镇职工社会保险累计满12个月，包括以灵活就业身份缴纳城镇职工社会保险的无雇工个体工商户负责人 |
| **申请条件** | 1. 2018年6月6日及以后在青岛行政区域内首次就业创业；2. 博士研究生年龄40周岁（不含）以下，硕士研究生年龄35周岁（不含）以下；3. 具有青岛市户籍（外籍及港澳台研究生不受户籍限制）；4. 在青岛市各类企业或国家机关、事业单位、民办非企业单位、社会团体等用人单位就业的，须与就业单位签订劳动（聘用）合同；在青创业的，须登记注册成立企业、民办非企业单位、社会团体、事务所以及个体工商户1年及以上，并担任法定代表人或个体工商户负责人；5. 申请人须于2018年6月6日及以后在青岛行政区域内购买唯一商品住宅（购房时间以合同网签备案时间为准）；6. 未婚的，所购商品住宅须为个人在青唯一商品住宅且产权为申请人单独所有；已婚的，所购商品住宅为夫妻在青唯一商品住宅，产权为申请人或配偶单独所有，或夫妻共有 |
| **申请流程** | 1. 登录青岛人才网"青岛人社·学历汇"完成学历确认；2. 登录青岛人才网"在青创新创业安家费申报"模块录入信息，或持相关材料到用人单位或创业实体注册登记地所在区（市）人力资源社会保障部门现场办理；3. 区（市）人力资源社会保障部门自次月首个工作日开始线上核实；4. 对不动产登记、享受产权型人才住房、保障性住房等信息进行核实比对；5. 审核通过后公示5个工作日；6. 报同级财政部门审核，补贴资金到账后5个工作日内发放到申请人 |
| **申请平台** | 青岛人才网（https://rc.qingdao.gov.cn/）或各区（市）人力资源社会保障部门窗口 |
| **申请材料** | 1. 申请人及配偶身份证；2. 不动产权证或不动产登记证明；3. 结婚证（已婚）；4. 在青岛行政区域外缴纳社会保险的，需提交当地社保部门出具的社会保险缴纳情况相关材料；5. 学历学位证书及认证材料（学历汇已确认的可共享） |
| **审核周期** | 线上核实自次月首个工作日起开始，审核通过后公示5个工作日 |
| **发放方式** | 一次性发放至申请人账户 |
| **生效时间** | 2024年4月1日 |
| **失效时间** | 2027年3月31日 |
| **政策状态** | 有效。已享受本市引进优秀高层次人才购房安家补贴、博士后聚青资助的不再享受；已享受安家费的，申请上述补贴时按就高原则扣减；已申领高校毕业生住房补贴的，在安家费中予以扣减；夫妻双方均符合条件的可分别享受 |
| **官方来源** | http://www.qingdao.gov.cn/zwgk/xxgk/rlshbz/gkml/gwfg/202404/t20240411_7983148.shtml |
