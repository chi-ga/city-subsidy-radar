#### 1. 无锡市优秀青年人才来锡租房补贴

| 字段 | 内容 |
|------|------|
| **政策名称** | 无锡市优秀青年人才来锡租房补贴 |
| **政策文号** | 《关于更大力度支持优秀青年人才来锡发展的十项措施》（锡人才办〔2024〕3号）、关于2025年度优秀青年人才来锡租房补贴申领的通知 |
| **适用对象** | 2024年9月1日后新来锡就业参保的以下人才：全日制普通高校本科及以上学历并取得相应学位的毕业生（其中本科、硕士人才本人所学专业须进入《无锡市2025年度重点产业紧缺人才需求目录》）；在校期间获得技师（二级）技能等级（国家职业资格）的职业院校（技工院校）毕业生；获得高级技师（一级）及以上技能等级（国家职业资格）的人员。均须从事紧缺岗位工种 |
| **补贴金额/标准** | 本科（技师）1万元/年、硕士（高级技师及以上）1.5万元/年、博士3万元/年，累计支持最长两年 |
| **学历要求** | 全日制普通高校本科及以上学历并取得相应学位；职业院校（技工院校）毕业生需取得技师及以上技能等级 |
| **年龄限制** | null |
| **社保要求** | 毕业后5年内新来无锡市区工作，缴纳企业职工基本养老保险或在锡科研院所全职工作；来锡新增参保满一个月即可申领；申领时须为社保正常缴费状态 |
| **申请条件** | 1. 2024年9月1日后新来锡就业参保；2. 毕业5年内的全日制普通高校本科及以上学历毕业生（本科、硕士专业须进入年度紧缺人才需求目录）或技师/高级技师技能人才；3. 完成“太湖人才计划”人才分类认定；4. 在无锡市区范围内自费租房并进行房屋租赁登记备案；5. 不包括机关、其他事业单位工作人员（含编外）；6. 已申领购房补贴、市级其他安居补贴或区级租房补贴的不再享受；租住直系亲属房产的不纳入范围；就读相应学历期间缴纳社保者不符合条件（在校参军或6个月以内短暂缴纳除外） |
| **申请流程** | 1. 完成“太湖人才计划”人才分类认定；2. 申请并激活太湖人才卡；3. 通过“无锡人才”微信公众号或无锡人才工作网提交租房补贴申请；4. 用人单位在无锡人才工作网复核；5. 参保地街道（镇）相关职能部门10个工作日内完成初审；6. 参保所在区人社部门次月前10个工作日内完成复审；7. 区人社部门网站公示7个自然日；8. 单位确认发放名单和金额；9. 按季发放至太湖人才卡账户 |
| **申请平台** | “无锡人才”微信公众号—无比爱才—人才服务—人才补贴—青年人才租房补贴；无锡人才工作网（https://www.wxrcgz.com） |
| **申请材料** | 1. 毕业证书和学位证书；2. 国（境）外毕业生须同时提供教育部留学服务中心出具的学历学位认证报告；3. 如学历学位认证报告未明确所学专业或无法与紧缺目录专业精准匹配的，须补充提供相关佐证材料；4. 职业院校（技工院校）毕业生提供毕业证书及技能等级（国家职业资格）证书；5. 房屋租赁登记备案证明；6. 本人亲笔签名的《优秀青年人才补贴申领诚信承诺书》 |
| **审核周期** | 街道（镇）10个工作日内初审，区人社部门次月前10个工作日内复审，公示7个自然日 |
| **发放方式** | 按月审核、按季发放至申请人太湖人才卡账户 |
| **生效时间** | 2024年9月1日 |
| **失效时间** | 待核实 |
| **政策状态** | 有效。须在毕业5年内提出申请，并在初次申领3年内领取完毕，逾期不再受理。本政策需先完成“太湖人才计划”人才分类认定，属于专业通道层政策 |
| **官方来源** | https://www.wuxi.gov.cn/doc/2025/10/11/4675443.shtml |
| **最后核实日期** | 2026-07-01 |

| **申请材料** | 1. 毕业证书和学位证书；2. 国（境）外毕业生须同时提供教育部留学服务中心出具的学历学位认证报告；3. 如毕业证、学位证、学历学位认证报告未明确所学专业或无法与紧缺目录专业精准匹配的，须补充提供相关佐证材料；4. 职业院校（技工院校）毕业生提供毕业证书及技能等级（国家职业资格）证书；5. 不动产权登记证书；6. 契税凭证；7. 结婚证（已婚提供）；8. 《优秀青年人才购房申领诚信承诺书》 |
| **审核周期** | 街道（镇）及区人社部门初审、复审，公示7个自然日 |
| **发放方式** | 按月审核、按季发放；10万元及以下一次性发放，10万元以上分3年按4:3:3比例发放至太湖人才卡账户 |
| **生效时间** | 2024年9月1日 |
| **失效时间** | 待核实 |

#### 4. 无锡市区大学生一次性生活补贴

#### 5. 无锡市区大学生购房补贴

| 字段 | 内容 |
|------|------|
| **政策名称** | 无锡市区大学生购房补贴 |
| **政策文号** | 《无锡市区大学生租房补贴、购房补贴、一次性生活补贴申领审核办法》（锡人社发〔2023〕39号） |
| **适用对象** | 2017年7月1日及以后毕业的大学生或出站的博士后，毕业（出站）在无锡市区工作缴纳社保后在市区购买首套住房，由所在单位在锡累计缴纳社会保险1年及以上，且申领时毕业（出站）不超过五年 |
| **补贴金额/标准** | 待核实（按锡人社发〔2023〕39号执行） |
| **学历要求** | 2017年7月1日及以后毕业的大学生或出站博士后 |
| **年龄限制** | null |
| **社保要求** | 由所在单位在锡累计缴纳社会保险1年及以上，申领时处于正常缴纳社保状态 |
| **申请条件** | 1. 2017年7月1日及以后毕业的大学生或出站博士后；2. 毕业（出站）在无锡市区工作缴纳社保；3. 在市区购买首套住房并取得不动产权证书；4. 申领时毕业（出站）不超过五年；5. 首套住房指申请人单独购买或与配偶共同购买的自住商品住宅，建筑面积不低于54平方米，总价不低于申请的补贴金额（以契税发票为准），低于补贴金额的按实际价格发放；6. 取得不动产权证书前本人名下没有市区居住用房屋所有权登记记录；7. 夫妻双方共同购买且双方都符合条件的只支持其中一人申请；8. 已申领购房补贴的不再发放租房补贴 |
