#### 1. 嘉兴市企业人才生活补贴

| 字段 | 内容 |
|------|------|
| **政策名称** | 嘉兴市企业人才生活补贴 |
| **政策文号** | 待核实 |
| **适用对象** | 2020年及以后初次来嘉兴市级（南湖区、秀洲区、嘉兴经济技术开发区）企业工作，或自主创业的高校毕业生 |
| **补贴金额/标准** | 博士研究生6万元、硕士研究生3万元；"双一流"高校及2024年度QS世界大学排名前300名高校全日制应届本科毕业生2万元；其他高校全日制应届本科毕业生1万元；全日制应届专科毕业生0.6万元。一次性发放 |
| **学历要求** | 全日制专科及以上；本科分双一流/QS前300与其他高校两档 |
| **年龄限制** | null |
| **社保要求** | 来市级企业工作并连续缴纳社保满6个月 |
| **申请条件** | 1.2020年及以后初次来嘉兴工作；2.与市级企业签订劳动合同或自主创业；3.连续缴纳社保满6个月；4.申请时仍在岗；5.在机关事业单位、国有企业以及烟草、电力、通信、石化、邮政、银行、保险等特定行业工作的本科及以下学历人才不纳入 |
| **申请流程** | 1.登录嘉兴市人社局官网"嘉兴人才在线"或"浙里办"APP"嘉兴人才码"；2.完成人才认定；3.进入"资金申领"→"个人补贴申请"；4.填写信息并上传材料；5.初审通过后打印申请表，签字盖章后上传；6.审核通过后拨付 |
| **申请平台** | 嘉兴市人力社保局官网"嘉兴人才在线"（https://rlsbj.jiaxing.gov.cn）或"浙里办"APP"嘉兴人才码" |
| **申请材料** | 1.申请人证件照；2.身份证（正反面）或护照（本人页）；3.毕业证书；4.学位证书；5.学信网《教育部学历证书电子注册备案表》；6.劳动合同（自主创业的提供营业执照副本）；7.社保证明；8.国（境）外就读的提供教育部留学服务中心出具的国（境）外学历学位认证书 |
| **审核周期** | 待核实 |
| **发放方式** | 一次性发放 |
| **生效时间** | 2020年1月1日 |
| **失效时间** | 待核实 |
| **政策状态** | 有效（2025年度申报时间为4月28日至5月31日，2026年度拟于9月初启动） |
| **官方来源** | https://m.jx.bendibao.com/cyfw/13599_2.shtm |
| **最后核实日期** | 2026-06-30 |

| **审核周期** | 待核实 |
| **发放方式** | 一般分两次发放，首次50%，满3年后发剩余50%；已工作满3年可一次性全额 |
| **生效时间** | 2017年1月1日（博士/硕士）；2020年1月1日（本科） |
| **失效时间** | 待核实 |
| **政策状态** | 有效（2025年度申报时间为4月28日至5月31日） |
