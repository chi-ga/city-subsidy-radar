#### 1. 产业领域急需紧缺青年人才生活补助

| 字段 | 内容 |
|------|------|
| **政策名称** | 产业领域急需紧缺青年人才生活补助 |
| **政策文号** | 南人社发〔2026〕1号 |
| **适用对象** | 首次引进到《南宁市产业发展急需紧缺人才企业名录》企业的应届高校毕业生，或《名录》以外南宁市其他企事业单位的应届博士研究生 |
| **补贴金额/标准** | 引进到《名录》单位：博士研究生/副高级及以上职称每年5万元、最高25万元；全日制硕士研究生/2017年起国家统一下达招生计划的非全日制硕士研究生/高级技师/中级职称每年2万元、最高10万元；全日制本科毕业生/技师/初级职称一次性1万元。引进到《名录》以外单位：博士研究生每年2万元、最高10万元 |
| **学历要求** | 全日制本科、硕士、博士（《名录》单位）；博士（《名录》以外单位） |
| **年龄限制** | 40周岁（含）以下 |
| **社保要求** | 在引才单位依法参加南宁市（或自治区本级）职工基本养老保险并连续缴纳6个月（含）以上 |
| **申请条件** | 1. 首次引进到《名录》单位的应届高校毕业生，或《名录》以外南宁市其他企事业单位的应届博士研究生；2. 引进时间须在2022年9月18日（含）之后（《名录》单位须在名录有效期内）；3. 与引才单位签订3年（含）以上劳动合同；4. 连续缴纳职工基本养老保险费6个月（含）以上；5. 年龄40周岁（含）以下；6. 申报时仍在首次引才单位就业；7. 须在符合上述条件之日起6个月内申报；8. 2018年1月1日以后从南宁市机关企事业单位离职再到《名录》单位工作的不在申报范围 |
| **申请流程** | 1. 个人登录南宁人社网上办事大厅或微信公众号、微信小程序申报；2. 首次引才单位网上核验确认；3. 人社部门审核、公示5个工作日；4. 拨付至申请人社会保障卡银行账户 |
| **申请平台** | 南宁人社网上办事大厅（https://ggfw.nn12333.com:8081/nnmh/）、“南宁人社”微信公众号、“南宁智慧人社”微信小程序 |
| **申请材料** | 1. 劳动合同扫描件或照片；2. 毕业证、学位证扫描件或照片；3. 专业技术资格证书、职业技能等级证书、任职文件扫描件或照片（如有）；4. 社会保障卡（需开通金融功能） |
| **审核周期** | 15个工作日内审核，公示5个工作日 |
| **发放方式** | 博士研究生、副高级及以上职称、硕士研究生、高级技师、中级职称及市属国有企业引进人才按服务年限每年度发放；全日制本科毕业生、技师、初级职称由属地县（市、区）、开发区人社部门一次性发放 |
| **生效时间** | 2026-01-12 |
| **失效时间** | 待核实 |
| **政策状态** | 有效。已享受青年人才生活补助但同时符合高层次人才首次购房补贴及毕业生专项生活补助条件的，按“就高不重复”原则补足差额 |
| **官方来源** | https://rsj.nanning.gov.cn/zwgk/fdzdgknr/xzffxwj/t6526706.html |
| **最后核实日期** | 2026-06-30 |

#### 2. 大中专院校毕业生专项生活补助

| 字段 | 内容 |
|------|------|
| **政策名称** | 大中专院校毕业生专项生活补助 |
| **政策文号** | 待核实 |
| **适用对象** | 2022年9月1日—2025年12月31日期间毕业的中等专业学校、中等职业学校、技工学校普通全日制应届毕业生，普通高等学校应届毕业生（含普通高等教育全日制大专及以上学历毕业生、普通高等教育非全日制研究生），以及取得国（境）外学士及以上层次学历学位证书或高等教育文凭的毕业生 |
| **补贴金额/标准** | 每人每月1000元（税前），最长10个月，最高10000元（税前） |
| **学历要求** | 大专及以上、中专/中职/技校全日制应届 |
| **年龄限制** | null |
| **社保要求** | 与引进企业签订1年及以上劳动合同，并在该企业依法连续缴纳南宁市或自治区本级企业职工基本养老保险费3个月及以上 |
| **申请条件** | 1. 毕业一年内新引进至南宁市重点企业就业（以毕业后在引进企业首次参加企业职工基本养老保险的年月为准）；2. 签订1年（含）以上劳动合同；3. 连续缴纳企业职工基本养老保险3个月及以上；4. 申报时仍在该企业就业，且该企业仍属于南宁市重点园区的重点企业；5. 未享受过南宁市产业领域急需紧缺青年人才生活补助、2022年度应届高校毕业生万元就业奖励 |
| **申请流程** | 1. 登录“南宁智慧人社”APP、微信公众号或小程序、网上办事大厅申报；2. 填写信息并上传材料；3. 审核通过后按月发放 |
| **申请平台** | “南宁智慧人社”APP、“南宁人社”微信公众号、“南宁智慧人社”微信小程序、南宁人社网上办事大厅 |
| **申请材料** | 1. 《南宁市大中专院校毕业生专项生活补助申报表》扫描件或照片；2. 劳动合同原件扫描件或照片；3. 毕业证原件扫描件或照片（或《教育部学历证书电子注册备案表》），国（境）外院校毕业生需提供教育部留学服务中心学历学位认证书；4. 社会保障卡原件正反面扫描件或照片 |
| **审核周期** | 待核实 |
| **发放方式** | 按月发放，最长10个月 |
| **生效时间** | 2023-08-01 |
| **失效时间** | 2025-12-31（毕业时间截止），申报截止时间见说明 |
| **政策状态** | 有效。2022年9月1日—12月31日毕业的申报截止2024年10月31日；2023年、2024年、2025年内毕业的申报截止时间均为2026年10月31日 |
| **官方来源** | https://m.nn.bendibao.com/live/78247.shtm |
| **最后核实日期** | 2026-06-30 |

### 良庆区

> 最后核实日期：2026-06-30
> 数据来源：良庆区人民政府网、广西人社厅

> **互斥关系说明：** 南宁各区均可按广西统一政策申领高校毕业生一次性创业扶持补贴，同一人员只享受一次，不与其他区级人才补贴构成互斥；本项属于创业补贴，与前述市级生活补助、就业奖励可叠加享受。

#### 1. 高校毕业生一次性创业扶持补贴

| 字段 | 内容 |
|------|------|
| **政策名称** | 高校毕业生一次性创业扶持补贴 |
| **政策文号** | 桂人社规〔2022〕3号 |
| **适用对象** | 首次在设区市辖区内创办小微企业且所创企业自工商注册登记之日起正常运营6个月（含）以上的毕业年度和毕业5年内高校毕业生 |
| **补贴金额/标准** | 一次性5000元创业扶持补贴 |
| **学历要求** | 高校毕业生（大专及以上） |
| **年龄限制** | null |
| **社保要求** | null |
| **申请条件** | 1. 首次在设区市辖区内创办小微企业；2. 所创企业自工商注册登记之日起正常运营6个月（含）以上；3. 该小微企业的注册登记日应距申请奖补之日不超过18个月（含），且申请奖补时仍在经营；4. 同一人员创办多个创业实体、符合多个创业补贴申请条件，只享受一次；5. 同一创业地址有多个创业实体的，只有1家创业实体享受创业扶持补贴 |
| **申请流程** | 1. 准备申请材料；2. 线上通过“广西人社服务”微信公众号或“广西人社”APP申报，或线下到人社服务大厅窗口提交；3. 窗口预审；4. 财政部门复核；5. 拨付至申请人银行账户 |
| **申请平台** | “广西人社服务”微信公众号、“广西人社”APP、南宁市各级人社服务大厅 |
| **申请材料** | 1. 《高校毕业生一次性创业扶持补贴申请表》；2. 有效居民身份证或社会保障卡；3. 高校毕业生提供毕业证书，毕业年度的学生提供《学生证》或学校证明（核原件）；4. 营业执照（可通过部门内部核查的，无需申请人提供）；5. 体现企业正常经营6个月（含）以上的材料（企业发放工资银行流水、进销货单据等），如可查询企业社会保险缴费情况的，可免提供生产经营情况佐证材料；6. 创业者所创企业的银行基本账户或创业者本人社会保障卡金融账户复印件 |
| **审核周期** | 12个工作日内短信告知结果 |
| **发放方式** | 一次性 |
| **生效时间** | 2022年（桂人社规〔2022〕3号） |
| **失效时间** | 待核实 |
| **政策状态** | 有效。属广西统一政策，南宁各区均可申领，办理地点为企业注册地所在城区 |
| **官方来源** | http://www.liangqing.gov.cn/zwfw/bmfw/yjsfw/jycy/cybtsl/ |
| **最后核实日期** | 2026-06-30 |
