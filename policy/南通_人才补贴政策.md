#### 1. 南通市区人才房票（购房补贴）

| 字段 | 内容 |
|------|------|
| **政策名称** | 南通市区人才房票 |
| **政策文号** | 《南通市区人才房票实施办法》、《南通市区人才房票购房补贴实施细则》（通住建房〔2025〕113号） |
| **适用对象** | 2021年1月1日（含）以后从南通行政区域外首次到南通市区就业创业并缴纳社保的大专及以上学历人才；同时覆盖意向来（留）南通市区工作的全日制普通高校本科及以上在校生。适用范围为南通市区（崇川区、通州区、海门区、南通经济技术开发区、苏锡通科技产业园区、通州湾示范区） |
| **补贴金额/标准** | 全日制大专生（或高级工）8万元；全日制本科生（或技师）10万元；硕士研究生（或副高级职称、高级技师）15万元；博士研究生（或正高级职称、特级技师）30万元；江海英才计划引进高层次人才40-150万元。夫妻双方均符合条件的，按较高标准1.2倍执行 |
| **学历要求** | 全日制大专及以上学历（含技工院校高级工班、预备技师/技师班） |
| **年龄限制** | null |
| **社保要求** | 截至申报时在南通市区缴纳企业职工养老保险满3个月（以提交申请之月往前算起）；硕博档须在首次来通之前取得研究生学历和硕士以上学位 |
| **申请条件** | 1.2021年1月1日（含）以后从南通行政区域外首次到南通市区就业；2.按规定在南通市区缴纳企业职工养老保险满3个月；3.与南通市区用人单位签订3年及以上劳动合同；4.在市区房源库内购置新建商品住房；5.人才房票与现行人才购房补贴等政策按"不重复"原则施行 |
| **申请流程** | 1.人才申请：符合条件的人才在人才房票管理平台办理资格申请，如实填报信息并上传材料；2.资格审核：市委人才办、市人社局联审，审核结果推送市住建局，市住建局官网公示；3.房票发放：公示无异议后由市住建局向申请人发放人才房票；4.房票使用：申请人在人才房源库中选择新建商品住房，将人才房票交由开发企业核验，按票面金额一次性抵扣购房款；5.房票结算：开发企业凭房票、已网签备案的购房合同等材料申请资金结算，12个月内审核拨付 |
| **申请平台** | 人才房票管理平台（南通市住建局，平台地址另行通知） |
| **申请材料** | 1.学历学位证明（最高学历学位证书，学信网下载的教育部学历证书电子注册备案表；留学回国人员提供教育部留学服务中心出具的国外学历证明）；专业技术人才提供江苏省高级专业技术资格证书；技能人才提供职业资格证书或职业技能等级证书；2.申请人才与用人单位签订的三年及以上劳动合同；3.结婚证（申请人和配偶共同登记的需提供） |
| **审核周期** | 联审核验后公示，公示无异议后发放房票 |
| **发放方式** | 人才房票抵扣购房款（不含首付款），一人一票，专款专用；房票自开具之日起12个月内有效，逾期未使用自动失效；享受房票的房产自合同备案之日起5年内不得变更产权所有人 |
| **生效时间** | 2026-01-01 |
| **失效时间** | 2026-12-31（有效期1年） |
| **政策状态** | 有效。2026年1月1日起施行，将本科（10万元）、大专（8万元）人才购房补贴从无到有提档升级 |
| **官方来源** | https://m.nt.bendibao.com/mip/68969.shtm ； https://www.nantong.gov.cn/ntsrmzf/spxw/content/49e9bd2e-eeaa-4668-8f3f-49b8876663ec.html |
| **最后核实日期** | 2026-07-01 |

| **申请平台** | 南通市网上申报管理平台（南通市人社局） |
| **申请材料** | 1.最高学历学位证书（留学回国须提供教育部留学服务中心出具的国外学历学位认证书）、专业技术职务、职业资格等相关证件；2.所在单位的营业执照；3.与用人单位签订的劳动合同 |
| **审核周期** | 集中审核，审核通过后公示 |
| **发放方式** | 按月发放至社会保障卡银行账户 |
| **生效时间** | 2021-01-01 |
| **失效时间** | 待核实（政策持续执行中） |
| **政策状态** | 有效，政策持续执行中 |
| **官方来源** | https://m.nt.bendibao.com/live/60684.shtm ； https://m.nt.bendibao.com/live/60681.shtm |

### 区级政策互斥关系说明

1. **县（市）购房类政策互斥：** 南通市下辖的如东县（人才房票）、如皋市（人才房票）、海安市（人才购房补贴）、启东市（人才购房券）均有独立的人才购房补贴政策，四者之间以及与南通市区人才房票之间按人才就业/购房所在地互斥，同一申请人一般只能在其就业或购房所在县（市、区）享受一项购房类补贴，不得跨区域重复享受。

2. **市区统一政策：** 南通市区（崇川区、通州区、海门区、南通经济技术开发区、苏锡通科技产业园区、通州湾示范区）执行市级统一的人才政策（人才房票、硕博士生活津贴、青年人才综合补贴、博士后生活津贴），市区内不因行政区划不同而互斥。

3. **购房类与生活津贴类可叠加：** 市级人才房票（购房补贴）与硕博士生活津贴、青年人才综合补贴属于不同类型补贴，符合条件的可分别申请；但人才房票与原硕博士购房补贴按"不重复"原则执行（房票为购房补贴的升级形式）。

4. **崇川区大专生综合补贴：** 崇川区大专生人才综合补贴为区级政策，与市级青年人才综合补贴（大专档）按就高不重复原则执行，不叠加享受。

5. **县（市）政策与市区政策互斥：** 如东、如皋、海安、启东的县（市）级人才购房政策与南通市区人才房票按就业/购房所在地互斥；人才流动到市区企业就业的，其最长享受月数须扣除原在县（市）缴纳企业职工养老保险的月数。
