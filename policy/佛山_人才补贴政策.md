#### 1. 佛山市新引进博士博士后安家补贴

| 字段 | 内容 |
|------|------|
| **政策名称** | 佛山市新引进博士博士后安家补贴 |
| **政策文号** | 佛发〔2018〕2号《中共佛山市委关于印发<佛山市人才发展体制机制改革实施意见>的通知》 |
| **适用对象** | 新进入佛山市公务员系统或经佛山市党委、政府人才部门办理人才引进手续且社会保险关系在佛山市的博士和博士后；新进入企事业及民办非企业等单位须与所在单位签订2年及以上劳动合同或聘用合同。带高新技术研发成果、专利技术等自主知识产权项目新来佛山成功创办企业的博士、博士后（个人实缴出资额不少于20万元）视同全职新引进 |
| **补贴金额/标准** | 全职新引进博士每人20万元安家补贴；全职新引进博士后每人30万元安家补贴。按50%、50%的比例分2年发放 |
| **学历要求** | 博士研究生、博士后 |
| **年龄限制** | null |
| **社保要求** | 社会保险关系在佛山市，新进入企事业及民办非企业等单位须签订2年及以上劳动合同或聘用合同 |
| **申请条件** | 1.新进入佛山市公务员系统或经佛山市党委、政府人才部门办理人才引进手续；2.社会保险关系在佛山市；3.新进入企事业及民办非企业等单位的须与所在单位签订2年及以上劳动合同或聘用合同；4.具有博士学位或博士后证书；5.领取安家补贴一年后续领须进行资格复核，如更换工作单位需重新提交首次申报相关材料 |
| **申请流程** | 申报（申请人通过"优粤佛山卡APP"提出申请，用人单位通过优粤佛山卡服务平台电脑端管理后台进行申报）→初审（区属单位由各区人才工作部门网上初审，市直单位直接报市人社局）→审核（市人社局审核）→公示（佛山人才网公示5个工作日）→核准（市人社局核准）→发放（划拨至申请人或申请单位） |
| **申请平台** | 优粤佛山卡APP（申请人端）；优粤佛山卡服务平台电脑端管理后台（用人单位端） |
| **申请材料** | 1.安家补贴申请表；2.申请人身份证或护照等有效证件；3.公务员录用审批表、劳动合同或聘用合同；4.申请人毕业证、学位证、博士后证书（国境外博士博士后另提供学历学位认证书）；5.社会保障卡；6.市外调动（引进）或毕业生就业报到凭证；7.国境外博士博士后须提供外国人来华工作许可证或台港澳人员就业证；8.用人单位法人证书或营业执照 |
| **审核周期** | 20个工作日 |
| **发放方式** | 按50%、50%的比例分2年发放，划拨至申请人或申请单位 |
| **生效时间** | 2018年（佛发〔2018〕2号印发） |
| **失效时间** | 待核实 |
| **政策状态** | 有效。全职新引进博士20万元、博士后30万元安家补贴，分2年按50%/50%发放，领取一年后续领须进行资格复核。已享受安家补贴、租房补贴等政策待遇的不重复享受 |
| **官方来源** | https://m.fs.bendibao.com/live/125629.shtm |
| **最后核实日期** | 2026-07-01 |

#### 2. 佛山市新引进中初级人才租房补贴

| 字段 | 内容 |
|------|------|
| **政策名称** | 佛山市新引进中初级人才租房补贴 |
| **政策文号** | 佛发〔2018〕2号《中共佛山市委关于印发<佛山市人才发展体制机制改革实施意见>的通知》（配套《佛山市新引进中初级人才租房补贴工作实施细则》） |
| **适用对象** | 在佛山市行政区域内工商行政管理部门登记的企业和编制部门登记的事业单位全职新引进且办理人才引进手续的中初级人才（不含参照公务员法管理事业单位、公益一类事业单位在编人员）。中初级人才指：中级职称专业技术人才、高级技师、具有硕士学位、具有本科学历的人才 |
| **补贴金额/标准** | 中级职称专业技术人才、高级技师及具有硕士学位的人才每人每月750元；具有本科学历的人才每人每月500元。发放期限不超过36个月。每年9月集中发放租房补贴，发放计算期限从上年度9月至本年度8月。发放至申请人社会保障卡金融账户 |
| **学历要求** | 本科、硕士（或中级职称、高级技师） |
| **年龄限制** | null |
| **社保要求** | 社会保险关系在佛山市，与用人单位签订2年及以上劳动合同或聘用合同；社保缴费中断6个月以上（非本人原因除外）将停止发放 |
| **申请条件** | 1.在佛山市企业和事业单位全职新引进且办理人才引进手续；2.符合中级职称、高级技师、硕士学位、本科学历条件之一；3.经我市党委、政府人才部门办理市外调动（引进）手续或毕业生就业报到手续，与我市用人单位签订2年及以上劳动合同或聘用合同，社会保险关系在我市；4.已享受安家补贴、租房补贴等政策待遇的不重复享受 |
| **申请流程** | 申报（通过佛山人才网"新引进中初级人才租房补贴信息系统"提出申请）→审核（市、区党委政府人才部门按用人单位登记注册地受理审核）→公示（佛山人才网公示5个工作日）→发放（公示结束后次月起计发，每年9月集中发放至社保卡金融账户）。续发补贴须在次年8月10日前再次提交申请表进行资格复核 |
| **申请平台** | 佛山人才网（fsrlzy.cn）"新引进中初级人才租房补贴信息系统" |
| **申请材料** | 1.新引进中初级人才租房补贴申请表；2.与我市用人单位签订的劳动合同或聘用合同；3.申请人身份证；4.中级专业技术资格证、高级技师职业资格证、硕士学位证、本科毕业证、国（境）外学历学位认证书（按相应条件提供）；5.社会保障卡；6.申请人户口簿户主页及本人页；7.市外调动（引进）或毕业生就业报到凭证 |
| **审核周期** | null（经受理、审核、公示5个工作日后发放） |
| **发放方式** | 按月计发，每年9月集中发放上年度9月至本年度8月的补贴，发放至社保卡金融账户，最长不超过36个月 |
| **生效时间** | 2019年9月2日（实施细则印发） |
| **失效时间** | 待核实（实施细则有效期5年，2017年1月1日前的参照执行） |
| **政策状态** | 有效。硕士及中级职称/高级技师750元/月、本科500元/月，发放期限不超过36个月，按月计发每年9月集中发放至社保卡。社保中断6个月以上（非本人原因除外）停止发放。已享受安家补贴等政策的不重复享受 |
| **官方来源** | https://www.fsrlzy.cn/article/1691?typeId=zcfg |
| **最后核实日期** | 2026-07-01 |
