#### 1. 中山市新培养引进博士生活补贴

| 字段 | 内容 |
|------|------|
| **政策名称** | 中山市新培养引进的博士生活补贴 |
| **政策文号** | 中山组发〔2022〕1号（《中山市新时代人才高质量发展二十三条》第15条），《中山市落实青年后备人才待遇实施细则》 |
| **适用对象** | 2022年4月1日以来用人单位新培养引进的、学历或学位经教育部认可的博士（不含公务员和参照公务员管理人员） |
| **补贴金额/标准** | 20万元生活补贴；其中国（境）外引进的博士可另获10万元生活补贴。生活补贴按4:2:4的比例分三年发放 |
| **学历要求** | 博士研究生（学历或学位经教育部认可，含国（境）外引进） |
| **年龄限制** | 不超过45周岁 |
| **社保要求** | 须在中山市工作（以社会保险参保记录或个人纳税证明为准），工作满6个月 |
| **申请条件** | 1.2022年4月1日以来新培养引进；2.45周岁以下；3.博士学历或学位经教育部认可；4.在中山工作满6个月；5.不含公务员和参公管理人员；6.申领时须在中山市工作 |
| **申请流程** | 申请人登录广东政务服务网→全流程网上申领→审核→按4:2:4比例分三年发放至个人账户 |
| **申请平台** | 广东政务服务网（中山市），https://www.gdzwfw.gov.cn/ |
| **申请材料** | 1.本人身份证；2.博士学历学位证书（国（境）外学历需教育部留学服务中心认证）；3.社会保险参保记录或个人纳税证明；4.与用人单位签订的劳动（聘用）合同；5.在中山工作满6个月的相关证明；6.国（境）外引进的另需提供引进相关证明材料（具体材料以广东政务服务网申办指南提示为准） |
| **审核周期** | 待核实（全流程网上申领，按年发放） |
| **发放方式** | 分期，按4:2:4比例分三年发放 |
| **生效时间** | 2022-04-01（二十三条施行）；细则自2023-06-01起施行 |
| **失效时间** | 细则有效期五年（至2028-06-01） |
| **政策状态** | 有效。博士20万元生活补贴分三年发放，境内外引进另有10万元叠加 |
| **官方来源** | http://hrss.zs.gov.cn/xxgk/bmwj/qtwj/content/mpost_2096521.html |
| **最后核实日期** | 2026-07-01 |

#### 2. 中山市企业新培养引进全日制硕士生活补贴

| 字段 | 内容 |
|------|------|
| **政策名称** | 中山市企业新培养引进的全日制硕士生活补贴 |
| **政策文号** | 中山组发〔2022〕1号（《中山市新时代人才高质量发展二十三条》第15条），《中山市落实青年后备人才待遇实施细则》 |
| **适用对象** | 2022年4月1日以来企业新培养和引进的全日制硕士、副高级专业技术人员 |
| **补贴金额/标准** | 一次性3万元生活补贴 |
| **学历要求** | 全日制硕士研究生（副高级专业技术人员同等享受） |
| **年龄限制** | 不超过45周岁 |
| **社保要求** | 须在中山市工作（以社会保险参保记录或个人纳税证明为准），在企业工作满12个月 |
| **申请条件** | 1.2022年4月1日以来企业新培养和引进；2.45周岁以下；3.全日制硕士（或副高级专业技术人员）；4.在中山企业工作满12个月；5.申领时须在中山市工作 |
| **申请流程** | 申请人登录广东政务服务网→全流程网上申领→审核→一次性发放至个人账户 |
| **申请平台** | 广东政务服务网（中山市），https://www.gdzwfw.gov.cn/ |
| **申请材料** | 1.本人身份证；2.全日制硕士学历学位证书；3.社会保险参保记录或个人纳税证明；4.与企业签订的劳动（聘用）合同；5.在中山企业工作满12个月的相关证明；6.以副高级职称申报的须提供副高级专业技术职称证书（具体材料以广东政务服务网申办指南提示为准） |
| **审核周期** | 待核实（全流程网上申领） |
| **发放方式** | 一次性 |
| **生效时间** | 2022-04-01（二十三条施行）；细则自2023-06-01起施行 |
| **失效时间** | 细则有效期五年（至2028-06-01） |
| **政策状态** | 有效。企业新引进全日制硕士一次性3万元 |
| **官方来源** | http://hrss.zs.gov.cn/xxgk/bmwj/qtwj/content/mpost_2096521.html |
| **最后核实日期** | 2026-07-01 |

#### 3. 中山市全日制本科毕业生落户津贴

| 字段 | 内容 |
|------|------|
| **政策名称** | 中山市全日制本科毕业生落户津贴 |
| **政策文号** | 中山组发〔2020〕4号（《关于加强人才政策支持和服务保障的具体措施》） |
| **适用对象** | 毕业5年内的全日制本科毕业生，在中山市企业工作满1年且已落户中山 |
| **补贴金额/标准** | 每人一次性1万元政府津贴 |
| **学历要求** | 全日制本科 |
| **年龄限制** | null（以毕业年限5年内为准，不设年龄上限） |
| **社保要求** | 在中山市企业工作满1年（以社会保险参保记录为准） |
| **申请条件** | 1.毕业5年内的全日制本科毕业生；2.在中山市企业工作满1年；3.已落户中山；4.2015年1月1日后毕业 |
| **申请流程** | 申请人登录广东政务服务网→在线申领→审核→一次性发放至个人账户 |
| **申请平台** | 广东政务服务网（中山市），https://www.gdzwfw.gov.cn/ |
| **申请材料** | 1.本人身份证；2.全日制本科毕业证、学位证；3.中山市户口簿（落户证明）；4.社会保险参保记录；5.与中山市企业签订的劳动（聘用）合同；6.在中山市企业工作满1年的相关证明（具体材料以广东政务服务网申办指南提示为准） |
| **审核周期** | 待核实 |
| **发放方式** | 一次性 |
| **生效时间** | 2020年（中山组发〔2020〕4号） |
| **失效时间** | 待核实（现行有效，与青年后备人才待遇按"就高不就低、扣减差额"原则不重复核发） |
| **政策状态** | 有效。毕业5年内全日制本科、落户中山且在企业工作满1年，一次性1万元 |
| **官方来源** | https://m.thepaper.cn/newsDetail_forward_10791603 |
| **最后核实日期** | 2026-07-01 |
