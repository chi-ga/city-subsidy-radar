#### 1. 南京市高校毕业生生活补贴

| 字段 | 内容 |
|------|------|
| **政策名称** | 南京市高校毕业生生活补贴 |
| **政策文号** | 《南京市高校毕业生生活补贴操作办法》（宁人社规〔2025〕1号，待核实） |
| **适用对象** | 在南京市各类用人单位就业创业的普通高校（含国（境）外）毕业并取得学士及以上学位的全日制毕业生；首次就业创业的非全日制研究生 |
| **补贴金额/标准** | 学士1万元、硕士3万元、博士10万元，分两笔发放（每笔50%） |
| **学历要求** | 学士及以上学位 |
| **年龄限制** | null |
| **社保要求** | 在南京市依法纳税的企业，或机关事业单位、在宁部省属单位就业，按规定在南京市缴纳企业职工基本养老保险；连续参保满6个月可申领第一笔，连续参保满12个月可申领第二笔 |
| **申请条件** | 1. 全日制普通高校（含国（境）外）毕业并取得学士及以上学位；2. 首次就业创业的非全日制研究生（就读前及期间均无就业经历）；3. 在南京市符合条件的用人单位就业创业并缴纳社保；4. 连续参保首月应在毕业后1年内，第一笔申领应在毕业后1年半内，第二笔申领应在毕业后2年内；5. 同一人只能享受一次 |
| **申请流程** | 1. 登录"我的南京"APP；2. 进入智慧人社→补贴服务→生活补贴；3. 填写学历、户籍、连续参保首月等信息并上传证书；4. 系统比对参保信息；5. 审核通过后发放至江苏省社保卡金融账户 |
| **申请平台** | "我的南京"APP（智慧人社→补贴服务→生活补贴） |
| **申请材料** | 1. 毕业证书；2. 学位证书；3. 国（境）外学历学位认证书（留学人员）；4. 参保信息由系统数据比对获取 |
| **审核周期** | 每月前10个工作日完成上月数据审核，审核通过后公示5天 |
| **发放方式** | 分两笔发放，每笔50%，发放至江苏省社保卡金融账户 |
| **生效时间** | 2026-01-01 |
| **失效时间** | 待核实 |
| **政策状态** | 有效。2026年7月1日正式开放申领，目前可预登记 |
| **官方来源** | https://m.nj.bendibao.com/live/181782.shtm |
| **最后核实日期** | 2026-06-30 |

#### 2. 南京市高校毕业生住房租赁补贴

| 字段 | 内容 |
|------|------|
| **政策名称** | 南京市高校毕业生住房租赁补贴 |
| **政策文号** | 《南京市人力资源和社会保障局等七部门关于优化调整高校毕业生住房租赁补贴政策的通知》（宁人社规〔2025〕2号，待核实） |
| **适用对象** | 普通高校及高等职业院校毕业并取得学士及以上学位，或大专学历具有高级工及以上职业资格证书的全日制毕业生；研究生可扩大至非全日制（需首次就业参保） |
| **补贴金额/标准** | 博士2000元/月、硕士800元/月、学士（含高级工及以上）600元/月；重点产业领域企业新引进硕博毕业生博士2400元/月、硕士960元/月。累计不超过36个月 |
| **学历要求** | 学士及以上学位，或大专学历具有高级工及以上职业资格证书 |
| **年龄限制** | null |
| **社保要求** | 与南京市用人单位签订1年及以上劳动合同，依法参加南京市企业职工养老保险（或在省级参保并在南京市参加失业保险），每连续参保缴费6个月可申领一次 |
| **申请条件** | 1. 全日制普通高校及高等职业院校毕业并取得学士及以上学位，或大专学历具有高级工及以上职业资格证书；2. 研究生可扩大至非全日制，需首次就业参保；3. 毕业两年内在南京市各类企业或机关事业单位等就业或自主创业；4. 在宁无自有住房（或与父母共有家庭唯一一套住房）且租房居住；5. 首次申请应在毕业之日起2年半内提交，末次申请应在享受补贴首月起5年内提交 |
| **申请流程** | 1. 下载并登录"我的南京"APP；2. 进入智慧人社→租房补贴；3. 签署《租房承诺书》并填写租赁地址及房屋唯一标识；4. 上传学历学位证书；5. 每月前10个工作日审核上月申请；6. 公示5天后发放至江苏省社保卡金融账户 |
| **申请平台** | "我的南京"APP（智慧人社→租房补贴） |
| **申请材料** | 1. 国内全日制普通高校提交学历、学位证书；2. 全日制高等职业院校提交学历、高级工及以上职业资格证书；3. 海外留学人员提交教育部留学服务中心国（境）外学历学位认证书；4. 租房承诺书；5. 租赁地址及房屋唯一标识（不动产单元号、丘权号或商品房预售合同编号三选一） |
| **审核周期** | 每月前10个工作日完成上月数据审核，公示5天 |
| **发放方式** | 按月发放，发放至江苏省社保卡金融账户 |
| **生效时间** | 2025-10-01 |
| **失效时间** | 2030-09-30（有效期5年） |
| **政策状态** | 有效。2025年10月1日起施行，此前有关租房补贴规定继续执行，不一致的按本通知执行 |
| **官方来源** | https://rsj.nanjing.gov.cn/njsrlzyhshbzj/202510/t20251023_5674924.html |
| **最后核实日期** | 2026-06-30 |

#### 3. 雨花台区优秀高校毕业生生活补贴

| 字段 | 内容 |
|------|------|
| **政策名称** | 雨花台区优秀高校毕业生生活补贴 |
| **政策文号** | 《雨花台区高质量人口人才政策12条》（待核实） |
| **适用对象** | 经认定的科技型企业新引进的毕业2年内优秀高校毕业生 |
| **补贴金额/标准** | 硕士2万元/人、博士5万元/人，按1:1分两年拨付 |
| **学历要求** | 硕士、博士 |
| **年龄限制** | null |
| **社保要求** | 申领第一笔须在雨花台区符合条件的用人单位连续缴纳12个月及以上社会保险；申领第二笔须再次连续缴纳12个月及以上 |
| **申请条件** | 1. 引进单位为雨花台区现代产业领域规模以上工业企业、国家高新技术企业、省级以上专精特新企业、重大创新平台、高水平新型研发机构、独角兽/培育独角兽/瞪羚企业、区级以上重点人才工程入选者创办企业；2. 毕业生须为世界知名高校（入职当年QS世界大学排名前200）或"双一流"建设高校（学科）全日制毕业生；3. 2023年1月1日以后首次来南京在雨花台区就业参保；4. 从2025年1月1日起往前推算毕业时间不超过2周年；5. 申领时为符合条件的用人单位在职员工 |
| **申请流程** | 1. 以企业为单位申报，登录"及时雨政策资金全过程服务平台"；2. 选择"优秀高校毕业生生活补贴"；3. 填写并上传企业信息资料；4. 新增填写毕业生信息资料；5. 提交后由软件谷及街道（园区）、区人社局初审、复审；6. 审核通过并公示后分两年拨付 |
| **申请平台** | "及时雨政策资金全过程服务平台"（企业微信扫码登录） |
| **申请材料** | 1. 高校毕业生有效身份证明（身份证或护照）；2. 毕业证书、学位证书及教育部学历证书电子注册备案表；3. 境外学位需提供教育部留学服务中心学历认证；4. 入职以来与用人单位签订的劳动合同；5. 入职以来的《江苏省社会保险权益记录单（参保人员）》 |
| **审核周期** | 每年开展一次集中申报，具体时间以申报通知为准 |
| **发放方式** | 按1:1分两年拨付 |
| **生效时间** | 2025-01-01（待核实） |
| **失效时间** | 待核实 |
| **政策状态** | 有效。属于雨花台区"高质量人口人才政策12条"之一 |
| **官方来源** | https://m.nj.bendibao.com/mip/175793.shtm |
| **最后核实日期** | 2026-06-30 |
