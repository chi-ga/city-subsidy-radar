#### 1. 广州市博士安家费

| 字段 | 内容 |
|------|------|
| **政策名称** | 广州市博士安家费 |
| **政策文号** | 穗人才领字〔2020〕8号、粤组通〔2017〕46号 |
| **适用对象** | 全日制博士，到市属企事业单位全职工作或自主创业的博士人员 |
| **补贴金额/标准** | 境内博士：一次性10万元；国（境）外引进博士：20万元（分两期发放，每期10万元） |
| **学历要求** | 全日制博士并取得博士学位 |
| **年龄限制** | 签订劳动合同时年龄在35岁以下（含35岁） |
| **社保要求** | 连续1年在市属企事业单位缴纳社会保险 |
| **申请条件** | 具有本市户籍；与市属企事业单位签订3年以上劳动合同（聘用协议）；引进之日起2年内（境内）或3年内（境外）提出申请；已获省财政住房补贴者不享受 |
| **申请流程** | 个人登录"智慧人才家园"广州市博士博士后管理系统或广东政务服务网/穗好办APP→填写资料→提交法人单位审核→法人单位提交主管部门审核→主管部门提交市博管办窗口审核→公示→拨付 |
| **申请平台** | 智慧人才家园：https://gzrsj.rsj.gz.gov.cn/vsgzhr/login_home.aspx；广东政务服务网；穗好办APP |
| **申请材料** | 1.户口簿；2.博士学历证书、学位证书（持港澳台及国外证书的需同时提交相关鉴定材料）；3.劳动合同或聘用协议；4.法人证书（非企业单位提供）；5.获省财政住房补贴或市级同类资助情况说明及佐证材料 |
| **审核周期** | 约30-60个工作日 |
| **发放方式** | 境内博士一次性拨付至个人账户；境外博士分两期拨付至个人账户 |
| **生效时间** | 2020年（穗人才领字〔2020〕8号） |
| **失效时间** | 待核实（长期有效） |
| **政策状态** | 有效（2025年9月公示第四批博士安家费拟发放名单） |
| **官方来源** | https://rsj.gz.gov.cn/ywzt/ztbd/bwyc/bsh/content/post_10302727.html |
| **最后核实日期** | 2026-06-28 |

#### 2. 广州市博士后研究人员安家费

| 字段 | 内容 |
|------|------|
| **政策名称** | 广州市博士后研究人员安家费 |
| **政策文号** | 穗人社规字〔2024〕5号、粤组通〔2017〕46号 |
| **适用对象** | 博士后出站后到市属企事业单位全职工作或自主创业的博士后人员 |
| **补贴金额/标准** | 30万元，分三期平均发放（每期10万元） |
| **学历要求** | 博士后出站人员 |
| **年龄限制** | 无统一年龄限制 |
| **社保要求** | 在市属单位全职工作满1年/2年/3年后分别申请各期 |
| **申请条件** | 出站后1年内到市属单位全职工作签订3年以上合同，或出站后1年内在广州自主创业；出站前在市属单位工作不超过1年；非党政机关参公人员（专业技术岗除外）；在站从事研究不少于12个月（境外） |
| **申请流程** | 个人在"智慧人才家园"广州市博士博士后管理系统发起申请→法人单位初审→主管部门复核→市人力资源社会保障局审核→公示5个工作日→拨付 |
| **申请平台** | 智慧人才家园：https://gzrsj.rsj.gz.gov.cn/vsgzhr/login_home.aspx |
| **申请材料** | **境内博士后**：1.中国博士后网出站备案证明；2.聘用协议/劳动合同/增人卡；3.博士后证书；4.获市级财政同类资助情况说明。**境外博士后**：1.博士后进修证明（附翻译件）；2.留学回国人员证明/驻外使馆推荐意见；3.博士学位证书；4.聘用协议/劳动合同/增人卡；5.身份证明 |
| **审核周期** | 约30-60个工作日 |
| **发放方式** | 分三期拨付至个人账户（每期10万元） |
| **生效时间** | 2024年12月24日（穗人社规字〔2024〕5号） |
| **失效时间** | 2029年12月23日（有效期5年） |
| **政策状态** | 有效（2024年12月23日更新实施办法） |
| **官方来源** | https://rsj.gz.gov.cn/ywzt/ztbd/bwyc/bsh/content/post_10302738.html |
| **最后核实日期** | 2026-06-28 |

#### 3. 广州市在站博士后研究人员生活补贴

| 字段 | 内容 |
|------|------|
| **政策名称** | 广州市在站博士后研究人员生活补贴 |
| **政策文号** | 穗人社规字〔2024〕5号 |
| **适用对象** | 市属单位设立的博士后科研流动站、工作站（分站）、市级博士后创新实践基地招收的博士后人员 |
| **补贴金额/标准** | 工作站、分站、市级创新实践基地：每人每年18万元，资助期限2年；流动站：每人每年8万元，资助期限2年 |
| **学历要求** | 具有博士学位 |
| **年龄限制** | 一般在35周岁以下（人文社科、重点专业领域、临床医学或企业工作站可放宽至40周岁） |
| **社保要求** | 在站期间由设站单位缴纳社保 |
| **申请条件** | 市属单位在站博士后；开题考核合格且在站满3个月（第一期）/中期考核合格且在站满1年（第二期）；非在职人员；非已入职设站单位超过1年后又进原单位开展博士后研究的人员 |
| **申请流程** | 个人在"智慧人才家园"广州市博士博士后管理系统发起申请→法人单位初审→主管部门复核→市人力资源社会保障局审核→公示5个工作日→拨付至设站单位账户 |
| **申请平台** | 智慧人才家园：https://gzrsj.rsj.gz.gov.cn/vsgzhr/login_home.aspx |
| **申请材料** | 1.博士后研究人员进站备案证明；2.博士学位证书；3.开题通过佐证材料（第一期）；4.中期考核佐证材料（第二期） |
| **审核周期** | 约30-60个工作日 |
| **发放方式** | 分两期拨付至设站（基地）单位银行账户，由单位核发至博士后个人 |
| **生效时间** | 2024年12月24日 |
| **失效时间** | 2029年12月23日（有效期5年） |
| **政策状态** | 有效 |
| **官方来源** | https://rsj.gz.gov.cn/ywzt/rcgz/bsh/ywsb/gryw/content/post_10302521.html |
| **最后核实日期** | 2026-06-28 |

| **审核周期** | 约15-30个工作日 |
| **发放方式** | 分3年等额发放，2025年起按月免申即享发放至个人账户 |
| **生效时间** | 2024年（2024年修订版） |
| **失效时间** | 待核实 |

#### 1. 花都区引进优秀人才住房补贴

| 字段 | 内容 |
|------|------|
| **政策名称** | 花都区引进优秀人才住房补贴 |
| **政策文号** | 花府办规〔2025〕1号、花人社函〔2025〕269号 |
| **适用对象** | 2023年1月1日至2025年12月31日期间新引进花都区的5类优秀人才 |
| **补贴金额/标准** | 1.企业家和高级管理人才：10万元，分3年申请发放；2.优秀外国专家：10万元，分3年申请发放；3.港澳地区优秀人才：10万元，分3年申请发放；4.高级职称人才（副高级及以上）：5万元，分2年申请发放；5.理工经管类专业博士：5万元，分2年申请发放 |
| **学历要求** | 博士/硕士（部分类别不限学历，按人才类别要求） |
| **年龄限制** | 企业家和高级管理人才/优秀外国专家：首次申请时未达到法定退休年龄；港澳优秀青年：不超过44周岁；高级职称人才/理工经管类专业博士：不超过40周岁 |
| **社保要求** | 在花都区企业全职工作并连续缴纳社保满6个月（港澳/外籍提供2020年至今的个税记录） |
| **申请条件** | 2023年1月1日至2025年12月31日期间新引进；在花都区企业工作；在花都区有合法长期住所（自有房产/租赁备案/单位宿舍证明） |
| **申请流程** | ①进入「花都区政府」微信小程序，点击「人才管家」系统 → ②从「人才管家」首页点击「人才引进」，选择「花都区引进优秀人才住房补贴」栏目的「我要申报」 → ③如实填写申请表并上传相关证明材料，经所在单位审核后提交 → ④区人力资源社会保障局审核（材料不符要求的10个工作日内补齐补正） → ⑤审核通过后在花都区人民政府门户网站公示5个工作日 → ⑥公示无异议后按年度拨付至个人账户 |
| **申请平台** | 花都区政府微信小程序—人才管家系统 |
| **申请材料** | 1.广州市花都区引进优秀人才住房补贴申请表；2.有效身份证明（身份证/港澳居民身份证/护照）；3.劳动合同复印件；4.本人境内银行卡复印件；5.社会保险费缴纳清单（通过粤省事查询）；6.花都区居住证明（房产证明/租赁备案证明/单位宿舍证明）；7.人才资质证明（按类别提供：高管任职文书/外国人工作许可证A证/专业技术资格证书/学历学位证书）；8.授权委托书（非本人申报时提供） |
| **审核周期** | 约30-45个工作日 |
| **发放方式** | 分2年或3年按年度发放至个人账户 |
| **生效时间** | 2025年1月1日 |
| **失效时间** | 待核实 |
| **政策状态** | 有效（2026年3月30日至6月30日受理2023年至2025年新引进优秀人才住房补贴申请） |
| **官方来源** | https://www.huadu.gov.cn/zfxxgkml/gzshdqrlzyhshbzj/content/post_10747460.html |
| **最后核实日期** | 2026-06-28 |
