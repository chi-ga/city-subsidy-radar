#### 1. 苏州市新引进应届高校毕业生租房补贴

| 字段 | 内容 |
|------|------|
| **政策名称** | 苏州市新引进应届高校毕业生租房补贴（安家生活补贴） |
| **政策文号** | 待核实 |
| **适用对象** | 2025年3月27日（含）后首次从苏州大市以外新引进的全日制应届博士、硕士、本科毕业生 |
| **补贴金额/标准** | 全日制应届博士1500元/月、硕士1000元/月、本科800元/月，补贴期一般不超过2年。经认定的人工智能、量子科技领域企业和新型研发机构新引进的急需紧缺专业全日制应届博士、硕士及本科生，补贴期可延长至3年 |
| **学历要求** | 全日制本科、硕士、博士（境外高校毕业生需经教育部留学服务中心认证） |
| **年龄限制** | null（以毕业时间为准：境内高校毕业生首次申领时限为毕业后3年内，境外高校毕业生为毕业后6年内） |
| **社保要求** | 境内高校毕业生首次申领时需满足毕业后2年半内在苏州符合条件的用人单位连续缴纳社保（不含补缴）满6个月，且连续参保的首月在毕业后2年内；境外高校毕业生首次申领时需满足毕业后5年半内连续缴纳社保满6个月，且连续参保的首月在毕业后5年内。续领时需在苏州符合条件的用人单位正常参保（不含断缴、补缴） |
| **申请条件** | 1. 2025年3月27日（含）后首次从苏州大市以外新引进（此前未在苏州大市范围内有过参保记录）；2. 具有全日制本科及以上学历（境外高校需认证）；3. 在符合条件的用人单位工作并连续缴纳社保满6个月（不含补缴）；4. 在苏州大市范围内无房产；5. 公务员、参公事业单位人员不适用；6. 国有企业人员按国资监管部门有关规定执行 |
| **申请流程** | 1. 下载"苏周到"APP并实名认证→2. 依次点击"苏州人才"→"拿补贴"→"应届高校毕业生租房补贴"→3. 在线填写学历证书编号（境外高校毕业生填写教育部留学服务中心学历学位认证书编号），其他信息由系统自动采集→4. 首次申领成功后兑现6个月补贴→5. 续领补贴"免申即享"，系统自动比对参保信息 |
| **申请平台** | "苏周到"APP（"苏州人才"→"拿补贴"→"应届高校毕业生租房补贴"） |
| **申请材料** | 1. 学历证书编号（境内高校毕业生）；2. 教育部留学服务中心学历学位认证书编号（境外高校毕业生）；3. 其他所需信息由系统自动采集（劳务派遣人员需额外提供：申请人学历证书、实际用人单位与派遣公司签订的有效期内的派遣协议（合同）及申请人与派遣公司的劳动合同等） |
| **审核周期** | 每季度集中兑现一次，兑现时间一般为2月、5月、8月、11月 |
| **发放方式** | 首次申领成功后兑现6个月补贴，续领按季度集中兑现 |
| **生效时间** | 2025年3月27日 |
| **失效时间** | 待核实 |
| **政策状态** | 有效。2025年10月已开放首次申领。不限户籍，满足条件即可申请。补贴分首次申领和续领两个阶段，续领"免申即享"。 |
| **官方来源** | https://m.suzhou.bendibao.com/live/133277.shtm |
| **最后核实日期** | 2026-07-01 |

#### 1. 苏州工业园区虚拟优租房补贴（国资人才公寓租金补贴）

| 字段 | 内容 |
|------|------|
| **政策名称** | 苏州工业园区国资人才公寓"虚拟优租房"租金补贴 |
| **政策文号** | 《苏州工业园区国资人才公寓管理办法（试行）》 |
| **适用对象** | 在园区就业、创业并缴纳社保、个税或住房公积金，个人及家庭（含未成年子女）在园区范围内无自有住房的本科及以上学历人才 |
| **补贴金额/标准** | 博士最高1000元/月、硕士最高900元/月、本科最高600元/月，以上补贴不超过合同租金，时间最长3年。获得人才优租房的，停止发放"虚拟优租房"补贴；"虚拟优租房"补贴与人才优租房租赁期限合并计算最长不超过3年 |
| **学历要求** | 全日制本科、硕士、博士（大专及以上学历可租住国资人才公寓，但租金补贴仅限本科及以上） |
| **年龄限制** | null |
| **社保要求** | 在园区就业、创业并缴纳社保、个税或住房公积金（尚未入职或入职后社保/个税尚未缴交的，可签订一次3个月的短期租赁合同，期满符合条件予以续租） |
| **申请条件** | 1. 在园区就业、创业并缴纳社保、个税或住房公积金；2. 个人及家庭（含未成年子女）在园区范围内无自有住房；3. 具有大专及以上学历人才、具有初级及以上职称人才、具有职业技能等级四级及以上人才或园区人才办认定的其他符合条件的人才（租金补贴仅限本科及以上）；4. 租住园区国资人才公寓 |
| **申请流程** | 1. 租住园区国资人才公寓（房源纳入"iHome"人才安居融合服务平台统一备案管理）→2. 通过"iHome"人才安居融合服务平台申请"虚拟优租房"补贴→3. 平台审核资格→4. 按月发放租金补贴（不超过合同租金） |
| **申请平台** | "iHome"人才安居融合服务平台 |
| **申请材料** | 1. 有效身份证件；2. 学历证明（学信网《教育部学历证书电子注册备案表》或教育部留学服务中心学历学位认证书）；3. 社保/个税/公积金缴纳证明；4. 国资人才公寓租赁合同；5. 无房证明（系统核验） |
| **审核周期** | null |
| **发放方式** | 按月发放（不超过合同租金），最长3年 |
| **生效时间** | 2024年1月1日 |
| **失效时间** | 2026年12月31日（试行3年） |
| **政策状态** | 有效。2024年1月1日起试行3年。所有房源纳入"iHome"平台统一备案管理，租金应不高于同地段、同品质市场租金。 |
| **官方来源** | https://m.bjnews.com.cn/detail/1705672201169490.html |
| **最后核实日期** | 2026-07-01 |

#### 2. 苏州工业园区人才房票补贴

| 字段 | 内容 |
|------|------|
| **政策名称** | 苏州工业园区人才房票补贴 |
| **政策文号** | 《苏州工业园区人才房票补贴实施细则（试行）》 |
| **适用对象** | 在园区就业、创业并连续正常缴纳6个月及以上社保或个税，且个人及家庭（含未成年子女）在园区内无自有住房的全日制大专及以上人才 |
| **补贴金额/标准** | 全日制普通高校大专生7万元、本科生10万元、硕士研究生15万元、博士研究生25万元，一次性抵扣房款 |
| **学历要求** | 全日制大专、本科、硕士、博士（海外留学归国人员须经教育部认定具备同等学历学位） |
| **年龄限制** | null |
| **社保要求** | 申请时在园区就业、创业并连续正常缴纳6个月及以上社保或个税（不含补缴） |
| **申请条件** | 1. 在园区就业、创业并连续正常缴纳6个月及以上社保或个税（不含补缴）；2. 个人及家庭（含未成年子女）在园区内无自有住房（含商品住房、动迁安置房和农村宅基地等）；3. 全日制大专及以上学历或学位；4. 购买园区范围内销售备案均价低于5万元/平方米的新建商品住房（须在房源库内） |
| **申请流程** | 1. 符合条件的人才与房地产开发企业签订意向购房合同→2. 由房地产开发企业协助申请人在苏州工业园区人才房票申办平台办理资格申请→3. 上传申请材料（身份证件、学历证明、婚姻状况证明、户口本、未成年人出生证明、材料真实性承诺书、房地产开发企业承诺书、意向购房合同等）→4. 园区人才办、规建委会同相关部门审核资格、类别和房票金额→5. 学信网或教育部留学服务中心学历核实→6. 住房信息核实→7. 社保或个税核实→8. 平台发放人才房票→9. 购房时将房票交由房地产开发企业核验，按票面金额一次性抵扣房款（可抵首付）→10. 开发企业凭房票、已备案购房合同等材料报规建委核查结算 |
| **申请平台** | 苏州工业园区人才房票申办平台 |
| **申请材料** | 1. 有效身份证件；2. 学历证明（学信网《教育部学历证书电子注册备案表》或《中国高等教育学位在线验证报告》截图，需在有效期内并包含可供扫描查验的二维码信息；境外学历需提供教育部留学服务中心国（境）外学历学位认证）；3. 婚姻状况证明（结婚登记证件或婚姻登记档案）；4. 户口本；5. 未成年人出生证明（如有未成年子女）；6. 材料真实性承诺书；7. 房地产开发企业承诺书（申办平台可下载模板）；8. 买受人签字且房地产开发企业盖章的意向购房合同 |
| **审核周期** | null |
| **发放方式** | 一次性抵扣房款（房票可抵首付，不得超过购房款总额） |
| **生效时间** | 2025年1月1日 |
| **失效时间** | 2025年12月31日（试行1年），续期政策待核实 |
| **政策状态** | 试行期满/待核实。本细则自2025年1月1日起试行，政策有效期1年。人才房票自发放之日起30日内有效，逾期未使用自动失效。一人一票、专款专用，与人才购房（安家）补贴的本级配套有重复、交叉的，按照"从优、从高"原则执行。续期政策请关注园区官方通知。 |
| **官方来源** | https://m.suzhou.bendibao.com/news/128037.shtm |
| **最后核实日期** | 2026-07-01 |

#### 1. 苏州高新区人才房票补贴

| 字段 | 内容 |
|------|------|
| **政策名称** | 苏州高新区人才房票补贴 |
| **政策文号** | 《苏州高新区人才房票补贴实施细则（试行）》 |
| **适用对象** | 在高新区就业创业的全日制大专（高级工）及以上人才，或意向来（留）苏州工作的全日制大专及以上在校生，或在高新区申报入选区级及以上人才计划且目前仍符合申报条件的人才 |
| **补贴金额/标准** | 全日制博士研究生25万元、硕士研究生15万元、本科生10万元、大专生（含对应学段在校生）7万元；高级工以上技能人才7万元；人才计划入选者最高补贴200万元 |
| **学历要求** | 全日制大专、本科、硕士、博士（含对应学段在校生）；高级工以上技能人才 |
| **年龄限制** | null |
| **社保要求** | 在高新区就业创业（需提供近1个月内社保记录或个税记录；在校生无需提供） |
| **申请条件** | 符合以下任一条件即可：1. 在本区申报入选区级及以上人才计划且目前仍符合申报相关条件的人才；2. 在本区就业创业的全日制大专（高级工）及以上人才；3. 意向来（留）苏州工作的全日制大专及以上在校生 |
| **申请流程** | 1. 在"乐居高新人才房票"小程序点击房票申请→2. 填写个人基础信息（姓名、性别、国籍、身份证号码、单位名称、单位所属功能片区、手机号、联系地址）→3. 上传附件材料（基础材料+关联证明材料）→4. 审核通过后在小程序中领取相应额度的"苏州高新区人才房票"电子票→5. 购房时出示电子票由开发企业核验后生成房票使用顺序号→6. 开发企业在小程序中上传复核材料（商品房认购协议、定金付款凭证等）→7. 开发企业将材料递交至高新区行政服务中心住建窗口提交审核→8. 名单公示（公示期2个工作日）→9. 公示无异议后，开发企业将人才房票按票面金额一次性抵扣房款 |
| **申请平台** | "乐居高新人才房票"小程序 |

| **审核周期** | null |
| **发放方式** | 一次性抵扣房款（房票可抵首付但不得超过购房款总额） |
| **生效时间** | 2025年3月13日 |
| **失效时间** | 2025年10月31日（试行期满，续期政策待核实） |
| **政策状态** | 试行期满/待核实。本细则自2025年3月13日起试行，有效期至2025年10月31日。人才房票实行实名制，每名申请人限申领一张、只享受一次，不得抵押、提现、转让、赠与。所购住房须在政策执行期内办理认购及网签备案手续。与人才安家（购房）补贴、重点产业紧缺人才补贴等同类政策按照"从优、从高、不重复"原则执行。续期政策请关注高新区官方通知。 |
