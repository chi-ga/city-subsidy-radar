#### 1. 重庆市12个县大学生就业补贴政策

| 字段 | 内容 |
|------|------|
| **政策名称** | 重庆市12个县大学生就业补贴政策 |
| **政策文号** | 渝人社规〔2025〕22号 |
| **适用对象** | 在重庆市12个县企业就业，签订劳动合同并缴纳社会保险费的毕业年度高校毕业生和离校5年内高校毕业生 |
| **补贴金额/标准** | 本科学历1000元/人/月、硕士学历2000元/人/月、博士学历3000元/人/月，经认定后最长可补贴3年 |
| **学历要求** | 全日制本科及以上 |
| **年龄限制** | null（以离校5年内为限） |
| **社保要求** | 须在12个县企业新参保缴费；离校5年内高校毕业生须为重庆市首次参保；毕业年度高校毕业生不受重庆市首次参保限制 |
| **申请条件** | 1. 从2025年2月1日起在12个县企业就业；2. 签订劳动合同并缴纳社会保险费；3. 参保企业和实际就业企业须保持一致，且实际工作地须在12个县内；4. 须在12个县所属企业（以注册地为准）参保 |
| **申请流程** | 1. 在12个县企业就业参保后办理就业登记；2. 通过“渝快办”平台“渝悦·就业”应用提出个人申请；3. 街道（乡镇）公共就业服务机构审核实际就业情况；4. 县公共就业服务机构复审；5. 公示5个工作日；6. 财政部门确认后按季度划入申请人个人社保卡银行账户 |
| **申请平台** | “渝快办”平台“渝悦·就业”应用 |
| **申请材料** | 1. 个人身份证明；2. 劳动合同；3. 社保缴纳证明；4. 学历学位证书；5. 个人社保卡银行账户信息 |
| **审核周期** | 公示5个工作日，按季度发放 |
| **发放方式** | 按季度划入申请人个人社保卡银行账户，有条件的县可以按月划拨 |
| **生效时间** | 2025-02-01 |
| **失效时间** | 补贴对象认定工作至2025-12-31截止，到期后根据实际情况确定是否延续 |
| **政策状态** | 现行有效 |
| **官方来源** | http://www.cq.gov.cn/zt/zcztclwz/detail.html?policyId=5626 |
| **最后核实日期** | 2026-06-30 |
