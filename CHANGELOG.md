# 更新日志

本项目遵循 [Semantic Versioning](https://semver.org/)。

## [0.1.0] - 2026-06-28

### 新增
- 首页中国地图展示，支持点击城市跳转
- 三种查询路径：单城查询、多城对比、政策库浏览
- 条件输入表单，支持院校/专业自动补全
- 规则匹配引擎，支持互斥组处理、多通道认定
- 结果分析页，展示匹配补贴、预估金额、反向匹配
- 落地待办清单，支持勾选完成
- 城市专属字段适配（深圳/北京/上海/广州）
- 结果分享功能（Web Share API / 剪贴板复制）
- 政策库浏览页，支持多维筛选
- 响应式设计，移动端适配

### 数据
- 北京补贴政策 11 条
- 上海补贴政策 14 条
- 深圳补贴政策 14 条
- 广州补贴政策 6 条
- 院校数据 528 所
- 本科专业 796 个

### 技术
- React 19 + TypeScript
- Vite 构建
- Zustand 状态管理
- Tailwind CSS 样式
- ECharts 图表/地图
- Vitest 测试框架
